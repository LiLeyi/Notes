    抽样分布

    统计量的分布

    卡方分布

    设\(X_1, X_2, \cdots, X_n\)是来自总体\(N(0,1)\)的样本，则称统计量

    \[\chi^2 = \sum_{n=1}^i X_i^2\]

    为服从自由度为n的\(\chi^2\)分布，记为\(\chi^2 \sim \chi^2(n)\)，其中n表示随机变量的个数。概率密度：

    \[f(x) = \begin{cases} \frac 1 {2^{n/2}\Gamma(n/2)} x^{\frac n 2 - 1} e^{- \frac x 2}, & x>0 \\ 0, & x \leq 0 \end{cases}\]
    \[\Gamma(\alpha) = \int_0^\infty x^{\alpha - 1} e^{-x} dx\]

    自由度为2的卡方分布，为指数分布\(E(\frac 1 2)\)。n越大越接近正态分布（中心极限定理）

    推论：

    1. 若\(X_1, X_2, \dots, X_n\)独立，且都服从\(N(\mu, \sigma^2)\)，则\(\frac 1 {\sigma^2} \sum(X_i - \mu)^2 \sim \chi^2(n)\)（即先进行标准化）
    2. 设\(X\sim \chi^2(n)\)，则\(E(X) = n, D(X) = 2n\)
    3. 可加性。\(X_1\sim \chi^2(n_1), X_2\sim \chi^2(n_2)\)，且\(X_1, X_2\)相互独立，则\(X_1 + X_2 \sim \chi^2(n_1 + n_2)\)。可推广

    t分布

    设\(X\sim N(0,1),Y\sim \chi^2(n)\)，且X,Y相互独立，称随机变量

    \[t = \frac X {\sqrt{\frac Y n}}\]

    服从自由度为n的t分布，记为\(t \sim t(n)\)。密度函数

    \[f(x) = \frac {\Gamma((n+1)/2)} {\Gamma(n/2)\sqrt{n\pi}} (1+x^2/n)^{-\frac {n+1} 2}\]

    为偶函数。n趋于无穷时，分母趋于1，t分布收敛到标准正态分布（n>45时可近似认为是正态分布）。n较小时与正态分布差异较大，此时在尾部比正态分布有更大的概率

    对于\(1<r<n\)，

    \[E(T^r) = \begin{cases} n^{\frac r 2} \frac {\Gamma((r+1)/2)\Gamma((n-r)/2)} {\sqrt{\pi}\Gamma(n/2)}, & r \text{为偶数} \\ 0, & r \text{为奇数} \end{cases}\]

    特别的，\(E(T) = 0, D(T) = n/(n-2)\)

    n=1时，变为柯西分布（期望不存在），密度函数为

    \[f(x) = \frac 1 {\pi(1+x^2)}\]

    F分布

    设\(X\sim \chi^2(m),Y\sim \chi^2(n)\)，且X,Y相互独立，称随机变量

    \[F = \frac {X/m} {Y/n}\]

    服从自由度为(m,n)的F分布。密度函数为

    \[f(x) = \begin{cases} \frac {\Gamma((m+n)/2)} {\Gamma(m/2)\Gamma(n/2)} (\frac m n)^{\frac m 2} x^{\frac m 2 - 1}(1+\frac m n x)^{-\frac {m+n} 2}, & x>0\\ 0, & x \leq 0 \end{cases}\]

    性质：

    • 若\(X\sim F(m,n)\)，则\(1/ X \sim F(n,m)\)
    • 若\(T\sim t(n)\)，则\(T^2 \sim F(1,n)\)

    样本与总体的关系

    设\(X_1, X_2, \dots, X_n\)是来自总体X（不管是什么分布，只需要均值方差存在），则：

    \[E(\bar X) = E(X), D(\bar X) = \frac {D(X)} n, E(S^2) = D(X)\]

    设\(X_1, X_2, \dots, X_n\)，是来自正态总体\(N(\mu, \sigma^2)\)的样本，则

    \[\bar X \sim N(\mu, \frac {\sigma^2}n), 即\frac {\bar X - \mu} {\sigma/\sqrt{n}}\sim N(0,1)\]
    \[\frac {(n-1)S^2}{\sigma^2}\sim \chi^2(n-1)\]
    \[\frac {\bar X -\mu} {S/\sqrt{n}} \sim t(n-1)\]

    且\(\bar X\)与\(S^2\)相互独立

    设\(X_1, X_2, \dots, X_n\)是来自正态总体\(N(\mu_1, \sigma^2)\)的样本，设\(Y_1, Y_2, \dots, Y_n\)是来自正态总体\(N(\mu_2, \sigma^2)\)的样本，且两个样本相互独立，\(\bar X, \bar Y, S_1^2, S_2^2\)分别为两个样本的样本均值与方差，则

    \[\frac {\bar X - \bar Y - (\mu_1 - \mu_2)} {S_\omega \sqrt{\frac 1 n + \frac 1 m}}\sim t(n + m - 2)\\ S_\omega^2 = \frac {(n-1)S_1^2 + (m-1)S_2^2} {n + m - 2}\]
    \[\frac {S_1^2/S_2^2} {\sigma_1^2/\sigma_2^2} \sim F(m-1, nx-1)\]

    分数位

    定义

    设\(\alpha \in (0, 1)\)，对连续型随机变量X，则称\(P\{X > x_\alpha\} = \alpha\)的点\(x_\alpha\)为X的概率分布的上\(\alpha\)分位数（分位点）

    即密度函数在\(x_\alpha\)右侧面积为\(\alpha\)

    标准正态分布

    对于标准正态分布，称满足\(P\{X>z_\alpha\} = \alpha\)的点为标准正态分布的上\(\alpha\)分位数

    • \(\Phi(z_\alpha) = 1 - \alpha\)。由分布函数定义得
    • \(z_{1-\alpha} = -z_\alpha\)。由对称性得
    • \(P\{|Z| > z_{\alpha/2}\} = \alpha, P\{|Z| \leq z_{\alpha/2}\} = 1 - \alpha\)

    \(\chi^2\)分布

    设随机变量\(X \sim \chi^2(n)\)，对于\(\alpha \in (0, 1)\)，称满足\(P\{X > \chi^2_\alpha(n)\} = \alpha\)的点\(\chi^2_\alpha(n)\)为自由度为n的\(\chi^2\)分布上的\(\alpha\)分位数（分位点）

    • n充分大时（\(n >45\)）：\(\chi^2_\alpha(n) \approx \frac 1 2 (z_\alpha + \sqrt{2n-1})^2\)
    • \(P\{\chi^2 > \chi^2_{\alpha/2}(n)\} + P\{\chi^2 < \chi^2_{1-\alpha/2}(n)\} = \alpha\)
    • \(P\{\chi^2_{1-\alpha/2}(n) \leq \chi^2 \leq \chi^2_{\alpha/2}(n)\} = 1 - \alpha\)

    t分布

    设随机变量\(X \sim t(n)\)，对于\(\alpha \in (0, 1)\)，称满足\(P\{X > t^2_\alpha(n)\} = \alpha\)的点\(t_\alpha(n)\)为自由度为n的t分布上的\(\alpha\)分位数（分位点）

    • \(t_{1-\alpha}(n) = -t_\alpha(n)\)
    • 若\(P\{|X| > c\} = \alpha\)，则\(c = t_{\alpha / 2}(n)\)
    • 当n较大时（n>45），\(t_\alpha(n) \approx z_\alpha\)
    • \(P\{|t| > t_{\alpha / 2}(n)\} = \alpha\)
    • \(P\{|t| \leq t_{\alpha / 2}(n)\}= 1 - \alpha\)

    F分布

    设随机变量\(X \sim F(m,n)\)，对于\(\alpha \in (0, 1)\)，称满足\(P\{X > F_\alpha(m, n)\} = \alpha\)的点\(F_\alpha(m, n)\)为自由度为m,n的t分布上的\(\alpha\)分位数（分位点）
