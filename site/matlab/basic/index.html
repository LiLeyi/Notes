    基础

    语句末尾带分号

    变量

    变量：用 变量名 = 值 赋值

    结果格式

    • format long 显示小数点后十六位
    • format long e 允许以指数形式显示，十六位加上指数
    • format short 显示小数点后三位
    • format short 允许以指数形式显示，五位加上指数
    • format bank 四舍五入到小数点后两位
    • ...

    特殊变量和常量

    • ans：最后计算得到的答案
    • eps：浮点精度
    • i、j：虚数单位
    • Inf：无穷
    • NaN：未定义的数值结果（非数字）
    • pi：圆周率

    变量命名

    变量名称由字母组成，后跟任意数量的字母，数字或下划线。

    MATLAB是 case-sensitive。

    变量名称可以是任意长度，但是，MATLAB仅使用前N个字符，其中N由函数给定 namelengthmax。

    保存加载

    • save 文件名：保存为文件名.mat
    • load 文件名：从文件名.mat中读取数据
    • cd：更改当前目录
    • delete：删除文件
    • dir：列出当前目录中所有文件
    • path：显示搜索路径
    • pwd：显示当前目录
    • type：显示文件内容
    • what：列出当前目录下所有mat文件
    • wklread：读取.wk1电子表格文件
    • ...

    常见命令

    • clc：清除工作区
    • clear：清除变量
    • exist：检查文件或变量的存在
    • global：声明变量为全局变量
    • help：搜索帮助主题
    • lookfor：在帮助条目中搜索关键字
    • quit：停止matlab
    • who：查看所有变量
    • whos：知道所有变量的全部信息

    输入输出

    • disp：显示数组或字符串内容
    • fscanf：从文件中读取格式化的数据
    • format：控制屏幕显示格式
    • fprint：执行对屏幕或文件的格式化写入
    • input：显示提示并等待输入

    格式代码

    • %s：格式化为字符串
    • %d：格式化为整数
    • %f：格式化为浮点数
    • %e：格式化为指数形式
    • %g：格式化为指数形式或浮点数，即%f或%e
    • \n：换行
    • \t：制表符

    向量数组矩阵命令

    • cat：连接数组
    • find：查找非零元素索引
    • length：返回元素数
    • max：返回最大值
    • min：返回最小值
    • sort：对每列排序
    • sum：对每列求和
    • cross：计算矩阵叉积
    • dot：计算矩阵点积
    • det：计算行列式
    • inv：计算逆矩阵
    • pinv：计算伪逆矩阵
    • rank：计算矩阵秩
    • rref：计算行最简形矩阵
    • ...
