    正态分布、大数律

    多元正态分布

    设\(\mu = (\mu_1, \mu_2, \dots, \mu_n)\)为n维常数向量，B为\(n\times m\)常数矩阵，\(\epsilon = (\epsilon_1, \epsilon_2, \dots, \epsilon_n)\)为随机向量，其中每个元素均服从标准正态分布。若

    \[X = \mu + B\epsilon\]

    就称随机向量X服从n元正态分布，记作\(X \sim N(\mu, BB^T)\)

    性质：

    • 多元正态分布的边缘分布为一元正态分布
    • 若\(X\sim N(\mu, \Sigma),A为m\times n矩阵\),则\(Y = AX \sim N(A\mu, A\Sigma A^T)\)。特别的，对于向量\(\vec a\)，有\(\Sigma_{i=i}^n a_i X_i = \vec a^T X \sim N(\vec a^T \mu, \vec a^T \Sigma \vec a)\)
    • 设\(X \sim N(\mu,\Sigma)\)，当\(\Sigma\)正定时，X为连续型随机向量，有联合密度\(f(x) = \frac 1 {(\sqrt{2 \pi})^n\sqrt{det\Sigma}} exp(- \frac 1 2 (x - \mu)^T \Sigma^T (x - \mu))\)
    • 若\(\Sigma\)正定，分量相互独立的充要条件为\(\Sigma = diag(\sigma_1^2, \sigma_2^2, \dots, \sigma_n^2)\)
    • 分量独立且都服从标准正态分布的充要条件\(\mu = 0,\Sigma = I\)，即\(X \sim N(0,I)\)
    • 多元正态分布的标准化：\(若存在可逆矩阵A，使\Sigma = AA^T\)，则\(Y = A^{-1}(X - \mu) \sim N(0,I)\)

    正态分布可叠加性

    设多个随机变量\({X_i}\)均服从正态分布\(N(\mu_i, \sigma_i^2)\)，则

    \[X = b + \sum_{i=1}^n a_iX_i \sim N(b+\sum_{i=1}^n a_i\mu_i, \sum_{i=1}^n a_i^2\sigma_i^2)\]

    大数律

    切比雪夫不等式

    \(P(|X-E(X)|\geq \epsilon) \leq \frac {D(X)} {\epsilon^2}\)

    控制一个随机变量大于某个值的概率（一般取\(\epsilon\)值比较大）

    不管分布类型是否已知，只需要已知期望与方差，则可以对随机变量落入数学期望附近的概率进行估计。方差越小，落在平均值附近的概率越大

    随机序列

    称随机变量的序列为随机序列。对随机序列\(\{\xi_n\}\)，若对于任何\(\epsilon>0\)，有

    \[\lim_{n\to \infty} P(|\xi_n - \xi| \geq \epsilon) = 0\]

    则称\(\xi_n\)依概率收敛到\(\xi\)，记作\(\xi_n \stackrel{P}{\to} \xi\)。

    若\(P(\lim_{n\to \infty} \xi_n = \xi) = 1\)，就称\(\xi_n\)以概率1收敛到\(\xi\)，记作\(\xi_n \to \xi,wp1,或a.s.\)

    定理：若\(\xi_n \to \xi,wp1,\)，则\(\xi_n \stackrel{P}{\rightarrow} \xi\)

    依概率收敛和以概率1收敛的区别：

    • 依概率收敛：允许无限多的大偏差发生，但概率逐渐减小
    • 以概率1收敛：只允许有限多的大偏差发生

    例如\(X_n独立，且P(X_n = 1) = \frac 1 n, P(X_n = 0) = 1 - \frac 1 n\)，则\(X_n\)依概率收敛到0，但不以概率1收敛

    若\(X_n \stackrel{P}{\rightarrow} a, Y_n \stackrel{P}{\to} b\)，则在(a,b)附近连续的函数\(g(X_n,Y_n)\stackrel{P}{\to} g(a,b)\)

    切比雪夫大数定律

    随机变量\(\{X_i\}\)相互独立（可以弱化为互不相关），且数学期望和方差存在，且方差有共同上界，即存在\(D(X_i) \leq M\)，则对任意\(\epsilon > 0\)

    \(\lim_{n\to \infty} P(|\frac 1 n \sum X_i - \frac 1 n\sum EX_i | \leq \epsilon) = 1\)

    即

    \(\frac 1 n\sum X_i - \frac 1 n\sum EX_i \stackrel{P}{\to} 0\)

    伯努利大数定律

    设随机序列\(\{X_n\}\)服从独立同分布，若\(\mu = E(X_1)\)有限，则有

    \[\bar X_n \stackrel{P}{\to} \mu (n \to \infty)\\ 其中\bar X_n定义为\lim_{n\to \infty} \frac 1 n \sum_{i=1}^n X_i\]

    P表示依概率收敛

    或可以描述为伯努利大数定律：\(S_n\)为n重伯努利实验中事件A发生的概率，p是一次发生的概率，则

    \[\lim_{n\to\infty} P(|\frac {S_n} n - p| < \epsilon) = 1\]

    即\(\frac {S_n} n \stackrel{P}{\to} p\)

    伯努利大数定律给出了频率稳定性的严格数学表述

    辛钦大数定律

    设序列独立同分布，期望为\(\mu\)（对方差不做要求），则有

    \[\frac 1 n \sum X_i \stackrel{P}{\to} \mu\]

    直观理解

    在n次独立重复试验中，引入

    \[X_j = \begin{cases} 1,&当第j次实验成功\\ 0,&当第j次实验失败 \end{cases}\]

    则\(S_n = \sum_{j=1}^n X_j\)是第n次实验中成功的次数，则对成功的频率\(\bar X_n = \frac {S_n} n\)，有

    \[\lim_{n\to \infty} \bar X_n = E(X_1)\]

    强大数律

    对独立同分布随机序列：

    \[P(\lim_{n\to \infty}\bar X_n=E(X)) = 1\]

    强大数律比弱大数律更严格，因为：

    \(\xi_n \to \xi,wp1.,则\xi_n \stackrel{P}{\to} \xi\)

    区别

    在序列独立同分布条件下，强弱大数律条件一致。若非独立同分布，则弱大数律条件更弱。

    中心极限定理

    林德伯格-莱维中心极限定理（独立同分布的中心极限定理）

    设\(X_i\)为独立同分布的随机变量序列，且\(E(X_i) = \mu, Var(X_i) = \sigma^2>0\)，则对任意实数x，有

    \[\lim_{n\to P} \{P{\frac {\sum X_i - E(\sum X_i)} {\sqrt{D(\sum X_i)}}} \leq x\} = \lim_{n\to \infty} P\{\frac {\sum X_i - n\mu} {\sigma\sqrt{n}} \leq x\} = \Phi(x)\]

    若随机序列独立同分布，有共同的期望\(\mu\)和方差\(\sigma^2\)。部分和\(S_n = \Sigma_{j=1}^n X_j\)，则\(S_n\)的标准化\(\xi_n = \frac {S_n - n\mu} {\sqrt {n\sigma^2}} = \frac {\bar X_n - \mu} {\sigma/\sqrt n}\)依分布收敛到标准正态分布，即对任何x，\(\lim_{n\to \infty}P(\xi_n \leq x) = \Phi(x)\)（正态分布分布函数）

    • 依分布收敛记为\(\xi_n \stackrel{d}{\to} N(0,1)\)（此处表示依概率收敛到标准正态分布）
    • \(S_n \stackrel{d}{\to} N(n\mu, n\sigma^2)\)
    • 可衡量收敛速度：对标准化后的\(S_n\)，收敛速度为\(O(\frac 1 {\sqrt n})\)。而大数律只能通过切比雪夫不等式，说明方差有限时，概率衰减速度为\(O(\frac 1 n)\)

    推论：

    $$P{a\leq \sum X_i \leq b} = P {\frac {a - n\mu} {\sqrt {n\sigma^2}} \leq \frac {\sum X_i - n\mu} {\sigma \sqrt{n}} \leq \frac {b - n\mu} {\sqrt {n\sigma^2}}}\

    \approx \Phi(\frac {b - n\mu} {\sqrt {n\sigma^2}}) - \Phi(\frac {a - n\mu} {\sqrt {n\sigma^2}})$$

    连续型矫正

    \(P(a\leq S_n \leq b) \approx P(a-0.5 \leq X_n \leq b-0.5)\)

    其中\(S_n\)为离散整数值，\(X_n\)为连续值

    棣莫弗-拉普拉斯中心极限定理

    n重伯努利实验中，A每次发生概率为p，\(Y_n\)为n次中A发生的次数，则对于x，有

    \[\lim_{n\to \infty} P\{\frac {Y_n - np} {\sqrt{np(1-p)}}\leq x\} = \Phi(x)\]

    即\(Y_n \stackrel{d}{\to} N(np, np(1-p))\)

    该定理可看做独立同分布的中心极限定理的一种特殊情况

    李雅普诺夫中心极限定理

    设\(X_i\)为相互独立的随机变量序列，他们的期望与方差都存在，记\(B_n = \sqrt{\sum \sigma_k^2}\)，若存在正数\(\delta\)，使得\(\lim_{n\to \infty} \sum \frac 1 {B_n^{2+\delta}}E\{|X_k - \mu_k|^{2+\delta}\}\)，则对任意正数x，\(\frac {\sum X_i - n\mu} {B_n} \stackrel{d}{\to} N(0,1)\)
