    有限自动机

    字符串与语言

    字母表是任意一个有限集。符号为字母表中的元素。字符串是字母表中符号组成的有限序列，俗称单词。空串用\(\epsilon\)表示。串的长度，用\(|\cdot|\)表示。用\((A)*(B)\)表示字符串A与B进行链接。串的反转记为\(A^R\)。语言指给字母表上一些字符串的集合（可以有限可以无限）。

    有限自动机是一个五元组\((Q,\Sigma,\delta,s,F)\)

    • Q是有限集，称为状态集
    • \(\Sigma\)为有限集，称为字母表
    • \(\delta: Q\times\Sigma\rightarrow Q\)是转移函数
    • \(s\in Q\)为起始状态
    • \(F\subset Q\)是接受状态集

    状态图、转移表等价于形式定义。注：确定型有限自动机，输入符号后跳转状态唯一。有限自动机的读写头不能改写，只能右移。输入读完处于接受状态则接受，否则拒绝

    设M是一个DFA，\(w=w_1\dots w_n\)是字母表上的一个字符串。若存在Q中的状态序列\(r_0,\dots,r_n\)，满足：

    • \(r_0 = s\)
    • $ \delta(r_i,w_{i+1}) = r_{i+1}$
    • \(r_{n} \in F\)

    则M接受w。对有限自动机M，若\(A=\{w\in \Sigma^*|\text{M接受w}\}\)，则称A是M的语言，记为\(L(M) = A\)，也称M识别A。M的语言唯一

    若存在DFA识别语言A，则称A为正则语言。两个有限自动机等价，则他们识别的语言相同（等价条件）

    正则语言

    正则语言的并、交是正则语言

    正则语言运算（运算后仍然是正则语言）：

    • 并\(A\cup B\)
    • 连接\(A\circ B = \{x_1x_2 |x_1\in A, x_2\in B\}\)
    • 星号\(A*=\{x_1 x_2 ... x_n | x_i\in A, n\geq 0\}\)

    非确定型有限自动机NFA

    每步可以0到多种方式进入下一步。转移箭头可以是空串

    与确定型的区别：

    • \(\delta: Q\times \Sigma_\epsilon \mapsto P(Q)\)

    NFA和DFA等价，NFA的确定化：

    • 从NFA N的起始状态S出发经过任意条弧所能到达的状态组成集合作为确定化后的DFA M的起始状态S'
    • 从S'触发，经过任意输入符号的状态转移能到达的状态所组成的集合作为M的新状态
    • ...

    正则语言性质与正则表达式

    正则语言对并、连接、星号封闭。可通过画NFA来证明

    若R满足：

    • \(a, a\in \Sigma\)
    • \(\epsilon\)
    • \(\Phi\)
    • \((R_1 \cup R_2)\)，两个都是正则表达式
    • \((R_1 \circ R_2)\)，两个都是正则表达式
    • \((R_1^*)\)，\(R_1\)是正则表达式

    则称R为一个正则表达式。每一个正则表达式R表示一个语言，记为\(L(R)\)

    定理：语言A正则，等价于A可用正则表达式描述

    由A有正则表达式证A正则显然

    由A正则证A有正则表达式：

    构造广义非确定有限自动机GNFA。GNFA：

    • 为非确定有限自动机
    • 转移箭头可以用任何正则表达式作标号

    证明中的特殊要求：起始状态无射入箭头，有唯一的接受状态（无射出箭头）

    手段：一个一个去掉中间状态

    从NFA转换为正则表达式（反向变换可以反向得到NFA）：

    对于：

    flowchart LR
A --$$e_1$$--> B --$$e_2$$--> C

    替换为：

    flowchart LR
A --$$e_1e_2$$-->C

    对于：

    flowchart LR
A --$$e_1$$--> B
A --$$e_2$$--> B

    替换为

    flowchart LR
A --$$e_1|e_2$$--> B

    对于

    flowchart LR
A --$$e_1$$--> B --$$e_3$$--> C
B --$$e_2$$--> B

    替换为

    flowchart LR
A --$$e_1e_2^* e_3$$--> C

    也可以通过删除一个中间状态进行。设\(q_{rip}\)为待删中间状态，则对任意两个状态\(q_i,q_j\)都需要修改箭头标号。将：

    flowchart LR
A(($$q_i$$)) --$$R_4$$--> B(($$q_j$$))
A --$$R_1$$--> C(("$$q_{rip}$$"))
C --$$R_3$$-->B
C --$$R_2$$--> C

    变为：

    ```mermaid flowchart LR A((\(\(q_i\)\))) --\(\(((R_1)(R_2^*)(R_3)) \cup R_4\)\)--> B((\(\(q_j\)\)))

    非正则语言与泵引理

    设A为正则语言，则存在\(p>0\)，使得对任意\(w\in A, |w|\geq p\)，存在分割\(w=xyz\)满足：

    • 对任意\(i\geq 0, xy^i z\in A\)
    • \(|y|>0\)
    • \(|xy|\leq p\)

    可用鸽巢原理证明，读前p个符号，里面必然可以产生重复状态（即包含环，即产生y的部分）

    等价描述：若\(\forall p > 0\)，\(\exist w\in A(|w| \geq p), \forall x,y,z(|y| > 0, |xy| \leq p, w=xyz), \exist i\geq -, xy^i z \notin A\)，则A是非正则语言

    例如可用于证明\(B=\{0^n 1^n|n\geq0\}\)非正则：令\(w=0^p 1^p\)，对任意分割\(w=xyz, |y|>0, |xy|\leq p\)，令\(i=0\)，\(xz = 0^{p-|y|} 1^p \notin B\)，所以B非正则语言

    \(C=\{ww|w\in \{0,1\}^*\}\)非正则
