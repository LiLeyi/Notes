    环面系统

    Hamilton系统

    n个自由度的动力学系统可以用n个广义坐标和n个广义动量描述其运动状态，全体(q_i,p_i)构成2n维相空间。不受非有势力作用时，建立形式

    $ \dot q_i = \frac{\partial H}{\partial p_i},\dot p_i = -\frac{\partial H}{\partial q_i} $

    的微分方程，称为Hamilton正则方程。H=H(q,p,t)称为系统的Hamilton函数。对于力学系统，H为系统的能量（势能+动能）。用正则方程描述的动力学系统称为Hamilton系统

    若H不显含时间，则H在运动过程中不随t变化，H为守恒量。这类系统称为保守系统

    若H显含时间，将原来的2n维相空间增广为2n+1维相空间\((\bar q,\bar p)\)，其第n+1个广义坐标为\(\bar q_{n+1}\)第n=1个广义动量为\(\bar p_{n+1}\)，增广后的Hamilton函数为(看ppt!!!)

    所有系统都可以转化为保守系统

    耗散系统因存在能量散失，所以可能有吸引子存在，相空间所有轨线都会收缩到吸引子，初始一定大小的相空间在运动中逐渐减小，时间足够大时趋近于0

    保守系统里不存在吸引域，不会有吸引子，相空间是守恒的，相轨线是保体积的

    微元可能在流或映射的作用下被拉伸、压缩或折叠，呈现出异常复杂的形状。微元内相邻两轨道，在经历有限的时间会在有些方向上迅速分离，丧失相似性，彼此不再靠近，使整个系统的运动呈现高度的复杂性

    正则变换

    引入变量(Q,P)，Q=(q,p)，P=(q,p)，微分方程也相应变化为\(\dot Q=\dot Q(Q,P)，\dot P=\dot P(Q,P)\)。若变换后的方程仍然存在正则方程的结构，则此变换称为正则变换。正则变换的逆变换、复合变换也是正则变换

    若能构造正则变换将(q,p)变为\((I,\theta)\)，使新变量只依赖于I，与\(\theta\)无关，则H=H(I)，相应Hamilton方程为

    \(\dot I_i = \frac{\partial H}{\partial \theta_i} = 0,\dot \theta_i = -\frac{\partial H}{\partial I_i} = \Omega_i(I_1,I_2,...,I_n)\)

    积分得到

    $ I_i(t)=I_i(0)（可知I是常数，即n个维度上为常数，由能量守恒约束）,\theta_i(t)=\Omega_i(I_1(t),...,I_n(t))t + \theta_i(0) $

    其中2n个由初始条件确定的常数I(0)和\(\theta(0)\)可以由(q(0),p(0))得到，\((I,\theta)\)称为作用-角度变量

    由上式知，对于给定初值，系统的运动由n个角度坐标\(\theta_i\)唯一确定，这样的系统称为可积Hamilton系统

    同胚于环面

    对于n维可积Hamilton系统，存在n个彼此独立的孤立运动积分\(I_i(q,p)=C_i,i=1,2...n，其中C_i为常数。函数彼此独立指相应微分dI_i彼此线性无关\)。由于有n个孤立运动积分的存在，系统在2n维相空间中的运动限制在一个与n维环面同胚的n维流形上（2n维相空间，但由于约束的存在，相轨迹只在一个n维的空间内）

    由于其上出发的相轨迹始终留在该流形内，这些流形称为不变环面

    系统用作用-角度变量表示时，作用量I_i给出的n维环面的n个半径，角度变量\(\theta_i\)是在环面上的n个坐标

    例如：n=1时，周期运动被限制在\(R^2\)上的一维环面（一个闭曲线）上。n=2时，周期运动被限制在\(R^4\)上的二维环面上，有两个角速度，半径由两个作用量确定

    只有可积Hamilton系统同胚于环面

    无理、有理环面

    若\(\Omega_i\)非有理通约，即只有所有\(k_i\in Z\)同时为0使\(\Sigma k_i \Omega_i=0\)（理解为两个周期，只要有一个为无理数，则两个周期复合后周期为无穷），则相轨迹在n维环面上不闭合地无穷环绕。此时相轨迹在环面上是稠密的，系统的运动是准周期运动（或拟周期运动），这时的环面称为无理环面。

    若有理通约，则相轨迹在n维环面上闭合，非稠密，系统的运动时周期运动，这时的环面称为有理环面

    无理环面的Poincare截面有无限个点，有理环面的Poincare截面只有有限个点

    例：

    \(R^3中的T^2环面\)：对于n=1的情况，可以构造增广系统，将时间扩充为\(R^3\)相空间中的第三个坐标轴，动力学方程为\(\dot q = \frac {\partial H}{\partial p},...,\dot t = 1\)。这种构造也适用于非自治系统，将其变为自治系统

    例如简谐振子\(\ddot x + \omega^2 x = 0\)，解为\(x=Acos(\omega t + \phi),\dot x = -\omega Asin(\omega t + \phi)\)，画出其轨道，由于\(H=H(x,\dot x)\)守恒，这些轨道被限制在等能柱面上

    因此，可积Hamilton系统只能是周期或准周期的。但任取一个二自由度以上的Hamilton系统，它一定是不可积的。可积的Hamilton很少。可积Hamilton系统的相轨迹为缠绕在环面上的周期或准周期轨道，不可积Hamilton系统一般没有这个性质

    等能面

    保守系统总存在能量积分，因此给定初始能量后，2n维系统相空间被约束在2n-1为的等能面上。换句话说，系统的运动可以出现在2n-1维的等能面任意位置（统计物理学：系统在等能面上各处的分布概率相等）。是否可积都有等能面

    例：非线性单摆是哈密顿系统，相空间为二维，其能量守恒，则其相轨迹为一维的（闭合曲线）

    等能面和不变环面的关系

    等能面为2n-1维，不变环面为n维。存在以下三种情况

    • n=1，等能面维数等于不变环面维数
    • n=2，等能面维数比不变环面大1，此时等能面可以被不变环面包围并分割为互不联通的两个区域
    • n=3，等能面维数比不变环面的维数高两维以上，此时等能面无法被不变环面包围并分割成互不联通的两个区域

    n大于等于2时，可积Hamilton系统的运动并没有分布在整个等能面上，而是只能分布在n维的环面上，而这个n维环面只是等能面的一部分

    近可积Hamilton系统

    由可积Hamilton系统附加微小扰动形成的不可积系统称为近可积系统。近可积系统的Hamilton可写为

    \(H(I,\theta)=H_0(I)+\epsilon V(I,\theta,t)，H_0为未扰可积Hamilton函数，\epsilon为无量纲参数，且远小于1\)

    若：

    1.微扰很小，即\(\epsilon\)远小于1 2.可积系统\(H_0(I)\)的Jacobi行列式不等于0，即\(|\frac {\partial \Omega}{\partial I}|\neq 0\)

    则不可积系统的运动仍然限制在n维环面上。条件II保证各运动频率间不相关，即可积系统为无理环面，即各频率运动不会产生共振，适用KAM定理

    KAM定理

    对无理环面

    该定理保证扰动很小时，大多无理环面不会消失，只会发生微小变形。在受扰相空间内仍存在不变的n维环面，环面上的运动仍然是准周期的，此环面称为KAM环面

    对于有理环面：若可积系统n个频率不满足第二条，即扰动频率与系统中某个运动的频率发生非线性共振，有理环面无法维持，会发生破裂。即一个很小的扰动就可导致发生重大改变。因为当发生非线性共振时，一级近似解中分式的分母等于0，得到发散的结果，这就是著名的小分母发散问题

    例如

    $ \ddot x + kx = f e^{i\omega t}， 解得x=Ye^{i \omega t}，Y= \frac f {- \omega ^ 2 + k} $

    当分子趋近于0时振幅会非常大

    研究2自由度的情况，首先考虑未受扰的可积Hamilton系统，其运动被限制在2维的环面上，取为\(\theta_2\)为常数的Poincare截面，则轨线与该截面的交点在\(I_1\)为常数的圆上。则轨线相邻两次穿越截面的时间间隔是\(\Delta t = \frac {2\pi} {\omega_2}\)，每次的该变量为\(\omega_1 \Delta t = 2 \pi \omega_1 / \omega_2\)。Poincare截面上的点运动为一个二维映射，称为扭转映射

    $ T_0{\theta_1,I_1}^T={\theta_1+2n\pi ,I_1}^T $

    对有理环面

    对于有理环面，两\(\omega\)比为有理数，则存在不动点，\(\Gamma_0 = T_0^m(\Gamma_0)\)，所有不动点组成了不动环面。此Poincare截面中，该不动环面上的扭转映射为等同映射，该不动环面将平面分为两侧，两侧扭转方向不同

    对应近可积的Hamilton系统

    $ T_\epsilon {\theta_1,I_1}^T={\theta_1+2n\pi \omega_1 / \omega_2 + \epsilon f_1(\theta_1,I_1),I_1+f_2(\theta_1,I_1)}^T $

    原可积系统中间存在一个面为有理环面，扰动后不存在有理。在原来逆时针与顺时针轨道中间连线，则连线上必有一点不发生旋转，所有这样的不旋转点组成一个保角度环面，此面不是原来不变环面扰动后形成

    由于\(T_\epsilon\)保体积，所以在\(\Gamma_\epsilon与T_\epsilon^m(\Gamma_\epsilon)\)体积相同，则两者必有交点，个数为偶数，这些交点为不动点，不旋转也不移动（类比驻波的不动点）

    观察这些不动点的轨迹走向，可分为椭圆点和双曲点。

    • 椭圆点周围发生旋转运动
    • 双曲点周围存在类似鞍面的运动

    原有不变环面加上扰动后，除了少数不动点外，时时在变化，即发生破裂：Poincare-Brikhoff定理

    对于新产生的椭圆点(Poincare截面上的中心型不动点)，周围是一层层闭曲线，每个闭曲线又分为无理环面和有理环面，其中有理环面在进一步扰动下，继续破裂，形成分形结构

    扰动对双曲点影响更复杂。每个双曲点上都有两类不变曲线，分别为稳定流形（收缩方向）和不稳定流形（发散方向），分别记为\(w^s,w^u。w^s,w^u不能与自己相交，否则违反唯一性的条件，但两类可以互相交叉\)。

    同时是稳定流形和不稳定流形归宿点称为同宿点，两头归宿与同一点的轨道称为同宿轨道

    一个定点的稳定轨道与另一个定点的不稳定轨道相连，这样连接的轨道称为异宿轨道。用异宿轨道连接的两个定点称为异宿点

    设a为某一稳定流形和不稳定流形的异宿点，显然在同一映射下，a的后继点也在稳定和不稳定流形上，这意味着只要稳定和不稳定流形发生一次相交，则必相交无数次（纠缠）

    环面破裂后双曲点和椭圆点交错排列，形成复杂结构，显示出较强随机性

    Hamilton系统的混沌

    扰动产生

    若扰动足够小，KAM环面仍可以保持，有理环面发生破裂。KAM环面将环面上规则与非规则的运动区域隔开。整个环面上共存看规则与非规则运动

    扰动增强时，不仅围绕着椭圆点的环面逐渐破裂，原来受扭曲的KAM环面（无理环面）也会被叠压在一起，最终可能会破裂，整个Poincare截面上结构呈现出高度随机性，等能面上流的走向异常杂乱，这种现象也被称为Hamilton系统的混沌现象

    Arnold扩散

    即使KAM环面不破裂，这种高随机性的运动仍然可能出现。因为大于二维的系统，KAM环面是n维，等能面是2n-1维，此时n维环面不足以分隔等能面，使不稳定的轨道有可能在等能面内，在各个KAM环面间自由的穿行，从而弥散到整个等能面内。运动在除KAM面以外的地方，呈现出混沌形态
