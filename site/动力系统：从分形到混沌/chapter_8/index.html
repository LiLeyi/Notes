    混沌的控制与同步

    目标

    人们总希望混沌系统能在特定的微小扰动下被引导到稳定的有序状态或期望的混沌状态。因此，如何在多种多样的非线性系统中，按照人们的意愿，获取所需的各种周期态或非周期态，并能实现其稳定的有效控制，是人们控制混沌索要实现的目标

    大体目标：

    • 对能产生实际需要的具有特定性质的混沌运动加以控制，使之产生实际需要的各种输出，以达到实现对各种周期态的转换和应用的目的
    • 通过合适的策略及方法抑制对实际系统有害的混沌运动，或将其消除

    具体目标：

    • 稳定控制系统中的某些平衡点
    • 有效的控制混沌稳定的或不稳定的周期态
    • 控制混沌吸引子中所需的非周期轨道
    • 控制系统在不同周期态，或周期态或混沌态之间的转变
    • 设置一定的条件，产生新的混沌态，促使系统产生有序的时空结构
    • 抑制或消除有害的混沌态
    • 实现多个混沌系统的同步

    按控制原理分类

    微扰反馈控制

    以原系统的固有状态为控制的目标状态，利用与时间有关的连续小扰动作为控制信号，将系统自身动力学变量的部分数据反馈回控制信号中，当微扰趋于0或很小时，则实现对特定所需的周期轨道或非周期轨道的稳定控制

    该方法的核心是目标轨道的局域稳定性

    • 参数微扰控制法OGY
    • 延迟反馈控制法DFC
    • 偶然正比反馈法OPF
    • 外力反馈控制法
    • 正比脉冲控制法

    无反馈控制

    该方法与一些特定的轨道无关，其控制信号不受系统变量实际变化的影响，完全避免对系统变量数据的持续采集和响应。当系统达到控制时，控制着的输入信号并不趋于0，并且受控后的动力学行为可能与原系统完全不同，即产生了新的动力学行为

    特点：实现控制后控制信号都保持非零值

    • 自适应控制法
    • 混沌信号同步法
    • 神经网络法
    • 人工智能法
    • 相位调节法
    • 参数共振法
    • 混沌信号同步法

    OGY方法

    基本思想：先确定吸引子中嵌入的各种不稳定周期运动（不稳定周期轨道，即不会吸引其他轨线），选择其中之一作为控制目标，等待混沌运动游荡到该周期运动附近时（因为已经进入混沌状态，一定会有时刻运动到其附近），对系统某一控制参数进行小扰动，将混沌运动稳定在该周期运动上

    步骤

    确定控制目标

    借助Poincare截面，可用n维Poincare映射来描述所关心的有限维动力系统

    \(u_{k+1}=P(u_k,p_k),u_k\in R^n,p_k\in R,k = 1,2,...，其中p_k为控制参数\)

    设控制参数\(p_k=\bar p\)时该映射呈现混沌状态，其混沌吸引子中稠密地嵌有任意周期的鞍型不动点，即原系统的不稳定周期运动。则需要选择其中某一不动点\(u_F(\bar p)\)为控制目标，且不妨假设其仅有一维不稳定流形\(W^u(u_F(\bar p))\)（即可以理解为一个方向不稳定，多个原理类似）

    等待移动至目标附近

    在不动点\(u_F(\bar p)\)处指定一邻域，以便对进入该邻域的映射点实施控制

    施加扰动

    一旦映射点\(u_k\)游荡到指定邻域内，就可以考虑对控制参数\(p_k\)的摄动（即修改参数）。将\(u_k\)稳定到\(u_F(\bar p)\)上。方法之一是通过改变参数p_k，使映射点\(u_{k+1}\)落到不动点\(u_F(\bar p)\)的稳定流形\(W^s(u_F(\bar p))\)上，从而使后继映射点沿着\(W^s(u_F(\bar p))\)趋于\(u_F(\bar p)\)

    例如二维时，原点为不稳定平衡点，x轴方向不稳定，y轴方向稳定，即有一个稳定一维流形和一个不稳定一维流形。在原点附近存在一个小邻域，运动到其内时，使其运动到稳定流形，即y轴上，实现控制

    为确定参数摄动量\(p_k - \bar p\)，将映射在不动点\(u_F(\bar p)和参数\bar p\)附近线性展开

    \(u_{k+1}=P(u_k,p_k)=u_F(\bar p) + A(u_k - u_F(\bar p)) + b(p_k - \bar p)\)

    其中\(A\in R^{n\times n},b\in R^n分别是映射在(u_F(\bar p),\bar p)处关于u_k的Jacobi矩阵和关于p_k的偏导数向量。A=\frac {\partial u_{k+1}}{\partial u_k},b=\frac {\partial u_{k+1}}{\partial p_k}\)。此处为泰勒展开，取一阶项。解A的特征值，因为不动点具有一维不稳定流形，则设其特征值\(\lambda_r\)满足\(\lambda_r > 1\)，相应左特征向量为\(\phi_r\)，该向量自然与其他特征值的右特征向量正交（一般的特征向量均指右特征向量），而这些右特征向量是稳定流形\(W^s(u_F(\bar p))\)的切向量，张成稳定流形的切空间。因此只要另

    \(\phi_r^T(u_{k+1}-u_F(\bar p))=0\)

    即下一次运动和周期运动轨道的差向量和\(\phi_r\)的内积为0，即垂直，即可使\(u_{k+1}\)落在稳定流形中

    \(由A,\bar p\)推导出\(p_k\):

    \(u_{k+1} - u_F(\bar p)= A(u_k - u_F(\bar p)) + b(p_k - \bar p)\)，带入\(\phi_r^T(u_{k+1}-u_F(\bar p))=0\)

    \(\phi_r^T A(u_k - u_F(\bar p)) + \phi_r^T b(p_k - \bar p)=0\)

    结合\(\phi_r\)为左特征向量，一般写向量为列向量，而左特征向量为行向量，即\(\phi_r^T\)为行向量，即有\(\phi_r^T A = \lambda_r\phi_r^T\)，可得:

    \(p_k=\bar p + \frac {-\lambda_r\phi_r^T[u_k-u_F(\bar p)]} {\phi_r^Tb}\)

    经过几次重复映射，映射点最终会收敛

    若有多个变量需要控制，则进入邻域后根据一个参数调整一个参数即可。例如有a，b两个参数，则可以根据a，调整a；根据a，调整b...即令p为被调整参数即可

    分析

    推导包含泰勒展开，保留一阶项，所以微扰\(p_k - \bar p\)要很小，总结：一旦系统状态位于目标轨道附近，就将其向稳定方向拉一下

    注意：

    • 可选择混沌吸引子中嵌入的任一周期作为控制目标
    • 所需摄动量要很小
    • 可通过实验数据来拟合线性化映射，无需原系统的数学模型
    • 最终扰动趋近于0，耗能小，适用于长期型稳定
    • 位于吸引盆边缘的点不宜作为控制目标，因为运动到吸引盆边缘的概率较小

    OGY方法的推广和变形统称镇定控制

    总结

    • 有混沌吸引子
    • 镇定控制理论上是针对映射提出的，但借助Poincare映射也可控制连续动力学系统的混沌
    • 要求已知，或可从数据中构造出映射或Poincare映射，控制时对精度要求不高

    自适应控制

    系统自身不断地识别被控的状态、性能或参量，从而认识或掌握系统当前的运行指标与期望并加以比较，进而做出决策，来改变控制器的结构、参数或根据自适应规律来改变控制作用，保证在其所期望指标下的最优或次优状态。依照这样的思想所建立的控制系统，称为自适应（控制）系统

    这种调节通过调节目前状态与目标状态的差值来实现

    步骤

    考虑一离散映射，在n时输出\(y_n\)，目标输出位移为\(d_n\)，\(e_n\)为实际输出与目标输出\(d_n\)之间的差。一种简单地控制策略为：指将差信号与系统的某个控制参量联系起来，构成下列自适应系统

    $ \begin{cases} y_{n+1}=F(y_n,P_n)\ P_{n+1}=P_n-k_{x_i} G(e_{x_in})\ e_{n+1}=y_{n+1}-d_{n+1} \end{cases} $

    \(k_{x_i}\)为自适应控制机在坐标\(x_i\)方向上控制的强度，G仅仅是\(e_{x_in}\)的函数，\(P\in R^1\)为可调参数

    考虑二维系统，不动点为

    $ x_F = F_1(x_F,y_F,P_0)\ y_F = F_2(x_F,y_F,P_0)\ 令P_{n+1} = P_n - k_x e_x - k_y e_y，带入进行泰勒展开，保留一阶项，可得\ (x_{n+1}-x_F,y_{n+1}-y_F)^T=\bar A (x_n - x_F,y_n - y_F)^T $

    (看ppt!!!) 所以要求\(\bar A\)为收缩的，即特征值绝对值均小于1，解出的自适应\(k_x,k_y\)应在一个范围内

    分析

    推导包含线性化，要求点到不动点距离不宜过大

    选择合适的\(k_i和邻域\)，可使系统以不同速率趋向于不动点

    混沌同步

    同步指两个振动系统相位间的协调一致现象。L.M.Pecora与P.L.Carroll相同的混沌子系统间，在不同的初始条件下，通过某种驱动（耦合），仍然可以实现混沌轨道的同步化。提出一种混沌同步方法简称P-C方法。

    分类

    • 恒等同步
    • 广义同步
    • 相位同步
    • 频率同步
    • 阵发同步
    • 爆炸式同步

    其余：几乎同步、完全同步、一致同步、不完全相同、滞后同步、相同步、广义同步、间歇滞后同步
