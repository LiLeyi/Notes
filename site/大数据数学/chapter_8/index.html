    奇异值分解

    介绍

    将任意实矩阵分解为若干个的乘积

    给定\(m\times n\)矩阵A，将其分解为三个矩阵之积\(A = U S V^T\)，其中：

    • \(U\)为$m\times n $正交矩阵
    • V是\(n\times n\)正交矩阵
    • S是\(m\times n\)矩形对角阵，且所有元素非负（不防设S的对角元素从高到底排列。如果要调换位置，那么U、V的列向量顺序也会相应变化，结果不变）

    U的各个列向量称为左奇异向量，V的各个列向量称为右奇异向量，S中元素称为奇异值（可以为0）

    存在性

    不妨设\(m\geq n\)

    设\(n\times n\)对称矩阵\(A^T A\)可以分解为\(QDQ^T\)

    若\(A=USV^T\)，则有\(A^T A = VS^T U^T USV^T = VS^TSV^T\)，所以可以取V为Q，\(S^TS\)为D。得到\(V,S\)

    设\(n\times n\)对称矩阵\(A^T A\)可以分解为\(\hat Q\hat D\hat Q^T\)

    则\(AA^T = USV^T V S^T U^T = USS^T U^T\)，取\(U=\hat Q\)，\(D = AA^T\)，则可以得到U.

    令S对角线元素为\(\sqrt {\lambda_1}, \dots, \sqrt{\lambda_n}\)（非负），即根号下\(A^TA\)的特征值。由\(A=USV^T\)可得\(AV = US = [\sqrt{\lambda_1} \vec u_1, \dots, \sqrt{\lambda_n} \vec u_n]\)，即有\(A\vec v_1 = \sqrt {\lambda_i} \vec u_i\)

    同样，令\(v_1,\dots, v_n\)分别为\(A^T A\)的标准正交特征向量，同样令S的对角线为\(\sqrt{\lambda_i}\)，取\(u_i = A\vec v_i /\sqrt{\lambda_i}\)，则各个u之间正交。若U是\(m\times m\)矩阵，即m个列向量，则剩下的\(u_{n+1}, \dots, u_m\)该如何确定？只需要使得U正交即可，具体取值不被A的取值影响（因为中间对角矩阵最下面几行为0）

    计算方法

    1. 计算\(A^TA\)。\(O(mn^2)\)
    2. 对\(A^TA\)进行特征分解，\(A^TA = QDQ^T\)，取V=Q。\(O(n^3)\)
    3. 计算AV。\(O(mn^2)\)
    4. 取\(s_i = ||Av_i||\)，且\(u_i = \frac {Av_i} {||Av_i||}\)。\(O(mn)\)
    5. 补齐正交矩阵

    \(k_1\times k_2\)的矩阵乘\(k_2\times k_3\)矩阵的复杂度为\(O(k_1k_2k_3)\)。复杂度为\(O(mn^2)\)，对于实际应用场景太大，实际中采用更快的方法计算

    等价形式

    \(A=USV^T\)等价于\(A = \sum_{i=1}^{min\{m,n\}}s_i \vec u_i \vec v_i^T\)，其中\(s_i\)为第i个奇异值，\(v_i^T\)是\(V^T\)的第i个行向量。\(u_iv_i^T\)为秩为1的\(m\times n\)矩阵

    奇异值分解将\(m\times n\)矩阵A展成\(min\{m,n\}\)个秩为1的矩阵的线性组合

    证明：从\(A=USV^T\)出发，写出每一个元素的表达式进行分析

    先算\(US\)，得到\(m\times n\)矩阵，第k列为\(s_k u_k\)（一共n列）

    再算\(USV^T\)，得到第i行第j列为\(\sum_{k=1}^n s_k(u_k\text{第i元素}) (v_k\text{第j元素})\)，与等价形式一致

    则等价定义奇异值分解，若恰好有\(r\leq n\)个非零特征值，则\(A=\sum_{i=1}^r s_i \vec u_i \vec v_i^T\)

    说明只需要关注U、V的前r列，其余列为凑正交矩阵所需。则有：给定秩为r的\(m\times n\)矩阵A，将其分解为三个矩阵之积\(A=USV^T\)，其中

    • U是\(m\times r\)矩阵，列正交
    • V是\(n\times r\)矩阵，列正交
    • S为\(r\times r\)对角矩阵，且所有对角线上元素为正

    即裁掉了无关向量

    几何意义：

    • \(V^T x\)旋转，把向量x在空间里转一下，得到一个新方向的向量
    • S沿标准轴拉伸
    • U旋转，把拉伸完的向量，在空间里再转一下，得到最终输出向量

    向量第一次旋转后，更偏向\(u_1\)的方向（因为S的第一个元素最大）

    低秩矩阵近似

    问题：低秩矩阵近似，即给定矩阵A，正整数k，求矩阵B，且\(rank(B) \leq k\)，满足：

    \[min_{B\in \mathbb R^{m\times n}} ||A-B||_F\]

    其中F范数定义为\(||A||_F = \sqrt {\sum_{i=1}^m \sum_{j=1}^n A_{ij}^2}\)

    任意矩阵B可以写为\(m\times rank(B)\)矩阵与\(rank(B)\times n\)矩阵相乘的形式。若找到这个矩阵B，则只需要记录\((m+n)k\)个数字而不是mn个数字。

    回顾奇异值分解等价形式：

    \(A = \sum_{i=1}^{min\{m,n\}} s_i \vec u_i \vec v_i^T\)，则只需保留前 k项即可。易得满足约束条件，后面证满足最优性。此时目标函数为：

    \[||\sum_{i=k+1}^{min\{m,n\}} s_i \vec u_i \vec v_i^T||_F\]

    用范数的性质：\(||A||_F = \sqrt{\sum\sum A_{ij}^2} = \sqrt{tr(A^TA)} = \sqrt{\sum_{i=1}^{min\{m,n\}} s_i^2}\)，则得误差为：

    \[\sqrt{\sum_{i=k+1}^{min\{m,n\}} s_i^2}\]

    证明为最优：

    • 引理：若Q为正交矩阵，\(||QA||_F = ||A||_F\)。\(||A||_F =\sqrt{tr(A^TA)} = \sqrt{tr(A^TQ^TQA)} = ||QA||_F\)
    • 则目标函数可改写\(||A-B||_F = ||USV^T - B||_F = ||S - U^T BV||_F\)，其中两个矩阵现在都变为了方阵
    • 因为S是对角矩阵，则对最优解B，\(U^TBV\)必为对角矩阵（反证，若\(U^TBV\)非对角阵，可找到同秩对角阵，更小化目标函数）
    • 取\(B=UDV^T\)，则目标函数变为\(||S-D||_F\)，易证k秩约束下的最优D

    低秩近似步骤：

    • 对A做奇异值分解
    • 计算矩阵\(B = \sum_{i=1}^k s_i u_i v_i^T\)

    选择k：令误差小于一定阈值

    主成分分析

    计算主成分

    若可以快速算出奇异值分解，如何快速得到主成分

    \(X=USV^T\)，\(X^TX = VS^TU^UUSV^T = VS^TSV^T\)，则V的列向量是\(X^TX\)的单位特征向量，取前k个列向量可得到前k个主成分方向。\(US\)为主成分得分矩阵

    适用于得到比较多的主成分

    解决低秩矩阵近似问题对比

    SVD解决

    \[\min_{B\in\mathbb R^{m\times n}} ||X - B||_F, s.t. rank(B) \leq k\]

    最优解：\(B^* = \sum_{i=1}^ks_i u_i v_i^T\)

    PCA:

    \[\min_{V\in\mathbb R^{n\times k}} || X - XVV^T ||_F, s.t.\text{V的列向量构成标准正交向量组}\]

    最优解设为\(V^*\)

    有关系\(B^* = XV^* (V^*)^T\)。证明方法：

    • 法一：取\(V^*\)为\(USV^T\)中V的前k列，将X展成\(USV^T\)，证\(XV^* (V^*)^T\)最多有k个非零奇异值
    • 法二利用\(\vec u_i = X\vec v_i / s_i\)化简\(\sum_{i=1}^k X \vec v_i \vec v_i^T\)，然后和\(XV^*(V^*)^T\)对比

    矩阵补全

    对矩阵A仅仅知道部分元素值，如何估测其余元素取值。例如估计观众对电影评分，行为观众，列为电影

    矩阵秩为1时，观众对所有电影喜好一致，评分全部都线性相关

    低秩矩阵补全。给定\(m\times n\)矩阵A及正整数k，解决：

    \[\min_{C\in \mathbb R^{m\times n}} \sum_{observed(ij)} (A_{ij} - C_{ij})^2, s.t.\ rank(C) = k\]

    其中observed表示已知值的地方

    1. 第一步对X进行补全，补噪声，得到\(\hat X_0\)
    2. 再SVD，进行低秩近似，得到\(\hat X_{n+1}\)。一直迭代

    潜在语义分析

    例如给定几句话，如何根据给定关键词，对这几句话的相关性进行排列

    将每句话的词变为向量，拼为矩阵后，进行低秩近似。然后可以得到目标词的坐标（看U矩阵的对应行）。然后计算\(SV^T\)，对应列为句子的坐标。则可以通过余弦相似度排序）

    或在论文中，不同方向论文中，出现的词汇不同，则进行SVD后，词的坐标就会有许多维度为0

    大语言模型低秩微调

    针对特定应用，大模型微调时会输入一些数据（比原来训练时少）

    LoRA：用有意义的矩阵很可能低秩，用这些矩阵来找到大模型需要变化的地方
