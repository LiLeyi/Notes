    洛伦兹变换

    相对性原理

    • 牛顿相对性原理：力学规律在一切惯性系中都取相同形式
    • 狭义相对论原理：一切物理规律在任何惯性系中形式相同
    • 广义相对性原理：一切物理规律在任何参考系中形式相同

    同时性具有相对性是光速不变原理

    伽利略变换

    设惯性系S和相对S运动的惯性系\(S'\)

    前提：

    • 两惯性系坐标轴相互平行
    • \(S'\)系相对S系沿x轴正方向，速度为u（为常数）
    • 两系重合时，两系时钟指在零点

    正变换：已知静系\(S\)中物理量，求动系\(S'\)中的物理量

    \[x' = x -ut, y' = y, z' = z, t' = t\]
    \[\frac {dx'} {dt} = \frac {dx} {dt} - u \frac{dt}{dt} \Leftrightarrow v' = v - u\]
    \[a' = a\]

    逆变换：

    \[x = x' + ut', y = y', z = z', t = t'\]
    \[v = v' + u\]
    \[a = a'\]

    两参考系中，\(\vec a' = \vec a\)，则力与参考系无关，质量与运动无关

    牛顿力学规律在伽利略变换下形式不变（牛顿力学规律是伽利略不变式）

    例如：

    • 动量守恒
    • \(\vec F = m \vec a\)

    但麦克斯韦电磁理论与伽利略变换、相对性原理出现冲突

    狭义相对论基本假设

    1. 相对性原理：一切物理规律在任何参考系中形式相同
    2. 光速不变原理：光在真空中的速度（速率）与发射体的运动状态无关

    狭义相对论中，只有光速在各个参考系都不变，而牛顿力学中质量、时间、长度等都不变

    同时性的相对性

    是光速不变原理的直接结果

    相关概念：

    • 事件：研究的物理现象
    • 时空坐标：确定事件位置的坐标\((x,y,z,t)\)，t必须是在坐标\((x,y,z)\)处测出的
    • 同步钟：在确定的参考系中存在的一系列同步钟。利用光信号校准异地钟。\(x_1\)处发射一光信号，经过时间\(\Delta t = (x_2 - x_1) / c\)光信号到达\(x_2\)处，此时把\(x_2\)处的中调到\(\Delta t\)处

    洛伦兹变换导出

    前提：

    • \(t' = t = 0\)时，\(O, O'\)重合，此时O点发出一闪光，S'相对S向x正方向运动。速度为u
    • 经过一段时间，传到P点。S参考系中，为\(P(x,y,z,t)\)，S'中，\(P(x',y',z',t')\)

    推导：

    1. 由光速不变，S中：\(x^2 + y^2 + z^2 = (ct)^2\)，S'中：\(x'^2 + y'^2 + z'^2 = (ct')^2\)
    2. 由运动方向：\(y' = y, z' = z\)
    3. 因为空间均匀且各向同性，\((x,t)\)与\((x',t')\)的变化应满足线性关系（两个惯性参考系间平权，变换与逆变换形式应该类似），则设\(x' = ax + bt, t' = \xi x + \eta t\)
    4. \(x = - \frac b a t\)，求导后得\(u = - \frac b a\)
    5. 将2、3带入1解出。

    正变换：

    \[x' = \gamma(x - ut), y' = y, z' = z, t' = \gamma(t - \frac u {c^2} x)\]

    其中\(\gamma = \frac 1{\sqrt{1 - \frac {u^2} {c^2}}}\)

    逆变换：

    \[x = \gamma(x' + ut'), y = y', z = z', t = \gamma(t' + \frac u {c^2} x')\]

    洛伦兹变换下看同时性的相对性

    S' S
    事件1 \((x_1', t_1')\) \((x_1,t_1)\)
    事件2 \((x_2', t_2')\) \((x_2,t_2)\)

    两事件在S'中同时发生，即\(t_1' = t_2'\)，此时\(\Delta t = 0\)

    根据洛伦兹变换：\(\Delta t' = t_2' - t_1' = \gamma(\Delta t - \frac u {c^2}\Delta x) = -\gamma \frac u {c^2}\Delta x\)，即在不同参考下下，不同位置看到的事件发生时间差肯定是不同的

    所以讨论的“同时”一般指异地的两事件，在某位置看的同时

    因果关系

    \(t_1' = \gamma(t_1 - \frac u {c^2}x_1), t_2' = \gamma(t_2 - \frac u {c^2}x_2)\)，设\(t_2 - t_1 > 0\)，即\(t_2\)的事件后发生

    \[t_2' - t_1' = \gamma (t_2 - t_1) (1 - \frac {u(x_2 - x_1)}{c^2(t_2 - t_1)})\]

    其中\(\frac {x_2 - x_1} {t_2 - t_1}\)为信号传递速度，小于c，即有\(uv < c^2\)，即有\(t_2' - t_1' > 0\)，保持了因果关系。

    因果关系的保持是由信号传递速度上限为c保证的。
