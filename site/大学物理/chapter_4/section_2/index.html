    波动

    波动的基本特征

    波的产生与传播

    一个物理量的扰动或振动在空间的传播称为波动

    • 机械波
    • 电磁波
    • 广义的波：凡是描述运动状态的函数具有时间和空间周期性的特征，都可以称为波。例如引力波、概率波等

    产生条件：

    • 波源
    • 媒质（对机械波必要）

    横波纵波

    大多数波为横波或纵波

    对于机械波，横波只能在固体中传播，纵波可以在固体、液体、气体中传播

    横波只能在具有剪切形变的介质中传播，纵波

    波的几何描述

    波线：指向波传播方向的线 波面：波传播过程中相位相同的点组成的平面 波前：波传播时最前面的波面

    一般情况，波线垂直于波面

    波面是平面（球面）的波称为平面（球面）波

    描述波的特征量

    波长\(\lambda\)

    波数\(\sigma = \frac 1 \lambda\)，单位长度中波长的个数

    角波数\(k = 2 \pi \sigma = \frac {2\pi} \lambda\)，单位长度产生的相位差

    周期T

    波传播一个波长所需的时间，或波线上某一点振动一次的时间

    频率：周期的倒数

    角频率：频率的\(2\pi\)倍\(\omega = 2\pi\nu = \frac {2\pi} T\)

    仅由波源决定，与介质无关

    波速u

    单位时间内振动状态传播的距离

    波速由煤质本身的性质决定

    \(u = \lambda \nu\)

    \(u = \sqrt{\frac{弹性模量} {密度}}\)

    横波对应剪切模量，纵波对应杨氏模量

    惠更斯原理

    波阵面上的各点都可以看作是发射子波的波源，其后任意时刻的波阵面就是这些子波的包络（包络面）

    衍射：波遇到障碍物而改变传播方向。例如在遇到小孔时，由惠更斯原理，可把小孔看作波源

    波的反射定律：入射角等于反射角

    波的折射定律：\(n = \frac {sin i} {sin r}\)，i为入射角，r为出射角

    平面简谐波的波函数

    简谐波波函数

    设原点振动函数为\(y_0 = A cos(\omega t + \phi_0)\)

    沿x+方向振动的：

    则波函数为\(y = Acos(\omega(t - \frac x u) + \phi_0)\)

    沿x-负方向振动的：

    则波函数为\(y = Acos(\omega(t + \frac x u) + \phi_0)\)

    质元速度与加速度\(v = \frac {\partial y}{\partial t},a=\frac {\partial^2 y}{\partial t^2}\)

    行波

    行波，\(y=f(t-\frac x u)\)。原点振动函数为y=f(t)，其下游各点的振动在时间上延迟\(\frac x u\)。显然\(\Delta t\)时间后，整体波形向右平移了\(u\Delta t\)，即波形在以\(u\)的速度移动。这种波称为行波

    波动方程

    行波波函数的一般形式为\(y = f(\beta) = f(t \pm \frac x m)\)

    y对x和t分别求二阶导\(\frac {\partial^2 y} {\partial t^2} = \frac {\partial^2 f}{\partial \beta^2},\frac {\partial^2 y}{\partial x^2} = \frac 1 {u^2}\frac {\partial^2 f}{\partial \beta^2}\)

    比较得\(\frac {\partial^2 y}{\partial x^2} = \frac 1 {u^2}\frac {\partial^2 y}{\partial t^2}\)

    任何物理量\(y(x,t)\)只要满足波动方程，这一物理量就能以波的形式传播

    波的能量

    质元的能量

    对波\(y=cos^2[\omega (t - \frac x u) + \phi]\)

    \(E_p = E_k = \frac 1 2(\rho dV) \omega^2 A^2 sin^2[\omega (t - \frac x u) + \phi]\)

    与振动的势能区别的原因：振动的势能在外界（例如弹簧），波动在内部（例如由形变产生）

    在平面简谐波中，同一质元的动能和势能总是同步变化的

    波的能量密度

    瞬时

    波动中介质单位体积内的总机械能。\(w = \frac {dE} {dV} = \rho \omega^2 A^2 sin^2[\omega (t - \frac x u) + \phi]\)

    质元机械能不守恒，时大时小，体现波的传播

    平均

    \(\bar w = \frac 1 2 \rho \omega^2 A^2\)

    波的能流与能流密度

    能流

    单位时间内通过垂直于波速的某一截面的能量

    \(P = \frac {wSudt}{dt} = wSu = \rho u \omega^2 A^2 S sin^2[\omega (t - \frac x u) + \phi]\)

    平均能流\(\bar P = \bar w Su = \frac 1 2 \rho u \omega^2 A^2 S\)

    波的平均能流密度（强度）

    \(I = \frac {\bar P} S = \bar w u = \frac 1 2 \rho u \omega^2 A^2\)

    波的叠加

    波在传播中相遇，各自独立，互不干扰。但相遇时，任意点位移为各个波在该点产生位移的合成

    满足条件的波：经典波、较弱波

    波的干涉

    波相遇时，空间上周期性的出现加强或减弱的现象，且图案不随时间改变

    相干条件：频率相同、相位差恒定、振动方向一致；振幅相差不大。相应的波叫相干波

    \(A = \sqrt{A_1^2 + A_2^2 + 2A_1 A_2cos\Delta\phi}\)

    • \(\Delta \phi = \pm 2k\pi,A = A_1 + A_2\)，振动加强
    • \(\Delta \phi = \pm (2k+1)\pi,A = |A_1 - A_2|\)，振动减少

    驻波

    同一介质中两列频率相同、振幅相同、波长都相同，但方向相反的波汇聚形成

    相邻两个波节距离为半波长

    半波损失

    波从波疏介质到波密介质垂直入射时，界面的反射波不是入射波的反向延伸，而有\(\pi\)的相位突变，也就是半个波的损失

    \(\rho u\)表示介质对波的疏密程度
