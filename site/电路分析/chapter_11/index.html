    耦合电感与理想变压器

    互感及同名端

    • 自感磁链：\(\psi_{L1} = N_1 \Phi_{11}=L_1i_1\)。表示电感1对自身的磁链。自感系数\(L_1 = \frac {N_1\Phi_{11}}{i_1}\)
    • 互感磁链：\(\psi_M = N_2 \Phi_{21} = Mi_1\)。表示电感1对电感2的磁链。互感系数\(M = \frac {N_2\Phi_{21}}{i_1}\)

    互感系数M是耦合电感的固有参数，取决于两个线圈的结构、大型、相对位置及材料性质

    同名端：产生互感电压的电流流入端，即参考方向下互感电压的正极端（参考方向）。也可以随意规定

    同名端一致原则：电流由同名端流入，且互感电压正参考方向在同名端侧

    耦合电感去耦

    四端元件，互感电压用受控电压源代替

    向电感2串联一个受控源，电压为\(M\frac {di_1} {dt}\)。参考方向：电流从正极流入。向电感1串联一个受控源，电压为\(M\frac {di_2} {dt}\)。参考方向：电流从正极流入。

    二端元件，耦合电感串联

    • 顺接串联：异名端相接，电流从同名端流入。等效电感为\(L_1 + L_2 + 2M\)
    • 反接串联：同名端相接，电流从异名端流入。等效电感为\(L_1 + L_2 - 2M\)

    三端元件，互感电路去耦

    电路为：端口1——电感\(L_1\)（同名端在左）——端口3——电感\(L_2\)（同名端在右）——端口2

    等效为\(L_1\)换为\(L_a\)，\(L_2\)换为\(L_c\)，端口3连接一个电感\(L_b\)再输出。其中：

    \(\begin{cases} L_a = L_1 - M\\ L_c = L_2 - M\\ L_b = M \end{cases}\)

    若有一个电流从异名端流入，则上式中M全变为-M

    耦合电感的VCR

    \(\begin{cases} u_1 = L_1 \frac {di_1} {dt} + M \frac {di_2} {dt}\\ u_2 = L_2 \frac {di_2} {dt} + M \frac{di_1}{dt} \end{cases} \Leftrightarrow \begin{cases} \dot U_1 = j \omega L_1 \dot I_1 + j \omega M \dot I_2\\ \dot U_2 = j \omega L_2 \dot I_2 + j \omega M \dot I_1 \end{cases}\)

    电流电压参考方向决定自感项符号。电流是否流入同名端决定互感项是否和自感项同号（若两个电流均从同名端流入，则同号。每有一个电流从异名端流入，反号一次。）

    耦合系数

    全耦合

    极限情况：\(\Phi_{21} = \Phi_{11}, \Phi_{12} = \Phi_{22}\)

    此时互感M值最大\(M_{max} = \sqrt{L_1 L_2}\)

    耦合系数

    \(k = \frac M {M_{max}} \in [0,1]\)

    • k = 1，全耦合
    • k > 0.5，紧耦合
    • k < 0.5，松耦合
    • k = 0，无耦合

    耦合电感储能

    \(w = \frac 1 2 L_1 i_1^2 + \frac 1 2 L_2 i_2^2 \pm M i_1 i_2\)

    电流均从同名端流入（流出）时取正号，否则为负

    变压器分析

    在耦合电感一端接电源，一端接负载。

    • 二次回路在一次回路的反映阻抗\(\frac {\omega^2 M^2} {Z_{22}}\)
    • 一次回路在二次回路的反映阻抗\(\frac {\omega^2 M^2} {Z_{11}}\)

    理想变压器VCR

    初级匝数\(N_1\)，次级匝数\(N_2\)，则参数匝比（变比）\(n = \frac {N_2} {N_1} = \sqrt {\frac {L_2} {L_1}}\)

    当自感、互感均趋于无穷大且全耦合无损耗情况下，\(u_2 = n u_1\)

    \(\dot I_1 = \frac {\dot U_1} {j \omega L_1} - \sqrt{\frac {L_2}{L_1}} \dot I_2\)，使\(\frac {\dot U_1} {j \omega L_1}\)趋于0，即\(L_1,L_2\)均趋于无穷

    • 两电压高电位与同名端一致时，\(u_2 = n u_1\)，否则有负号
    • 两电流从同名端流入时，\(i_2 = -\frac 1 n i_1\)，否则无负号

    能量：\(p = u_2 i _2 + u_1 i_1 = nu_1 (-\frac 1 n i_1) + u_1 i_1 = 0\)

    理想变压器是无记忆元件，是一种双口电阻元件

    理想变压器阻抗变换

    \(Z_i = \frac {\dot U_1} {\dot I_1} = \frac 1 {n^2}Z_L\)

    即二次电阻对一次侧的折合阻抗。

    理想变压器具有变换阻抗的性质，可实现最大功率匹配

    折合阻抗是输入阻抗
