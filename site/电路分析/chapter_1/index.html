    集总参数电路中的约束关系

    集总电路及分布电路

    当实际电路尺寸远小于使用时其最高工作频率所对应波长时，可以用几种集总参数原件来构成模型。每一种集总参数元件只反映一种基本电磁现象，例如：

    • 电阻只涉及消耗电能的现象
    • 电容只涉及与电场有关的现象
    • 点感只涉及与磁场有关现象

    由集总参数原件组成的电路，称之为实际电路的集总电路模型。在此模型中：

    • 不考虑电路中电场与磁场的相互作用
    • 不考虑电磁波的传播现象
    • 认为电能的传送是瞬间的

    当电路的尺寸大于最高频率所对应的波长，或两者属于同一数量级时，不能作为集总电路处理，应作为分布参数电路处理。对于小尺寸电路，可忽略漏电流，大尺寸则必须考虑

    集总电路分类：

    • 电阻性电路：只包含电阻元件和电源元件
    • 动态电路

    电路变量

    电流

    定义

    电荷量单位为库伦C，\(6.24\times 10 ^ 18\)个电子所具有的电荷量为1C。用q或Q表示电荷量。带电粒子有秩序的移动便形成电流，即单位时间内通过导体横截面的电荷量

    \[i(t)=\frac{dq}{dt}\]

    习惯把正电荷运动的方向规定为电流方向

    国际单位制中单位为安培A

    分类

    • 恒定电流（dc或DC）：大小、方向不随时间变化
    • 时变电流：非恒定电流
    • 交变电流（ac或AC）：大小、方向都随时间做周期性变化

    其他

    集总电路中，一个元件两端的电流是相同的

    电路中要提前规定参考方向，若参考方向和实际电流方向一致，为正值，否则为负值

    电压

    定义

    也称为电位差，用符号u表示，a，b间的电压表明了单位正电荷由a点转移到b点时所获得或失去的能量，即

    \[u(t)=\frac {dw} {dq}\]

    国际单位制中，电压单位为伏特V

    分类

    • 恒定电压（直流电压）：方向大小都不随时间改变
    • 时变电压
    • 交流电压

    其他

    若正电荷从a到b获得能量，则a为低电位，否则为高电位

    需在元件两侧标明参考极性，若真实极性和参考极性相同，电压为正。为了方便起见，采用关联参考方向，电流参考方向与电压参考+极到-极方向一致，这样就只需表明一种参考方向即可

    一般对负载设为关联参考方向，对电源设为非关联参考方向

    功率

    定义

    \[\begin{cases} dw=udq\\ i=\frac {dq} {dt}\\ p(t)=\frac {dw} {dt} \end{cases}\\ \Rightarrow p(t)=i\frac {dq} {dt}=ui\]

    方向

    • 进入功率（吸收功率）
    • 离开功率（提供功率）

    参考方向：在u、i为参考方向时，p为吸收功率

    也需提前规定方向，流入或者流出电路。若电流电压是关联的，且功率方向指向电路，则用p=ui计算功率时，正值表示能量流入电路，负值表示输出能量。若在前文所述方向计算功率，且改变一个参考方向，则功率公式改为p=-ui

    在所述参考方向下，从\(t_0\)到t吸收的能量为

    \[w(t_0,t)=\int _{t_0}^t p(\xi)d\xi=\int_{t_0}^t u(\xi)i(\xi)d\xi\]

    基尔霍夫定律

    基本概念

    • 二端元件（单口元件）：具有两个端子的元件
    • 多端元件：例如三极管（三端）、变压器（四端）、运算放大器（八端）
    • 支路：每一个二端元件或具有多个端钮而由多个元件串联而成的组合
    • 支路电流
    • 支路电压：支路两端的电压
    • 节点：支路的连接点。两个节点若用理想导线连接起来，则这两个节点等价，可算作一个节点
    • 回路：电路中任意闭合路径，路径不应经过一个节点多次
    • 网孔：回路内部不含有其他支路的回路

    内容

    KCL

    流入支点的电流等于流出支点的电流，一定时间内流入支点的电荷等于流出支点的电荷

    \[\Sigma i_入(t) = \Sigma i_出(t)\]

    若规定流入节点电流为正，流出为负，则

    \[\Sigma_{k=1}^K i_k(t)= 0\]

    KCL表达了电路中支路电流间的约束关系

    推广：可将一部分原件（一个闭合面）看做一个节点（超节点、广义节点），对超节点使用KCL

    KVL

    任意回路的电压降的代数和为0。一般电压降为+，电压升为-

    \[\Sigma_{k=1}^K u_k(t) = 0\]

    或者

    \[\Sigma u_降(t)=\Sigma u_升(t)\]

    KVL表达了回路中各支路电压间的约束关系

    推广：电路中任意两点间电压等于两点间任意一条路径经过的各元件电压的代数和

    约束

    约束

    拓扑约束：KCL、KVL

    元件约束：VCR

    独立的KCL有：节点-1（可选任意n-1个节点列方程）

    独立的KVL有：支路-节点+1（支路：对外有两个端钮。推论：若电路问平面电路。则网孔数为b-n+1，且这m个方程独立）

    寻找独立KVL：

    • 网孔法
    • 新支路法（找到的回路一定独立，但找到的独立回路数量可能会少于b-n+1个）

    2b分析法：

    若以所有支路电流和电压为待求解变量，则需要列2b个方程，其中：

    \[\begin{cases} KCL:n-1个,\\ KVL:b-(n-1)个,\\ VCR:b个, \end{cases}\]

    b为支路数，n为节点数

    1b支路电流法

    列KCL，用支路电流及各支路VCR表示b-(n-1)个KVL方程，一共b个方程。解得电流后再用VCR获得电压

    写KVL时，尽量不要选含电流源的回路，因为会引入新变量（电流源无VCR）

    1b支路电压法

    写KCL时，尽量不要选含无伴电压源的回路，因为会引入新变量（电流源无VCR）

    电阻元件

    概念

    电阻元件：反映电阻器对电流呈现阻力的性能

    VCR：电压电流关系

    VAR：伏安关系，和VCR同理

    伏安特性曲线：iu平面（或u-i平面）曲线

    得到VCR方法：

    • 物理法：推导
    • 黑箱法：测量

    电阻元件：任意时刻，若某一个二端元件两端的电压与电流的关系，可由u-i平面上的一条曲线（与时间无关）所决定，此二端元件称为电阻元件

    分类

    按是否经过坐标原点

    • 线性电阻
    • 非线性电阻

    根据VCR曲线是否随时间变化

    • 时变电阻
    • 非时变电阻

    按材料

    • 碳膜电阻：电阻范围大，偏移
    • 金属膜电阻：稳定性高，精度高
    • 线绕电阻：功率大

    按加工工艺

    • 插件电阻
    • 排电阻
    • 贴片电阻

    元件约束

    欧姆定律作为元件约束

    • 电压与电流为关联参考方向：\(R(t) = \frac {u(i)} {i(t)}\)
    • 电压与电流为非关联参考方向：\(R(t) = - \frac {u(i)} {i(t)}\)

    双向性：线性非时变电阻其伏安特性曲线关于原点对称 无记忆性：在任意时刻，电阻的电压或电流仅由当前时刻的电压或电流决定，与之前的历史无关

    耗能性：阻碍电流消耗电能产生焦耳热

    开路与短路

    短路：用理想导线把两点连接起来。对应伏安曲线u=0

    \[\begin{cases} R = 0,\\ u = 0,\\ i_{sc}由外电路决定 \end{cases}\]

    开路：某两点间电阻为无穷大。对应曲线i=0

    \[\begin{cases} R = \infty,\\ u_{OC}由外电路决定,\\ i = 0 \end{cases}\]

    典型非线性元件：二极管

    理想二极管

    • 加正向电压：导通\(R=0，u=0，i>0\)
    • 加反向电压：截止\(R=\infty，u<0，i=0\)

    实际二极管：

    存在正向导通电压，在0到正向导通电压间，仍不导通。存在反向击穿电压\(U_{BR}\)，反向电压过大时，依然可以导通

    电阻的功率

    \[p(t)=i^2(t)R=\frac {u^2(t)} R(不管是否为关联参考方向)\]

    R<0时，p<0，为负电阻，提供能量

    无源元件与有源元件

    无源元件

    对任意\(t>\infty\)元件吸收的能量满足

    \[w(t) = \int_{-\infty}^t u(\xi)i(\xi)d\xi \geq 0\]

    电容电阻电感

    有源元件

    可向外电路提供能量

    电压源，电流源，受控源，负电阻

    电源

    电压源

    输出电压由电压源本身决定，为定值或一定的时间的函数，与外电路无关。输出电流由电压源和外电路决定

    VCR：\(u_s \in const\)

    电压不同的电压源不能并联

    实际电压源可看做一个电阻元件和一个理想电压源

    电流源

    \(i_s\)

    输出电流由电流源本身决定，为定值或一定的时间函数，与外电路无关。输出电压由电流源和外电路决定

    不能开路

    实际电流源VCR:

    \(i = i_s - \frac u {R_0}\)

    可以看做理想电流源并联一个电阻

    特点

    实际方向为非关联方向，则提供功率

    实际方向为关联方向，则吸收功率

    受控源

    例如三极管：电流控制电流源。分控制支路、受控支路

    受控源理论上是四端元件，分为控制端口和受控端口

    压控：

    • 压控电压源VCVS。理想情况下，控制支路开路。VCR：\(i_1=0。u_2=\mu u_1\)。\(\mu\)为转移电压比
    • 压控电流源VCCS。理想情况下，控制支路开路。VCR：\(i_1=0。i_2=g u_1\)。h为转移电导

    吸收功率为两个端口的吸收功率：\(p=u_1 i_1 + u_2 i_2 = u_2 i_2（控制支路断路，i_1为0）\)

    流控：

    • 流控电压源CCVS。理想情况下，控制支路短路。VCR：\(u_1 = 0, u_2=ri_1\)。r为转移电阻
    • 流控电流源CCCS。理想情况下，控制支路短路。VCR：\(u_1 = 0, i_2 = \alpha i_1\)。\(\alpha\)为转移电流比

    吸收功率为两个端口的吸收功率：\(p=u_1 i_1 + u_2 i_2 = u_2 i_2（控制支路短路，u_1为0）\)

    以上四种情况中的参数为常数，则称为线性受控源

    受控源不能作为激励，仅描述器件内部支路间约束关系，没有独立源受控源无法工作
