    电容元件和电感元件

    电阻电路：任意时刻的响应只与该时刻的激励有关，无记忆

    动态元件：电容和电感元件的伏安关系都涉及对电流、电压的微分或积分，我们称这两种元件为动态元件

    动态电路：至少包含一个动态元件的的电路称为动态电路。任意时刻的响应与激励的全部过去历史有关，有记忆

    集总电路不是电阻电路就是动态电路，均服从两类约束

    电容元件

    定义

    储存电荷，建立电场，储存电场能（电荷积累）

    任意时刻，一个二端元件的电荷q(t)与其端电压u(t)的关系由q-u平面上的一条曲线所决定，此二端元件称为电容元件

    线性时不变电容：若u-q曲线是一条过原点直线，且不随时间而改变，此二端元件称为线性时不变电容元件。\(q(t)=Cu(t)\)

    电容量

    描述给定电压，电容器储存电荷的能力。对线性电容\(C= \frac {q(t)} {u(t)}\)，单位法拉（F），微法（\(\mu F\)）

    对平行板电容器：

    \(C = \frac {\epsilon_r S} {4 \pi kd}\)

    电容器参数：电容量C，额定电压

    理想与实际电容器

    理想电容器：只储存电场能量，本身不耗能

    实际电容器：其介质不能做到完全绝缘，故有一定程度的漏电。等效电路为C与R并联

    电容的VCR

    关联参考方向：电流从+流入，-流出

    对线性时不变电容，微分关系的VCR：\(i(t) = \frac {dq}{dt} = C\frac {du}{dt}\)。即直流电路中，为开路。隔直通交

    积分形式：\(u(t) = \frac 1 C \int_{-\infty}^t i(\xi)d\xi = \frac 1 C i(\xi)d\xi + \frac 1 C \int_{t_0}^ti(\xi)d\xi=u(t_0) + \frac 1 C \int_{t_0}^t i(\xi)d\xi(t\geq t_0)\)

    电容电压的连续和记忆性质

    连续性

    \(u_C(t_+) = u_C(t_-) + \frac 1 C \int_{t_-}^{t_+} i(\xi)d\xi\)

    若\(i(\xi)\)在\([t_-,t_+]\)有界，则\(\frac 1 C \int_{t_-}^{t_+} i(\xi)d\xi = 0\)，即\(u_C(t_+) = u_C(t_-)\)。（\(t_-\)为t的前一时刻，\(t_+\)为t的后一时刻）

    即\(u_C\)为连续的（前提为电流有界），可推出电荷也是连续的。连续意味着不能跃变。但电流可以跃变

    记忆性

    \(\frac 1 C \int_{-\infty}^t i(\xi)d\xi\)与过去所有时刻都有关。有记忆电流的性质，称为记忆元件或惯性元件

    必须考虑初始值

    等效电路

    \(u(t_0) + \frac 1 C \int_{t_0}^t i(\xi)d\xi\)可以等效为：初始电压为0的电容元件和一个电压源\(u(t_0)\)串联

    电容的储能

    功率

    关联参考方向下，吸收功率（瞬时）\(p(t) = u(t) i(t)\)

    储能

    \(w_C(t_1,t_2) = \int_{t_1}^{t_2}p(\xi)d\xi = \int_{t_1}^{t_2}u(\xi)i(\xi)d\xi = \int_{t_1}^{t_2}u(\xi)C \frac {du} {d\xi}d\xi = C \int_{u(t_1)}^{u(t_2)}udu = \frac 1 2 C [u^2(t_2) - u^2(t_1)]\)

    对任意时刻（设负无穷时刻电压为0）：\(\frac 1 2 C u^2(t)\)

    任意时刻，电容储能（吸收能力）\(\geq 0\)，电容是无源元件

    将\(u_C\)称为电路的状态变量，反映了电容的储能状态

    电容串并联

    与电导类似

    串联

    \(u(t) = u_1(0) + \frac 1 {C_1} \int_0^ti(\xi)d\xi + \dots = \Sigma u_k(0) + (\Sigma \frac 1 {C_k})\int_0^ti(\xi)d\xi\)

    \(\frac 1 {C_{eq}} = \Sigma \frac 1 {C_k}\)

    \(u_{eq}(0) = \Sigma u_k(0)\)

    等效于一个\(C_{eq}\)的电容，初始电压为\(u_{eq}(0)\)

    并联

    \(i(t) = \Sigma C_k \frac {du} {dt}\)

    \(C_{eq} = \Sigma C_k\)

    等效于一个\(C_{eq}\)的电容

    电感元件

    介绍

    磁通\(\Phi = BS\)，磁链\(\psi = N \Phi\)，N为线圈扎数。单位均为韦伯（Wb）

    楞次定律：感应电流的磁场要阻碍原磁通的变化

    \(感应电动势e(电压升)= - \frac {d\psi} {dt}\)

    \(感应电压u(电压降) = \frac {d\psi} {dt}\)

    定义

    任意时刻，一个二端元件的电流与其磁链的关系可由\(\psi-i\)平面上的一条曲线所决定，此二端元件称为电感元件

    线性时不变电感：若\(\psi-i\)曲线是一条过原点的直线，且不随时间改变，此二端元件称为线性时不变电感

    电感量L

    描述给定电流，电感器储存磁链的能力

    对线性电感：\(L = \frac {\psi(t)} {i(t)}\)，单位亨利（H）

    电感器参数：电感量L，额定电流

    理想与实际电感器

    理想电感器：导线电阻为0

    实际电感器：导线具有电阻，有能量损耗，可等效为L与R串联（更进一步，并联电容，在高频信号需要这样建模）

    电感的VCR

    关联参考方向：电流从+流入，-流出

    \(u(t) = \frac {d\psi} {dt} = L\frac {di} {dt}\)，即直流电路中短路

    积分关系：

    \(i_L(t) = \frac 1 L \int_{-\infty}^t u(\xi)d\xi = i_L(t_0) + \frac 1 L \int_{t_0}^t u(\xi)d\xi\)

    电感电流的连续性和记忆性、储能、串并联

    记忆电压，若电压有界，则电流连续，电流不能跳变

    一个电感可等效为：初始电流为0的电感元件和一个电流源\(i(t_0)\)并联

    储能：\(w_L(t) = \frac 1 2 i_L^2(t)\)，电感储能由当前时刻电流决定，电感是无源元件

    电感串联并联和电阻的串联一致

    电容与电感的对偶性，状态变量

    C与L为对偶量，q与\(\psi\)也为对偶量
