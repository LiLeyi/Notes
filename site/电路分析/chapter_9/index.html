    正弦稳态功率和能量

    电阻的平均功率

    \(p(t) = \frac {U_m} {\sqrt 2} \frac {I_m} {\sqrt 2} [1 + cos(2 \omega t)] = UI [1 + cos(2 \omega t)]\)

    p的角频率为电压或电流角频率的两倍，且p恒大于0

    平均功率\(U = \frac 1 T \int_0^T \frac 1 2 U_m I_m [1 + cos(2 \omega t)]dt = \frac 1 2 U_m I_m = UI = I^2 R = \frac {U^2} R\)

    使用有效值时，正弦稳态电路中电阻平均功率的计算与直流电阻电路公式相同。电阻平均功率大小与正弦电流/电压的频率及初相位无关

    电容电感的平均储能

    电感

    功率

    \(p(t) = UI sin(2 \omega t + 2\phi)\)，其中\(\phi\)为电压初相位

    平均功率为0

    无功功率：瞬时功率的振幅\(Q_L = \frac 1 2 U_m I_m = UI\)。无功功率表明电感与外电路之间能量交换的规模，单位为乏（volt ampere reactive）。无功功率指用于电路与电源的能量的交换，它不对外做功，但维持用电设备正常运行

    有功功率：瞬时功率的平均值。指电能用于对外做功被消耗掉的电功率，电能转换为机械能、光能、热能、化学能等

    储能

    瞬时储能\(w_L(t) = \frac 1 2 L i^2(t) = \frac 1 2 L I_m^2 sin^2(\omega t)>0\)

    平均储能\(W_L = \frac 1 2 LI^2\)

    无功功率与平均储能的关系\(Q_L = UI = \omega LI^2 = 2 \omega W_L\)

    平均储能越多，能量往返频率越高，则能量交换的规模就越大

    电容

    功率

    瞬时功率\(p(t) = - UI sin(2 \omega t + 2 \phi)\)，其中\(\phi\)为电压初相位

    平均功率为0

    无功功率\(Q_C = - \frac 1 2 U_m I_m = - UI\)

    储能

    瞬时储能\(w_C = \frac 1 2 Cu^2 = \frac 1 2 CU_m^2 cos^2(\omega t)\)

    平均储能\(W_C = \frac 1 2 CU^2\)

    无功功率和平均储能关系：\(Q_C = - UI = - \omega CU^2 = - 2 \omega W_C\)

    平均储能越多，能量往返频率越高，则能量交换的规模就越大

    关系

    • 电感瞬时功率\(p_L(t) = UI_L sin(2\omega t + 2\phi)\)
    • 电容瞬时功率\(p_C(t) = - UI_C sin(2\omega t + 2\phi)\)

    同频反相，具有互相补偿的作用，通常说L吸收无功（无功功率含为正），C发出无功（无功功率为负）

    单口网络相关

    瞬时功率及平均功率

    设\(u(t) = U_mcos(\omega t + \psi_u),i(t) = I_m cos(\omega t + \psi_i)\)

    则\(p(t) = U_mI_m cos(\omega t + \psi_u)cos(\omega t + \psi_i) = \frac 1 2 U_mI_mcos(\Delta \psi) + \frac 1 2 U_mI_mcos(2\omega t + \psi_u + \psi_i)\)。\(\Delta \psi > 0 感性，\Delta \psi < 0 容性\)。\(\Delta \psi = \psi_u - \psi_i\)

    前半部分为恒定部分，后半部分为可逆部分。

    平均功率（有功功率）：\(P = UI cos(\Delta \psi)\)，其中\(Ucos(\Delta \psi)\)可称为电压的有功分量

    则有：

    • 纯电阻单口网络\(\Delta \psi = 0, P = UI\)
    • 纯电感或电容混联单口网络\(\Delta \psi = \pm 90^\circ, P = 0\)
    • 电阻电感电容混联单口\(\Delta \psi = (-90^\circ, 90^\circ), P>0\)
    • 含受控源\(\Delta \psi\)可能大于\(90^\circ\)，P可能为负

    无源单口网络的有功、无功功率

    无源单口网络的平均功率（有功功率）等于其等效阻抗的电阻分量（实部）吸收（消耗）的功率。\(P = UIcos\Delta \psi = U_R I = I^2 Re(Z) = U^2 Re(Y)\)

    无源单口网络等效阻抗的电抗分量吸收或提供的功率即为单口网络的无功功率。\(P = I^2 Im(Z) = - U^2Im(Y)\)

    • 有功功率\(P = UIcos(\Delta \psi) = U_R I = I^2 ReZ = U^2 ReY\)，其中\(Ucos(\Delta \psi)\)为电压的有功分量
    • 无功功率\(Q = UIsin(\Delta \psi) = U_X I = I^2 ImZ = - U^2 ImY\)，其中\(Usin(\Delta \psi)\)为电压的无功分量

    无功功率与电路平均储能的关系\(Q = Q_L + Q_C = 2 \omega (W_L - W_C)\)

    两种储能在网络内部可自行交换，与外电路往返的能量为两种动态元件平均储能的差额

    视在功率及功率因数

    • 有功功率\(P = UIcos(\Delta \psi)\)，单位W
    • 无功功率\(Q = UIsin(\Delta \psi)\)，单位var
    • 视在功率\(S = UI\)，单位\(V \cdot A\)

    关系：\(S^2 = P^2 + Q^2\)，视在功率一般大于有功及无功功率

    功率因数\(\lambda = \frac P S = cos(\Delta \psi)\)

    单口网络内部，对总功率和内部各原件功率和的关系：

    • 瞬时、有功、无功功率相等
    • 视在功率不相等

    功率因数过低导致：

    • 电源设备的容量不能充分利用
    • 增加输电线路和发电机绕组的功率损耗

    原因：通常是由于存在电感性负载

    提高措施：

    • 并联储能性质相反的原件，如将适当的电容与电感性负载并联

    功率因数一般提高到0.9左右。一般不会提到1（会发生谐振，导致过电压过电流）

    最大功率传输定理

    外阻抗全可变

    \(P_L = I^2 R_L = \frac {U_{OC}^2 R_L} {(R_0 + R_L)^2 + (X_0 + X_L)^2}\)

    \(R_L = R_0，且X_L = - X_0\)时，即共轭匹配，\(Z_L = \bar Z_0 = R_0 - j X_0\)时，平均功率最大。\(P_{Lmax} = \frac {U_{OC}^2} {4R_0}\)

    外阻抗阻抗角固定，模可变

    \(|Z_L| = |Z_0|\)最大，模匹配
