    频率响应 多频正弦稳态电路

    再论阻抗与导纳

    单口网络的阻抗\(Z(j \omega) = R(\omega) + j X(\omega)\)

    频率的量变可能引起电路的质变，例如RLC的\(\omega\)变化时，可能导致电路的容性感性发生变化

    频率响应：电路响应与激励频率的关系。\(Z(j\omega) = R(\omega) + j X(\omega)=|Z(j\omega)|\angle \phi_Z(\omega)\)

    • 幅频特性：\(|Z(j\omega)| = \sqrt{R^2(\omega)+X^2(\omega)}\)
    • 相频特性：\(arctan\frac {X(\omega)} {R(\omega)}\)

    正弦稳态网络函数

    单一激励时，响应相量与激励相量之比

    \(H(j\omega) = \frac {响应相量} {激励相量}\)

    策动点函数：同一端口上响应相量与激励相量的比

    • 策动点阻抗：\(Z(j\omega) = \frac {\dot U_1} {\dot I_1}\)
    • 策动点导纳：\(Y(j\omega) = \frac {\dot I_1} {\dot U_1}\)

    转移函数：不同端口上响应相量与激励相量之比

    • 转移阻抗：\(Z_T(j\omega) = \frac {\dot U_2} {\dot I_1}\)
    • 转移导纳：\(Y_T(j\omega) = \frac {\dot I_2} {\dot U_1}\)
    • 电压转移函数：\(A_u(j\omega) = \frac {\dot U_2} {\dot U_1}\)
    • 电流转移函数：\(A_i(j\omega) = \frac {\dot I_2} {\dot I_1}\)

    滤波电路

    低通滤波

    将\(u_1,R,C\)串联，C的电压为\(u_2\)（输出电压），此时构成一个低通滤波电路。电压转移函数：

    \(H_u = \frac 1 {1 + j\omega CR} = \frac 1 {\sqrt{1 + (\omega CR)^2}} \angle - \arctan(\omega CR)\)

    幅频特性

    \(|H_u| = \frac 1 {\sqrt{1 + (\omega CR)^2}}\)

    \(\omega = \omega_C = \frac 1 {RC} = \frac 1 \tau\)时，\(|H_u| = \frac {U_2} {U_1} = \frac 1 {\sqrt 2} = 0.707\)

    功率与电压频率成正比，功率将降低\(\frac 1 2\)，此时\(\omega_C\)被称为半功率点。幅频特性表明此RC电路具有低通特性。工程上称\(\omega_C\)为截止频率，\(0 \sim \omega_C\)为低通网络的通频带

    相频特性

    \(\phi = - \arctan(\omega CR)\)

    • \(\omega = 0\)时，\(\phi = 0\)
    • \(\omega = \omega_C\)时，\(\phi = -45^\circ\)
    • \(\omega = \infty\)时，\(\phi = -90^\circ\)

    输出电压总是滞后于输入电压，称为滞后电路

    正弦稳态的叠加

    傅里叶级数

    若周期函数\(f(\omega t)\)，角频率为\(\omega\)，满足狄利赫里条件，则可以分解为

    \(f(\omega t0) = A_0 + A_{1m} cos(\omega t + \psi_1) + A_{2m} cos(\omega t + \psi_2) + \dots = A_0 + \Sigma_{k = 0}^ \infty A_{km} cos(k\omega t + \psi_k)\)

    • 直流分量：\(A_0\)
    • 基波（一次谐波）：\(A_{1m} cos(\omega t + \psi_1)\)
    • k次谐波：\(A_{km} cos (k \omega t + \psi_k)\)

    非正弦周期激励可转化为多个不同频率的正弦激励

    有效值：

    • \(I = \sqrt {I_0^2 + I_1^2 + \dots}\)
    • \(U = \sqrt {U_0^2 + U_1^2 + \dots}\)

    正弦稳态电路的叠加原理：由多个正弦电源和线性元件组成的线性时不变电路中，任意元件的电流或电压的稳态响应可看成是每一个独立源单独作用于电路时，在该元件上产生的电流或电压的和

    具体应用：对每一频率的正弦电源可应用相量分析法求解，然后在时域叠加

    平均功率的叠加

    叠加原理一般不适用于瞬时功率、平均功率的计算

    例外：多个不同频率（两两不同）的正弦电流或电压产生的平均功率满足叠加原理（因为均正交，交叉项\(\int_0^Ti_ji_kdt = 0\)，电压也是）

    所以对非正弦周期激励，平均功率等于恒定分量和各正弦谐波分量的平均功率之和

    谐振

    定义

    在具有电感和电容的电路中，若调节电路的参数或电源频率，使输入阻抗为纯电阻，即电压与电流同相，则称电路发生了谐振

    理解

    例如对RLC串联电路，\(\omega_0 = \frac 1 {LC}\)发生谐振，称为谐振角频率，此时功率因数为1。若感抗容抗大于R，会出现过电压，即电感电容的电压远高于电源电压。RLC会出现过电流

    一般认为谐振时电压（电流）最大（实际上在谐振频率附近时最大）

    品质因数

    单个动态元件无功功率比上电阻元件的有功功率

    对RLC串联电路

    \(Q = \frac {U_L} U = \frac {\omega_0 L} R = \frac 1 R \sqrt {\frac L C}\)

    \(Q = \frac {U_C} U = \frac 1 {\omega_0 C R} = \frac 1 R \sqrt {\frac L C}\)

    Q值越大，电路的选择性就越强（通过一个特定频率，抑制其余频率的能力），通频带越窄（该曲线为带通。截止频率：\(I = 0.707I_m\)时对应两个f所形成的区间，称为通带，其他区间称为阻带），I-f曲线越窄

    通频带计算

    \(BW_\omega = \frac R L = \frac {\omega_0} Q\\ BW_f = \frac R {2\pi L} = \frac {f_0} Q\)
