    栈与队列

    栈

    定义及主要操作

    只允许在表的一端进行插入和删除的线性表。允许插入和删除的一侧称为栈顶，另一侧称为栈底。特点为后进先出

    Stack CreateStack();
void DisposeStack(Stack s);
int IsEmpty(Stack s);
int IsFull(Stack s);
Stack MakeEmpty(Stack s);
int Push(ElementType x, Stack s);
int Pop(Stack s);
ElementType Top(Stack s);
int Size(Stack s);

    顺序栈

    是栈的顺序存储表示。借用一块连续的储存地址作为储存空间（C中使用一维数组）

    结构定义

    静态：

    #define maxSize 100
typedef int SElemType;
typedef struct {
    SElemType elem[maxSize];
    int top;
} SeqStack;

    进栈前需要判断是否满。

    动态：

    #define initSize 100
typedef int SElemType;
typedef struct {
    SElemType elem[maxSize];
    int maxSize;
    int top;
} SeqStack;

    若栈满，则需要重新申请连续存储空间，将所有栈元素转移到新的存储空间，修改指针，然后释放原空间

    双栈共有空间

    申请一块连续地址，两个栈底在两端，栈顶在入栈时向中间靠拢，若相遇，则认为发生溢出。

    可节省空间（例如两个站一个膨胀的慢，一个快，则需要把慢的空间给一部分给快的）

    链式栈

    是栈的链接存储表示。可以提高效率以及共享存储空间（因为物理位置可以不连续）

    链式栈的栈顶指针在链表头节点，但实际的栈顶在头结点后的首元结点

    混洗问题

    可以用递推得到公式。

    可能的退栈输出情况数：

    \[\sum_{i=0}^{n-1} m_i \times m _{n-i-1} = \frac 1 {n+1} C_{2n}^n\]

    其中\(m_1 = 1, m_2 = 2\)（卡特兰数推导有关点我）

    队列

    定义

    是一种限定存取位置的线性表。只允许在一头插入（队尾），另一头删除（队头）。先入先出FIFO

    void InitQueue(Queue &Q);
bool EnQueue(Queue &Q, QElemType x);
bool DeQueue(Queue &Q, QElemType &x);
int GetFront(Queue &Q, QElemType &x);
bool QueueEmpty(Queue &Q);
bool QueueFull(Queue &Q);
int QueueSize(Queue &Q);

    循环队列（队列的数组存储表示）

    使用一维数组进行存储，设置两个指针front、rear分别指示队头与队尾。在非空队时，front会指向元素，而rear不指向元素。入队时队尾指针加1,出队时队首指针加1（后进行模运算）。

    • 队空条件：front == rear
    • 队满条件：(rear + 1) % maxSize == front。此时实际是数组中空了一个元素，否则无法与队空区分

    结构定义：

    typedef char QElemType;
typedef struct {
    QElemType * elem;
    int maxSize;
    int front, rear;
} CircQueue;

    可以单独定义一个队列的信息记录结构：

    struct QueueRecord {
    int Capacity;
    int Front;
    int Rear;
    int Size;
    ElementType *Array;
}

    双循环队列

    两个队列共享环型向量空间，可以更好地利用存储空间。

    在两队列首尾相接时，向其中一个队列（元素加入位置满了的那个队列）加入元素时，可以先将另外一个队列移动一个位置，以空出一个位置插入新元素，但若无法移动，则满。

    链式队列

    基于单链表的存储表示。为操作方便，一般不设头结点，队头指针直接指向首元结点。多一个队尾指针指向单链表的尾节点。适合数据变动较大情形，因为不存在溢出。若程序使用多个队列，最好使用链式队列，更好的分配空间

    结构定义：

    typedef int QElemType;
typedef struct Node {
    QElemType data;
    structure Node * link;
} LinkNode;
typedef struct {
    LinkNode * front, * rear;
} LinkQueue;

    双端队列

    三种类型：

    • 不受限：两端都可插入、删除
    • 输出受限：两端都可插入，但只能一端删除
    • 输入受限：两端都可删除，但只能一端插入

    对于不受限双端队列，若进n个元素，则出队的序列有\(n!\)个

    可用循环队列表示，类似普通队列。也可用无头结点双端循环链表进行实现

    应用

    数制转换

    例如十进制转k进制，把每次除k得的余数，倒过来写即k进制。此时就用到栈

    括号匹配

    此时用到栈

    回文数

    用到栈与队列，先把数每一位都分别加入栈和队列，再同时弹栈、出队，比较这两个元素

    表达式计算与优先级处理

    后缀表达式求值

    向栈中压入数字，遇到二元运算符时就弹栈，弹两个元素进行该运算然后加入栈，遍历表达式即求得值

    中缀转后缀表达式

    用一个栈

    运算符优先级分为栈内优先级与栈外优先级（因为要考虑括号）。依次读入表达式字符ch

    • 若ch为操作数，直接输出，读入下一个字符ch
    • 若ch为操作符，比较ch的栈外优先级与当前栈顶操作符的栈内优先级
    • 若ch的栈外优先级高于栈顶的栈内优先级：ch进栈，读入下一个字符ch
    • 若ch的栈外优先级低于栈顶的栈内优先级：弹栈并输出，ch不变，继续比较
    • 若ch的栈外优先级等于栈顶的栈内优先级（这种情况只有ch为后括号栈顶为前括号，或遇到结束符时才触发）：退栈但不输出，若退出的是前括号，则读入下一个字符ch，否则继续比较

    栈与递归的实现

    递归的定义：若一个对象部分包含它自己。或自己给自己定义，则这个对象是递归的；若一个过程直接或间接地调用自己，则称这个过程是递归的过程

    每层递归所需保存的信息构成一个工作记录，包括：

    • 返回地址：即上一层中调用自己的语句的后继语句处
    • 在本次过程调用时，为与形式参数对应的实际参数创建副本
    • 为本层的局部变量值分配存储空间

    在每进入一层递归时，系统就要建立一个新的工作记录，加到递归工作栈栈顶。每退出一层递归，就弹栈一次

    递归类型：

    • 单向递归：只向一个方向递归，不会往回走然后又向某个方向走。例如阶乘
    • 回溯递归：会往回走，然后再向某个方向走（例如迷宫深度搜索，不满足条件会返回上一步）
    • 尾递归：在函数最后调用自身。可以转化为循环
    • 间接递归：例如A调用B，B又调用A

    打印杨辉三角

    使用队列

    广度优先搜索最短路径

    从出发点（路径长为0）开始，将相邻点加到队列中，将队首格子的没有设置路径长的相邻格子，设置路径长为队首格子的路径长+1，然后加入队尾。直到遍历到结束点为止。

    从终点反向寻找，在各个相邻点找编号最小者回退，直到退回到起点

    深度搜索求解迷宫

    递归：试探、回溯，若找到则逆向输出路径

    非递归：用一个栈记录即可
