    代数多项式插值

    插值法绪论

    函数解析式未知或太复杂，用函数p(x)去近似代替它，使得\(p(x_i)=y_i\)，这类问题称为插值函数

    • \(p(x)\)称为插值函数
    • \(x_0,x_1,\cdots,x_n\)称为插值节点
    • 插值节点所在区间称为插值区间
    • \(p(x_i)=y_i\)称为插值条件

    代数多项式插值

    \[P_n(x) = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n\]

    使满足\(P_n(x_i) = y_i\)，及利用\(P_n(x)\)进行插值计算的问题

    n个点节点，需要n次代数式确定唯一插值（大于n解不唯一），带入可解线性方程组得系数。但此时方程组可能是病态方程组，或计算量很大，当阶数n越高时，病态越重

    不等距条件下的牛顿基本差商公式

    差商（均商）

    • \(f(x)\)在\(x_i\)点的零阶差商位\(f[x_i]=f(x_i)\)
    • \(f(x)\)在\([x_i, x_j]\)上的一阶差商为\(f[x_i, x_j] = \frac {f[x_j] - f[x_i]} {x_j - x_i}\)
    • \(f(x)\)在\([x_i, x_j, x_k]\)区间上一阶差商的差商为二阶差商为\(f[x_i, x_j, x_k] = \frac {f[x_j,x_k] - f[x_i,x_j]}{x_k - x_i}\)
    • \(f(x)\)在\([x_1, x_2, \dots, x_n]\)区间上n阶差商为\(f[x_1, x_2, \dots, x_n] = \frac {f[x_2, \dots, x_n] - f[x_1, \dots, x_{n-1}]}{x_n - x_1}\)

    差商表...

    一阶差商可看做平均变化率（平均速度），二阶差商可看做平均加速度

    差商具有对称性

    \[f[x_0, x_1, \dots, x_k] = \frac {f(x_0)} {(x_0 - x_1)(x_0 - x_2)\dots(x_0 - x_k)} +\\ \frac {f(x_1)} {(x_1 - x_0)(x_1 - x_2)\dots(x_1 - x_k)} + \cdots +\\ \frac {f(x_k)} {(x_k - x_0)(x_k - x_1)\dots(x_k - x_{k-1})}\]
    • 差商值与变量的排序次序无关
    • \(f(n)\)为n次多项式时，可以证明它的n阶差商为常数

    牛顿基本差商公式

    设x为插值区间的一个节点

    \[f[x_0,x] = \frac {f[x] - f[x_0]} {x - x_0} \Rightarrow f[x, x_0] (x - x_0) = f(x) - f[x_0]\]
    \[f[x_1,x_0,x] = \frac {f[x_0,x] - f[x_1,x_0]} {x - x_1} \Rightarrow f[x, x_1, x_0] (x - x_1) = f[x_0,x] - f[x_1,x_0]\]

    牛顿基本差商公式：

    \[f(x) = f(x_0) + (x-x_0)f[x_1,x_0] + (x-x_0)(x-x_1)f[x_2,x_1,x_0] + \cdots \\ + (x-x_0)(x-x_1)\cdots(x-x_{n-1})f[x_n,x_{n-1},\dots,x_0]\\ + (x-x_0)(x-x_1)\cdots(x-x_{n-1})(x-x_n)f[x,x_n,x_{n-1},\dots,x_0]\]

    前面的项能被计算。最后一项为\(R_n(x)\)，称为余式。\(f(x) \approx P_n(x) + R_n(x)\)

    例如n=1时，\(f(x) = f(x_0) + (x-x_0)f[x_1,x_0] + (x-x_0)(x-x_1)f[x_2,x_1,x_0]\)，此时\(P_n(x) = f(x_0) + (x-x_0)f[x_1,x_0] = y_0 + \frac {y_0 - y_1} {x_0 - x_1}(x-x_0)\)，为直线。

    每多增加一个插值节点，只需要多增加一项

    余式估计

    x恰好落在插值节点时，误差\(R_n(x) = 0\)

    差商和导数的关系：在区间内存在一点\(\xi\)，多次使用Rolle中值定理以及零点存在定理得

    \[f[x_0, x_1, \dots, x_n] = \frac {f^{(n)}(\xi)} {n!}\]
    \[R_n(x) = \frac {f^{n+1}(\xi)} {(n+1)!} (x - x_0) (x - x_1) \cdots (x - x_{n-1})\]
    \[|R_n(x)| = |\frac {f^{n+1}(\xi)} {(n+1)!}||(x - x_0) (x - x_1) \cdots (x - x_{n-1})| \leq \frac {M_{n+1}} {(n+1)!}|\Pi_{n+1}(x)|\]

    常用n+1阶导的上界估计误差

    对于函数\(\Pi_{n+1}(x)\)，振幅两头大中间小，当外插时，其幅值较大，应尽量避免

    事后估计误差的方法（经验方法）：多取一个节点\(x_{n+1}\)，以前n+1与后n+1个节点，作两次插值，得到两个插值函数\(P_n(x), P_{n}^{(1)}(x)\)，余式\(R_n^{(1)}(x)\)与\(R_n^{(2)}(x)\)，则有\(R_n(x) \approx \frac {x- x_0} {x_0 - x_{n+1}}[P_n(x) - P_n^{(1)}(x)]\)

    等间距下牛顿基本差商公式

    差分

    等距节点\(x_{i+1} - x_i = h\)，函数在等距节点上的值为\(y_0,y_1,\cdots,y_n\)，称\(\Delta y_{i-1} = y_i - y_{i-1}\)为f(x)在\([x_{i-1},x_i]\)上的一阶差分。称\(\Delta^2 y_{i-1} = \Delta y_i - \Delta y_{i-1}\)为二阶差分...n阶差分\(\Delta^n y_{i-1} = \Delta^{n-1} y_i - \Delta^{n-1} y_{i-1}\)

    差商与差分的关系：\(f[x_i,x_{i+1},\dots,x_{i+n}] = \frac {\Delta^n y_i} {n! h^n}\)

    插值节点较多时会导致误差变大（看ppt的图）

    牛顿前插公式

    \(t = \frac {x - x_0} h\)，可看做x的下标。\(x-x_i = x - x_0 - (x_i - x_0) = (t - i)h\)

    \[P_n(x) = y_0 + \frac t {1!} \Delta y_0 + \frac {t(t-1)} {2!}\Delta^2y_0 + \dots + \frac {t(t-1)\dots(t-n+1)}{n!} \Delta^n y\\ = y_0 + C_t^1 \Delta y_0 + \dots C_t^n \Delta^n y_0\]

    费雷瑟图表

    看ppt。结论：对于函数列的某个函数值\(y_i\)和终止于同一个差分的任何路径，按规则建立的差值公式都是等价的

    规则：

    1. 正斜率：向右上方走。该差分与位于其下面的二项式系数之积作为项。牛顿后插公式
    2. 负斜率：向右下方走。该差分与位于其上面的二项式系数之积作为项。牛顿前插公式
    3. 水平：从函数值开始向右水平走。差分与位于其上下的二项式系数的算术平均值之积作为项，或系数与其上下的差分的平均值之积作为项。斯梯林插值公式
    4. 水平：从系数开始向右水平走。差分与位于其上下的二项式系数的算术平均值之积作为项，或系数与其上下的差分的平均值之积作为项。贝塞尔差值公式
