    迭代公式的改进

    • 提高精度以减少迭代次数
    • 减小q的值
    • 提高收敛的阶数r

    思路：重新构造等效方程。

    例如：取\(\theta\in (0,1)\)

    \(x=\phi(x) \Rightarrow x - \theta x = \phi(x) - \theta x\Rightarrow x = \frac 1 {1 - \theta} [\phi(x) - \theta x] = \psi (x)\)

    则可建立迭代公式：\(x_{n+1} = \frac 1 {1 - \theta} [\phi(x_n) - \theta x_n]\)

    艾特肯法

    \(y_1 = \phi(x_0), z_1=\phi(y_1)\)

    取以下平均变化率作为\(\theta = \theta_1\)

    \[\theta_1 = \frac {\phi(y_1) - \phi(x_0)} {y_1 - x_0} = \frac {z_1 - y_1} {y_1 - x_0}\]

    代入修正迭代公式，得\(x_1\)，再计算\(y_2,z_2\)，得\(\theta_2\)，再代入修正迭代公式，得\(x_2\)...

    代入化简得：\(x_{n+1} = \psi(x_n) = \frac {x_n z_{n+1} - y_{n+1}^2}{x_n - 2y_{n+1} + z_{n+1}}\)

    几何含义：两次迭代的点连线来逼近曲线，由这个连线与y=x的交点横坐标作为下一次迭代的初值

    收敛速度：

    • 如果原迭代公式的迭代序列为1阶收敛，则使用埃特肯法后为2阶收敛。（所以一般说艾特肯法阶数为2）
    • 若原迭代为p阶收敛，则修正后为\(2p - 1\)阶收敛
    • 如果原迭代公式的迭代序列不收敛，则使用埃特肯法后可能为2阶收敛。

    牛顿迭代法

    \(f(x)=0\Rightarrow \lambda f(x) = 0\Rightarrow x = x - \lambda f(x)\)，取\(\lambda\)值，使得\(\phi'(x) < 1\)，则有\(|\phi'(x)| = |1 - \lambda f'(x)|\)则取\(\lambda = \frac 1 {f'(x)}\)，\(x_{n+1} = x_n - \frac{f(x_n)} {f'(x_n)}\)

    几何意义：初值点的切线与横轴的交点为下一个点

    收敛阶数为2。对初值敏感，若初值满足：

    • f(a)f(b)<0。保证有根
    • \(f'(x)\neq 0\)。保证分母不为0
    • \(f''(x)\)不变号。
    • 初值\(x_0\)满足\(f(x_0)f''(x_0) > 0\)。使产生的序列单调有界，保证收敛

    则初值收敛

    优点：收敛性好，稳定性好，精度高

    缺点：对初值要求高，每次迭代都需要计算函数值与导数值，计算量大；当导数值提供有困难时牛顿法无法进行

    收敛阶数为2，有重根时为1

    牛顿下山法

    为了扩大收敛范围，引入参数

    \[x_{n+1} = x_n - \frac {f(x_n)} {f'(x_n)}\Rightarrow x_{n+1} = x_n - \lambda \frac {f(x_n)} {f'(x_n)}\]

    达到\(|f(x_0)|>|f(x_1)|>|f(x_2)|>...\)单调下降的目的。称\(|f(x_{n+1})|<|f(x_n)|\)为下山条件。只有牛顿迭代法失效时才启动下山法。每次迭代\(\lambda\)都从1开始，若不满足下山条件，则将\(\lambda\)缩小再计算是否符合下山条件，例如乘二分之一

    该迭代公式等价于\(x_{n+1} = \lambda [x_n - \frac {f(x_n)} {f'(x_n)}] + (1 - \lambda) x_n\)，令\(y_{n+1} = x_n - \frac {f(x_n)} {f'(x_n)}\)，得\(x_{n+1} = \lambda y_{n+1} + (1 - \lambda)x_n\)，可由旧值迭代得新值

    弦截法

    单点

    割线法。为了避免牛顿迭代法中的导数计算用割线的斜率代替牛顿迭代法中切线的斜率。将牛顿迭代格式中的导数用差商代替

    \(f'(x) \approx \frac{f(x_k) - f(x_0)} {x_k - x_0}\)

    则有：

    \(x_{k+1} = x_k - \frac {f(x_k)} {f(x_k) - f(x_0)} (x_k - x_0) = \frac {x_0f(x_n) - x_n f(x_0)} {f(x_n) - f(x_0)}\)

    单点弦截法收敛阶数为1，在靠近零点时收敛速度很快

    单点弦截法收敛条件：

    1. f(a)f(b) < 0
    2. \(f'(x)\neq 0\)
    3. \(f''(x)\)不变号
    4. 不动点\(x_0\)满足\(f(x_0)f''(x_0) > 0\)，\(x_1\)与\(x_0\)的函数相异

    双点

    切线斜率约等于割线斜率，即\(f'(x_k) \approx \frac {f(x_k) - f(x_{k-1})} {x_k - x_{k-1}}\)

    \[x_{k+1} = x_k - \frac {f(x_k)} {f(x_k) - f(x_{k-1})} (x_k - x_{k-1}) \\= \frac {x_{n-1}f(x_n) - x_n f(x_{n-1})}{f(x_n) - f(x_{n-1})}\]

    收敛条件：

    1. f(a)f(b) < 0，且\(f'(x)\neq 0\)
    2. 双点弦截法对任意\(x_0,x_1\in [a,b]\)都收敛
