    1. 构造算法
    2. 分析算法
    3. 使用算法

    误差

    设A是精确值，a是近似值，则绝对误差定义为\(\Delta = a - A\)。绝度误差限定义为\(\epsilon = \{|\Delta|\}\)的上界（实际问题中，上界应该尽量小）。（有时绝对误差和绝对误差限未区分，看语境）

    由上述定义，有：\(a - \epsilon < A < a + \epsilon\)

    相对误差定义为绝对误差与精确值的比：\(\delta A = \frac \Delta A\)。但由于A一般未知，实际计算时经常用\(\delta A \approx \frac \Delta a\)，这样代替的误差是\(\frac \Delta A - \frac \Delta a = \frac {a - A} {Aa} \Delta = \frac 1 {Aa}\Delta^2\)，为\(\Delta\)的高阶小量，可以接受。

    绝对误差有量纲，相对误差无量纲

    舍入方法

    截断法

    \(A = a_0 a_1\dots a_m . a_{m+1} a_{m+2} \dots a_{m+n}|a_{m+n+1} \dots\)（截断）

    绝对误差限为\(10^{-n}\)（最末单位的一个单位）

    四舍五入

    \(0.5\times 10^{-n}\)（最末单位的半个单位）

    例如3.14159的误差限为\(0.5\times 10^{-5}\)

    有效数字

    如果数x的绝对误差不超过某一位数字的半个单位，若该位数字到x的第一位非零数字共有n位，则称x具有n位有效数字。

    四舍五入法得到的每一位都是有效的

    算数运算中的误差

    设\(x^*,y^*\)为准确值，\(x,y\)为近似值，则绝对误差为\(\Delta x = x - x^*,\Delta y = y - y^*\)。用微分来近似绝对误差，即\(dx\approx \Delta x,dy = \Delta y\)。且有\(dy \approx f'(x_0) \Delta x\)

    加减法

    \(C = x \pm y\)

    \(\Delta C = C - C^* \approx dC\)

    所以有\(dC \leq |dx \pm dy| \leq |dx| + |dy| \leq \epsilon_x + \epsilon_y\)

    所以在考虑多个数求和时，只需要把所有数都保留到误差最大的数的后一到两位计算即可。例如\(3.1,3.5429,3.5344\)，可以变为

    乘积

    乘积的绝对误差

    \(|dc| = |x|\epsilon_x + |y|\epsilon_y\)

    相对误差

    \(|\delta c| = |\frac {dx}x| + |\frac{dy}{y}|\)

    商

    \(\frac x y\)

    \(|dc| = |\frac {ydx - xdy} {y^2}|\)

    \(\delta c = |\delta x| + |\delta y|\)

    幂

    \(c = x^p,p\geq 1\)

    \(|dc| = |p x^{p-1} dx|\)

    \(|\delta c| = p|\delta x|\)

    总结

    运算次数导致误差累积

    大量运算时，误差的累积会很大

    对多项式\(P(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0\)，可改写为\(P(x) = x(a_n + x(a_{n-1} + x(\dots))) + a_0\)，极大减少运算次数

    大数吃小数

    计算机中，会发生对齐，要防止浮点数大数吃小数，计算机中小指数会先变成大指数，可能会导致截断。例如采用三位浮点数以截断方式进行运算：

    \(y = 1 + \frac 1 2 + \frac 1 3 + \dots + \frac 1 {10}\)

    从左到右：y = 2.91。从右到左：y=2.92。为了防止大数吃小数，所以应该从右向左算

    相近数相减易失有效位

    两个相近数相减，容易丢失有效位。例如\(y = \sqrt{1001} - \sqrt{1000}\)，直接计算得\(0.02\)，化为\(y = \frac 1 {\sqrt{1001} + \sqrt{1000}}\)，则算得\(y = 0.01581\)，更精确

    分母很小时差的绝对误差大

    \(\frac x {y^*} = \frac {x^* + \Delta x} {y^*} = \frac {x^*} {y^*} + \frac {\Delta x} {y^*}\)=商值+dc，\(dc = \frac {\Delta x} {y^*}\)，若\(y^*\)过小，则误差会被放大

    尽量避免绝对值太小的值做分母

    避免分母为两个相近数相减

    当分母为两个相近数相减时，会因为有效数字丧失而出现分母较小的情况

    数学问题解的误差估计

    解\(y = f(x_1, x_2, ..., x_n)\)，假定f在该点可微，则误差较小时，解的误差限可估计如下：

    \[|df(x_1, x_2, ..., x_n)| \\ \approx |\frac {\partial f} {\partial x_1}|\Delta x_1 + |\frac {\partial f} {\partial x_2}|\Delta x_2 + ... + |\frac {\partial f} {\partial x_n}|\Delta x_n \\ \leq \sum |\frac {\partial f} {\partial x_i}| |\epsilon_i|\]

    令\(A_i = |\frac {\partial f} {\partial x_i}|\)

    相对误差：

    \[|\delta f| \approx |\sum\frac{\partial f} {\partial x_i} \frac {x_i} f \frac {\Delta x_i} {x_i}|\leq \sum|\frac{\partial f} {\partial x_i}\frac {x_i}f|\delta_i\]

    令\(B_i = |\frac{\partial f} {\partial x_i}\frac {x_i}f|\)

    舍入误差对结果影响小的算法称为稳定的算法，否则称为不稳定算法
