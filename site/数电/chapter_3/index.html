    逻辑代数基础

    逻辑变量与逻辑函数

    逻辑变量：逻辑代数中的二值变量，取值只有0和1

    与（逻辑乘法）：\(A\cdot B, AB\)（也有\(A\land B, A\& B, A\cap B\)）

    或（逻辑加法）：\(A+B\)（也有\(A\cup B, A\lor B, A|B\)）

    非（逻辑取反）：\(Y = \bar A\)

    也可以采用波形图表示逻辑运算（将一个或多个输入与输出时间轴对齐画图）

    表示方式总结：

    • 名称
    • 真值表
    • 逻辑表达式
    • 逻辑符号（门）
    • 波形图

    运算次序：单变量的非>逻辑乘>逻辑加。括号内大于括号外。多变量非运算相当于加了括号

    逻辑函数

    概念：

    • 逻辑函数F=f(A,B,C,D)
    • 逻辑自变量（输入逻辑变量）A,B,C,D
    • 逻辑因变量（输出逻辑变量）F

    逻辑函数相等满足的条件：

    • 输入变量一致
    • 任何输入变量赋值相同，得到输出相同

    证明相等：

    • 真值表一样

    代数公理

    乘

    • 0*0=0
    • 0*1=1*0=0
    • 1*1=1

    加

    • 0+0=0
    • 0+1=1+0=1

    • 1+1=1

    • A*0=0, A+1=1

    • A*1=A,A+0=A
    • \(A\cdot \bar A = 0, A + \bar A = 1\)
    • 交换律
    • 结合律
    • 两种分配率
    • 幂等律\(A\cdot A = A, A+A = A\)
    • 德摩根律
    • 还原律\(\bar{\bar A} = A\)

    注意：

    • 加法乘法没有消去律

    定理：

    • 合并定理：\(AB + A\bar B = A, (A+B)(A+\bar B) = A\)
    • 吸收定理：\(A+AB = A, A(A+B) = A, A + \bar A B= A + B, A(\bar A +B) = AB\)
    • 添加项定理：\(AB + \bar A C + BC = AB + \bar A C, (A + B) (\bar A + C) (B + C) = (A + B) (\bar A + C),AB + \bar A C + BCD = AB + \bar A C, (A + B) (\bar A + C) (B + C + D) = (A + B) (\bar A + C)\)

    三个重要规则

    代入规则

    同样的逻辑函数可以互相替代

    反演规则

    反函数：两个函数输入变量相同，任何输入，两个输出不同

    求反函数，反演规则：

    • 与运算或运算互换
    • 常量01互换
    • 变量取反

    对偶规则

    • 与运算或运算互换
    • 常量01互换

    得到原式对偶式F'

    如果两个函数相等，则他们的对偶函数也相等：\(F=G\Leftrightarrow F'=G'\)

    无变量的情况下\(F'\)可以等于\(\bar F\)

    复合运算

    复合逻辑门：实现复合逻辑运算的逻辑门

    • 与非：\(\overline{A\cdot B}\)
    • 或非：\(\overline {A + B}\)

    • 与或非：\(\overline {\overline {A\cdot B} + \overline {A + B}}\)

    • 异或：\(\overline A B + A\overline B\)。对与有分配率

    多变量异或，取值为1的个数为奇数时为1，为偶数时为0。异或函数中，任意变量取反，则输出取反

    因果互换关系\(A\oplus B = C\)成立，则有\(A\oplus C = B\)成立。对多变量也一样

    • 同或：\(F = A \odot B = AB + \bar A\bar B\)

    对或有分配率

    多变量同或，取值为0的变量为偶数个时为1，为奇数个时为0

    具有因果互换关系

    逻辑完备性

    常见逻辑完备集

    • 与、或、非
    • 与非
    • 或非
    • 与或非

    最小项：是n个变量的积，每个变量都以原变量或反变量的形式出现一次。只有一组变量的取值使他的值为0。

    最大项：是n个变量的积，每个变量都以原变量或反变量的形式出现一次。只有一组变量的取值使他的值为1。任意最大项和为1，所有最大项乘为0

    \(\overline m_i = M_i\)

    变量数相同时，编号相同的最小最大项为反函数

    已知真值表写函数：

    • 最小项之合式（取值为1的最小项）
    • 最大项之积式（取值为0的最大项）

    给定F，可直接写出两种\(\overline F\)的表达式

    化简

    与或最简表达式（与或是先与再或的式子，类似\(A\bar B+C\)。或与可以对偶，化简后再对偶回去）：

    • 每个与项变量数最少
    • 与项最少

    化简方法：

    • 代数化简
    • 并项法：用合并定理\(AB + A\bar B = A\)和互补律\(A + \bar A = 1\)。例如\(A\bar B \bar C + \bar A \bar C + B\bar C = \bar C(A\bar B + \bar A + B) = \bar C (A\bar B + \overline {A\bar B}) = \bar C\)
    • 消项法：利用吸收定理\(A + AB = A\)和添加项定律\(AB+\bar AC + BC = AB+\bar AC\)。例如：\(\bar AC\bar D + \bar C E +DE+\bar ADE= \bar AC\bar D + E(\bar C + D) + \bar ADE = C\bar D\bar A + \overline {C\bar D} E + \bar A DE= C\bar D\bar A + \overline {C\bar D} E = \bar AC\bar D + \bar CE +DE\)
    • 消元法：利用吸收定律消\(A+\bar AB = A+B\)去多余项。例如\(C\bar D + \bar C E + DE + \bar A B \bar E=C\bar D + (\bar C + D)E + \bar A B \bar E=C\bar D +\overline{C\bar D} E + \bar AB\bar E = C\bar D + E +\bar AB\bar E = C\bar D + E +\bar AB\)
    • 配项法：利用互补律\(A+\bar A = 1\)，重叠律\(A+A=A\)，添加项定理\(AB+\bar AC + BC = AB+\bar AC\)等。例如\(A\bar B + B\bar C + \bar BC + \bar AB = A\bar C + B\bar C +(A+\bar A)\bar B C + \bar AB(C+\bar C) = A\bar B + B\bar C + A\bar B C + \bar A\bar B C + \bar ABC + \bar A B\bar C = A\bar B(1+C) + B\bar C(1+\bar A) + \bar AC(\bar B+B) = A\bar B +B\bar C + \bar AC\)
    • 卡诺图化简
    • 系统化简法（Q-M法）

    与非-与非表达式：

    • 表达式非最少（不包含单变量上的非）
    • 每个非下的变量个数最少

    方法：

    • 用求反加非（先整体加两个非），变化已化简的或与最简表达式

    或非-或非表达式：

    • 表达式非最少
    • 每个非下变量数最少

    方法：

    • 用求反加非（先整体加两个非），变化已化简的与或最简表达式

    \(F = \bar A \bar B + \bar A BD + A\bar B \bar D\) \(=(\bar A + \bar D)(\bar B + D)\) \(=\overline{\overline{(\bar A + \bar D)(\bar B + D)}}\) \(=\overline{AD\cdot B\bar D}\)

    与或非表达式：

    • 与项个数最少
    • 每个与项所含变量个数最少

    对\(\bar F\)的最简与或式求反

    先求反函数的最简与或式，再求反即可

    逻辑函数卡诺图化简法

    逻辑相邻最小项：两个变量个数相同最小项，只有一个变量不同（即互反），其余均相同，则称为逻辑相邻最小项。

    两个逻辑相邻最小项或在一起可以消掉一项

    • 行列坐标均为格雷码
    • 上下、左右逻辑相邻（注：同一行最左最右也逻辑相邻，同一列最上最下也逻辑相邻）
    • n变量卡诺图有\(2^n\)个格子，每个小方格表示一个最小项

    卡诺图表示逻辑函数：

    • 根据变量个数画出卡诺图的框
    • 根据函数表达式形式（表达式或真值表），填写卡诺图框（填0或1）（例如化简为主析取范式或与或式后，根据最小项填写对应位置为1）

    若给出最大项之积，则对应位置填0，其余位置填1

    卡诺图性质：

    • 所有格都是1，则F=1
    • 所有格都是0，则F=0

    对卡诺图同态的运算

    • 卡诺图求反：所有0，1互换，则得到反函数\(\bar F\)的卡诺图（卡诺图反演）
    • 卡诺图相加：若\(F = F_1 + F_2\)，则\(K(F) = K(F_1) + K(F_2)\)（对应格作和）
    • 卡诺图相乘：\(F = F_1 \cdot F_2\)，则\(K(F) = K(F_1) \cdot K(F_2)\)
    • 卡诺图异或：\(F=F_1 \oplus F_2\)，则\(K(F) = K(F_1) \oplus K(F_2)\)

    对于复杂函数，可先拆为简单函数，再进行同态运算

    化简：

    • 将相邻格子圈在一起，代表将逻辑相邻的最小项合并（或），从而产生一个与项并消去互补变量（圈只能为矩形。两个相邻格子消一个变量，四个相邻格子消两个变量，八个相邻格子消四个变量...）
    • 最小覆盖原则：卡诺图上，填1的小格都要被卡诺圈覆盖（每个1可以被覆盖多次）。满足前面条件下，卡诺圈要尽量少。每个卡诺圈包含1的小格个数尽量多，但必须是\(2^i\)个，每个卡诺圈至少要包含一个其他卡诺圈没有的小格

    最简或与式：

    1. 原函数圈0进行合并（即对反函数化简，得到反函数最简与或式）
    2. 对结果进行反演（1、0对调，变量求反，与或对调）

    或直接圈0写F的最简或与式

    利用公共项对两个函数进行化简（就是公共项逻辑门可以共用）

    非完全描述逻辑函数

    • 完全描述逻辑函数：对输入变量每一组取值都有确定函数值
    • 非完全描述逻辑函数：不是被完全定义的，某些变量的取值，函数有确定值，但有些则没有（这些输入项称为无关项或任意项，对应最小项用小写的\(\phi\)或d表示，最大项用\(\Phi\)或D表示。也可以用约束表示）

    例如：\(\Pi M(1,5,6) \cdot \Pi \Phi(3,4)\)

    \(F(A,B,C)= \Pi M(1,5,6),(A +\bar B + \bar C) (\bar A +B+C) = 1\)
