    组合逻辑电路

    概述

    • 组合逻辑电路：无反馈环节，无记忆功能，由逻辑门组成，输出仅由当前时刻输入决定
    • 时序逻辑电路：有反馈环节，有记忆功能，输出与之前的状态有关

    组合电路是由逻辑门组成的多输入、多输出（或单输出）的逻辑电路

    根据已知逻辑电路，利用逻辑表达式、真值表分析逻辑功能。一般步骤：

    1. 电路
    2. 逻辑式
    3. 真值表
    4. 分析功能

    表决电路：多数通过、一致通过

    加法器

    由五个逻辑门组成的二输入二输出的逻辑电路

    \[S = A \oplus B, C = AB\]

    表示两个一位二进制数\(A,B\)相加得到的二进制数\(\underline{SC}\)。这里没有考虑进位，称为半加运算，电路称为半加器。

    全加器：三输入二输出

    将多个全加器组成更多位全加器

    输出角标数字越大是越高位

    可以得x3电路（看ppt）

    减法器（看ppt）

    \[S_3S_2S_1S_0 = A_3A_2A_1A_0 + \bar B_3\bar B_2 \bar B_1 \bar B_0 + 1\]

    例：设计一个电路，四位全加器，以及受M控制。M=0，实现四位加法，M=1，实现四位减法

    减法用补码表示

    M=0:\(\underline{S_3S_2S_1S_0} = \underline{C_3C_2C_1C_0} + \underline{D_3D_2 D_1 D_0} + 0\)

    M=1:\(\underline{S_3S_2S_1S_0} = \underline{C_3C_2C_1C_0} + \underline{\bar D_3\bar D_2 \bar D_1 \bar D_0} + 1\)

    则有：\(\underline{B_3B_2B_1B_0} = \bar M(\underline{D_3D_2D_1D_0}) + M (\underline{\bar D_3\bar D_2 \bar D_1 \bar D_0})\)，\(\underline{A_3A_2A_1A_0} = \underline{C_3C_2C_1C_0}\)，\(CI=M\)

    数据选择器

    选择一个输入信号进行输出

    \[F = E(\bar A_1 \bar A_0 D_0 + \bar A_1 A_0 D_1 + A_1 \bar A_0 D_2 + A_1 A_0 D_3)\]

    \(\bar E = 1\)时，输出为0，否则输出靠\(A_0,A_1\)控制

    原理看ppt，有两种

    选择器自扩展

    实现八输入选择器：

    74153是两个不连接的四选一选择器，下方输入为\(D_{10},D_{11},D_{12},D_{13}\)，以及使能\(\bar E_1\)，输出\(F_1\)。上方输入为\(D_{20},D_{21},D_{22},D_{23}\)，以及使能\(\bar E_2\)，输出\(F_2\)。控制信号端口\(A_0,A_1\)

    输入信号为\(D_0,D_1,\dots,D_7\)，以及\(A_0,A_1,A_2\)，输入信号直接连输入。低位控制信号直接连\(A_0,A_1\)，\(A_2\)控制输出上方或下方信号，所以\(A_2\)直接连\(\bar E_1\)，经过反相连\(\bar E_2\)

    两输出根据或门连接（因为无效信号为0），再输出

    多路分配器

    在通道选择端作用下，将一个数据分别送到多个输出端

    \(\begin{cases} Y_0 = D \bar A \bar B\\ Y_1 = D \bar A B\\ Y_2 = D A \bar B\\ Y_3 = D A B \end{cases}\)

    编码器

    八线-三线普通编码器：输入八个信号，输出3位二进制数。输入高电平有效，且输入只允许一个1

    优先编码器：设计10-4优先编码器，输入\(I_0\sim I_9\)，输入低电平有效，输出反码，输出优先级为\(I_9I_8\dots I_0\)...

    74LS148为8线-3线优先编码器，输入低电平有效、输出反码

    \(\bar S = 1\)时，输出全是1。S为使能端

    \(\bar{Y_S} = 0\)时无有效输入。\(\overline {Y_{EX}} = 0\)时有有效输入

    优先级\(I_7I_6\dots I_0\)

    译码器

    是编码的逆过程，输入二进制代码，输出一组信号

    将n位二进制代码，译为特定含义\(2^n\)哥信号的译码器称为二进制译码器（或最小项译码器）

    设计3线-8线译码器，输入原码、输出高电平有效

    （默认A是最高位）

    74LS138

    选通端为100时有效。输出0为有效

    2-10进制译码器

    74LS42：输入原码，输出高电平有效

    显示译码器：把4位二进制代码，译为数码显示器所需信号。如7段数码显示器，则需要7个显示信号。分共阳极（需要负极低电平发光）和共阴极（需要正极高电平发光）

    • 74LS47：低电平有效
    • 74LS48：高电平有效

    • \(\bar {LT}\)为0时，所有数码管亮（测试）
    • \(\overline{BI/RBO}\)：灭零输出
    • \(\overline{RBI}\)：灭零输入，为低电平且\(A_3A_2A_1A_0=0\)时，数码管不显示（可接前一位的灭零输出）

    数值比较器

    功能：比较两个相同位数的二进制数的大小

    一位比较器：两输入，三输出

    74LS85集成化4位比较器

    低位输出连到高位输入

    若输入四个数都相等，则输出由级联输入\(I_{A>B}, I_{A=B}, I_{A<B}\)决定

    设计

    • 小规模集成电路：各种逻辑门
    • 中规模集成电路：译码器、选择器
    • 大规模集成电路：存储器、可编程器件

    步骤：

    1. 实际问题
    2. 真值表
    3. 逻辑表达式
    4. 化简
    5. 逻辑电路图

    中规模集成电路设计：采用中规模器件（例如译码器、数据选择器、全加器）

    数据选择器、译码器设计电路

    数据选择器\(F=m_0D_0 + m_1D_1 + \dots + m_nD_n\)

    输入变量数与控制端相同时，输入端输入0或1。若4选1实现3变量函数，则需分离变量，部分接到输入端

    一般情况下，n变量逻辑函数可用\(2^n\)或\(2^{n-1}\)选择器实现。最小项之和容易用译码器实现

    译码器：

    3-8可以实现任何3变量逻辑函数（因为输出是最小项，将最小项化为与非或其他形式即可实现）

    4-16可以实现任何4变量逻辑函数

    8421BCD转5421BCD

    前五个数字：

    \(\underline{S_3S_2S_1S_0} = \underline{D_3D_2D_1D_0} + 0000\)

    后五个数字：

    \(\underline{S_3S_2S_1S_0} = \underline{D_3D_2D_1D_0} + 0011\)

    即可以确定\(\underline{A_3A_2A_1A_0} = \underline{}{D_3D_2D_1D_0}\)

    \(\underline{B_3B_2B_1B_0}\)为一个关于输入的函数，化简后输入即可。

    竞争和冒险

    产生原因：实际逻辑门存在传输延迟时间

    例如两信号到达与门时间不同，存在竞争，可能(不一定每种情况都产生)产生干扰脉冲，称为冒险

    竞争产生时不一定产生冒险。冒险分为功能冒险和逻辑冒险：

    • 逻辑冒险：多个输入信号中某一个发生变化时（路径不同导致延迟不同）
    • 功能冒险：多个输入信号中多个发生变化时，由于变化快慢不同，导致在某个输入端产生时间差

    卡诺图中，两圆圈相切时，可能产生逻辑冒险。所以可以引入冗余项，消除逻辑冒险

    功能冒险解决：

    • 输出端加入滤波电容
    • 加入禁止脉冲
