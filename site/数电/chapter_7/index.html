    时序逻辑电路

    概述

    存储电路的输出反馈到组合逻辑电路的输入端

    x为时序逻辑输入，z为输出

    w为存储电路的驱动，y是存储电路的输出

    输出方程：

    \[Z(t_n) = F[X(t_n), Y(t_n)]\]

    驱动：

    \[W(t_n) = H[X(t_n),Y(t_n)]\]

    状态方程：

    \[Y(t_{n+1}) = G[W(t_n), Y(t_n)]\]

    按存储单元状态变化的特点：

    • 同步时序逻辑电路：所有触发器状态变化由同一CP控制
    • 异步时序逻辑电路：触发器变化不同时，有先有后

    按输出信号特点：

    • Mealy型：输出信号取决于存储电路与输入变量
    • Moore型：输出仅仅取决于存储电路的正态

    时序电路分析方法

    一般步骤：

    • 写方程
    • 时钟方程（同步时序逻辑电路不用写）
    • 驱动方程
    • 输出方程
    • 求状态方程
    • 将写出的方程带入触发器特性方程
    • 列状态转换真值表
    • 设初态
    • 求次态
    • 列状态转换真值表

    偏离状态：指大循环外的状态

    时序逻辑电路分析

    普通寄存器

    上升沿触发的D触发器构成的寄存器74LS175：

    时钟方程：

    \[CP_0 = CP_1 = CP_2 = CP_3 = CP\]

    驱动方程：

    \[D_0 = D_0, D_1 = D_1, D_2 = D_2, D_3 = D_3\]

    次态方程：

    \[Q^{n+1}_i = D_i\]

    异步清零，同步置数

    移位寄存器

    用途：存数、数据串行、并行转换，数值运算，数据处理

    单向位移寄存器CC4015

    只能一次右移一位

    双向位移寄存器74LS194

    全为下降沿触发的RS触发器

    https://www.bilibili.com/video/BV1np4y197sR/?spm_id_from=333.337.search-card.all.click&vd_source=19f4d03b8dddc2fc0cd0b04bcba5eb31

    计数器

    同步二进制加法计数器。下降沿时翻转。前面所有位在下降沿时反转

    减法计数器。下降沿时翻转

    十进制计数器：改变输入信号，列式子确定什么时候进位

    异步加法计数器。末尾+1。每位从1变0时，朝高进制发送进位信号。结构简单，可自启动

    缺点：

    • 因为是逐级翻转，所以工作较慢
    • 某些状态译码时会产生冒险、竞争

    加减可逆计数器74191（了解）

    • \(\bar U/D\)加减控制，0加，1减
    • \(\bar S\)使能控制，为1时为保持状态，0时为计数
    • \(\overline {LD}\)预置数端，直接将输入作为输出

    74LS194：双时钟同步十六进制加/减计数器，采用T'触发器（了解）

    移位计数器：在移位寄存器基础上增加反馈电路

    环形计数器

    用最左边的作为有效循环，不需要译码。缺点，不能自启动（方法：修改反馈逻辑、进入无效状态时异步置位恢复有效状态）

    但仍然利用率低，16个状态只有4个有效

    扭环形计数器

    不能自启动。有效循环中，每次只有一个信号改变（无竞争冒险）

    可以修改反馈逻辑达到自启动

    同步十进制加法计数器74160

    • \(\overline{CR}=0\)异步清零
    • \(\overline{LD}=0\)同步预置数
    • \(CT_P, CT_T\)均为1时在时钟上升沿进行记数，否则保持
    • CO为进位输出端
    • \(D_0-D_3\)为预置数输入端

    同步四位十六进制计数器74161。功能与74160相同，进制不同

    异步二-五-十进制计数器74290

    • \(\overline {CP_0}\)为\(FF_0\)的时钟，与\(FF_0\)构成二进制计数器
    • \(\overline{CP_1}\)为\(FF_1,FF_3\)的时钟，\(FF_2\)的时钟为\(Q_1\)。后三个构成五进制计数器
    • \(S_{0(1)}\)和\(S_{0(2)}\)为1时，输出为0
    • \(R_{9(1)}\)和\(R_{9(2)}\)为1时，输出为9
    • \(Q_0\)与\(\overline{CP_1}\)连接时，变为8421码十进制计数器
    • \(Q_3\)连到\(\overline{CP_0}\)时，为5421码十进制计数器

    同步四位二进制计数器CC4516

    • CR=1时，异步置0
    • LD=1时，异步置数
    • \(U/\bar D = 1\)时加计数，0时减计数
    • \(\overline{CI}\)为使能端，=1时保持，0时计数

    设计任意进制计数器。已有N进制计数器，设计M进制计数器

    • \(M<N\)只用一片N即可。跳过状态：
    • 清零法（通过清零端实现）。有瞬态
    • 置数法（通过置数端实现）。这是同步置数，无瞬态（相同电路会比清零法多一个状态，看ppt）。或者通过CO连接\(\overline{LD}\)，置数到某个数
    • 反馈归零法：跳过\(N-M\)个状态。同步清零方式在第M-1个状态有效（无瞬态），异步清零方式在第M个状态有效（有瞬态）
      • 异步清零：从\(S_0\)到\(S_M\)状态时回到\(S_0\)。列出\(S_M\)的二进制代码，然后链接相应输出连接到异步清零端。缺点：存在瞬态，易导致触发器的误动作，该电路不可靠。解决方法：对清零信号增加基本RS触发器，或采用同步清零
    • 预置数法：同样对同步、异步有是否有瞬态的区别。可用输出信号置数，或输出CO置数
    • \(M>N\)
    • 串行进位方式（适合于M可分解为乘积的形式）：低位片的进位输出作为高位片的时钟输入
    • 并行进位方式（适合于M可分解为乘积的形式）：低位片的进位输出作为高位片的使能信号，两片时钟输入端同时接输入信号
    • 整体清零（先设计大进制计数器，再通过整体清零做到指定进制。异步清零的进位信号应该放在瞬态之前，同步可以直接引出）
    • 整体置数

    注：考试时A是最高位

    顺序脉冲发生器

    需要产生一个节拍控制脉冲，产生其的电路称为节拍脉冲发生器（或顺序脉冲发生器）

    计数器型：由计数器、译码器构成。四进制计数器后接一个译码器。缺点：触发器翻转时刻不完全一致，可能存在干扰脉冲

    移位寄存器型：用环形或扭环形计数器构成顺序脉冲发生器，可避免译码过程中出现干扰脉冲

    时序逻辑电路设计

    小规模：触发器、门电路

    中规模：计数器、寄存器等

    flowchart TD
A[实际问题]--逻辑抽象-->
B[原始状态图/表]--状态化简-->
C[最简状态图/表]--状态分配-->
D[编码状态图/表]--次态卡诺图-->
E[逻辑方程组]-->G(是否自启动)
--是-->F[逻辑图]
G--否，进行调整-->E

    设计1111序列检测器，连续输入四个或四个以上的1时，输出1，否则输出0

    输入一位A，输出一位F

    状态：

    • \(S_0\)：没有1个1
    • \(S_1\)：有1个1
    • \(S_2\)：有2个1
    • \(S_3\)：有3个1
    • \(S_4\)：有4个以上1

    原始状态图

    flowchart TD
  A["$$S_0$$"] --0/0-->A
  A --1/0--> B["$$S_1$$"]
  B --0/0--> A
  B --1/0--> C["$$S_2$$"]
  C --0/0--> A
  C --1/0--> D["$$S_3$$"]
  D --0/0--> A
  D --1/1--> E["$$S_4$$"]
  E --0/0--> A
  E --1/1--> E

    用表表示：

    A 0 1
    \(S_0\) \(S_0\)/0 \(S_1\)/0
    \(S_1\) \(S_0\)/0 \(S_2\)/0
    \(S_2\) \(S_0\)/0 \(S_3\)/0
    \(S_3\) \(S_0\)/0 \(S_4\)/1
    \(S_4\) \(S_0\)/0 \(S_4\)/1

    等价状态：输入相同时，输出相同，次态也相同，称为等价状态。具有传递性

    化简：保证逻辑功能不变，消去多余状态，获得最简状态表

    • 观察法
    • 隐含表法（适合计算机）

    以上\(S_3,S_4\)等价，进行合并

    A 0 1
    \(S_0\) \(S_0\)/0 \(S_1\)/0
    \(S_1\) \(S_0\)/0 \(S_2\)/0
    \(S_2\) \(S_0\)/0 \(S_3\)/0
    \(S_3\) \(S_0\)/0 \(S_3\)/1

    再进行状态分配。四个状态可以用俩触发器实现。用二进制表示状态。需要：

    • 需要多少个触发器
    • 如何对各状态编码

    编码规则：

    1. 相邻分配规则：两个以上状态在同一个给定的输入条件下具有相同次态，这些原状态尽可能安排“逻辑相邻”的代码。
    2. 同一个状态在逻辑相邻的组合驱动下有两个以上的次态时，这些次态尽量安排逻辑相邻的代码
    3. 输出相同的状态，最好也安排为逻辑相邻的代码

    优先级\(1>2>3\)

    1. 由规则1：\(S_1,S_2\)逻辑相邻
    2. 由规则2：\(S_0S_1\)，\(S_0S_2\)，\(S_0S_3\)逻辑相邻
    3. 由规则三：\(S_0S_1S_2\)

    所以\(S_0S_1S_2S_3\)这样安排比较好

    A 0 1
    00 00/0 01/0
    01 00/0 11/0
    11 00/0 10/0
    10 00/0 10/1

    画次态卡诺图

    \(Q_1^nQ_0^n\)\A 0 1
    00 00/0 01/0
    01 00/0 11/0
    11 00/0 10/0
    10 00/0 10/1

    分为三个卡诺图，进行画圈合并

    \(Q_0\)：

    \(Q_2^nQ_1^n\)\A 0 1
    00 0 0
    01 0 1
    11 0 1
    10 0 1

    对\(Q_1, Z\)也这样，可得：

    \(Q_1^{n+1} = Q_0^n A + Q_1^n X,Q_0^{n+1} = \overline{Q_1^n} A, Z = Q_1^n + Q_0^n X\)，采用对应触发器即可

    序列信号发生器设计

    序列信号：周期性按某种规律排列的一串0、1

    • 移位寄存器
    • 计数器

    对移位寄存器，常取其一个触发器状态为输出

    例：设计序列信号为101100的移位寄存器型序列信号发生器

    需要3个寄存器

    \(Q_0\)输出为101100

    \(Q_1\)输出往后移动一位：010110

    \(Q_2\)输出往后移动两位：001011

    需要检查是否有相同状态，发现没有

    可以画次态卡诺图

    \(Q_2\backslash Q_1Q_0\) 00 01 11 10
    0 xx/x 01/0 11/0 10/1
    1 00/1 01/1 xx/x 10/0

    分为三个图化简得：

    \(Q_2^{n+1} = \overline{Q_1^n}, Q_1^{n+1} = Q_0^n, Q_0^{n+1} = Q_2^n \overline{Q_1^n} + \overline{Q_2^n}\overline{Q_0^n}\)

    设计过程中，若有重复状态，则加一个触发器即可，多一个状态

    对计数器，

    • 采用计数器+数据选择器+JK触发器（缓冲作用）。更改预置数端即可改变输出
    • 计数器和输出电路组成：计数器+输出网络。改变输出电路即可得到不同序列
