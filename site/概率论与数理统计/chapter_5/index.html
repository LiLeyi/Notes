    数学期望和方差

    数学期望

    定义

    • 离散情况

    设X有概率分布\(p_j=P(X=x_j),j=0,1,\dots\)，只要级数\(\Sigma_{j=0}^\infty|x_j|p_j\)收敛，就称\(E(X)=\Sigma_{j=0}^\infty x_jp_j\)为X或分布{p_j}的数学期望或均值

    在定义中要求\(\Sigma_{j=0}^\infty|x_j|p_j\)收敛的原因是要使级数有确切的意义。当所有的\(x_j\)非负时, 如果级数是无穷, 定义的E(X) 也有明确的意义, 它表明X的平均取值是无穷。这时也称X的数学期望是无穷。不难看出，只取有限个值的随机变量的数学期望总是存在的。

    可得\(\{p_j\}\)的重心为\(\mu\)

    • 连续情况

    若X为有概率密度f(x)的随机变量，若有\(\int_{-\infty}^\infty |x|f(x)dx<\infty\)，就称\(\int_{-\infty}^\infty xf(x)dx\)为X或f(x)的数学期望或均值

    • 一般随机变量

    可以用离散随机变量期望来逼近期望

    取\(\epsilon>0\)，把数轴分为长度为\(\epsilon\)的小区间，定义离散型随机变量\(X(\epsilon)\)，当X取值为第i个小区间时，\(X(\epsilon)\)定义为小区间左端点的值

    \[X(\epsilon)=\begin{cases} 0,&X\in [0,\epsilon),\\ \epsilon,&X\in [\epsilon,2\epsilon),\\ \dots,\\ -\epsilon,&X\in[-\epsilon,0),\\ -2\epsilon,&X\in[-2\epsilon,-\epsilon),\\ \dots \end{cases}=X(\epsilon)\\=floor(X/\epsilon)\cdot \epsilon\]

    设X为随机变量，如果\(E(X(\epsilon))\)存在，且\(E(X)=\lim_{\epsilon\to 0^+}E(X(\epsilon))\)存在，则定义\(E(X)\)为X的数学期望

    在连续或离散情况下，与上文定义相同。

    常见分布的期望

    • 两点分布：p
    • 二项分布：np
    • 泊松分布：\(\lambda\)
    • 几何分布：\(\frac{1}{p}\)
    • 指数分布：\(\frac{1}{\lambda}\)
    • \(\Gamma\)分布：\(\frac \alpha \lambda\)
    • 超几何分布：

    对称分布的期望

    若概率密度关于\(x=\mu\)对称，则均值为\(\mu\)

    随机向量函数的数学期望

    • 连续

    若X有联合密度f(x)，实函数g(x)使得

    \[\int\int\cdots\int_{R^n}|g(x)|f(x)dx_1dx_2\cdots dx_n<\infty\]

    则Y=g(X)有数学期望

    \[E(Y)=\int\int\cdots\int g(x)f(x)dx_1dx_2\cdots dx_n\]
    • 离散

    若X是离散型随机变量，有概率分布

    \[p_{j_1,j_2,\cdots,j_n}=P(X=(x_{j1},x_{j2},\cdots,x_{jn})),j_1,j_2,\cdots,j_n\geq 1\]

    实函数h(x)，使得

    \[\Sigma_{j_1,j_2,\dots,j_n} |h(x_{j_1},x_{j_2},\dots,x_{j_n})|p_{j_1,j_2,\dots,j_n}<\infty\]

    则Y=h(X)有数学期望

    \[E(Y)=\Sigma_{j_1,j_2,\dots,j_n}h(x_{j_1},x_{j_2},\dots,x_{j_n})p_{j_1,j_2,\dots,j_n}\]

    性质

    1. \(E(c_0+c_1X_1+c_2X_2+\dots+c_nX_n)=c_0+c_1E(X_1)+c_2E(X_2)+\dots+c_nE(X_n)\)
    2. 若\(X_1,X_2,\dots,X_n\)相互独立，则\(E(X_1X_2\dots X_n) = E(X_1)E(X_2)\dots E(X_n)\)
    3. 若\(X_1\leq X_2\)，则\(E(X_1)\leq E(X_2)\)
    4. \(P(A)=E(I_A)\)
    5. \(P(X=0)=1 \Leftrightarrow E(|X|)=0\)

    当一个事件发生概率为1，称其以概率1发生，记作wp1.，即with probability 1，也称作几乎处处或几乎必然发生，也可以简写为a.s.

    中心矩与原点矩

    \(E(X^m)<\infty\)则称为m阶原点矩，\(E((X-E(X))^m)\)称为m阶中心距，若m大于2，统称为高阶矩

    方差

    定义

    若随机变量X的数学期望有限，就称X的二阶中心矩

    \[E((X-E(X))^2)\]

    为X的方差，记作Var(X)或\(\sigma_{XX}\)。当\(Var(X)<\infty\)，称X的方差有限。称\(\sigma_X=\sqrt{Var(X)}\)为X的标准差

    计算

    • 离散
    \[Var(X)=\Sigma (x_j-E(X))^2 P(X=x_j)\]
    • 连续
    \[Var(X)=\int (x_j-E(X))^2 f(x)dx\]

    f(x)为X的概率密度

    性质

    • \(Var(X)=E((X-E(X))^2)=E(X^2 - 2E(X)X + E(X)^2)=E(X^2) -2(E(X))^2 + (E(X))^2 = E(X^2) - (E(X))^2\)
    • \(Var(a + bX) = b^2Var(X)\)
    • \(Var(X) = E((X-E(X))^2)<E(X-c)^2=Var(X)+(E(X) - c)^2，只要c\neq E(X)\)
    • \(Var(X)=0 \Leftrightarrow P(X=\mu)=1\)
    • \(Var(\Sigma_{k=1}^nX_k) = \Sigma_{i=1}^n \Sigma_{j=1}^n (E(X_iX_j)-E(X_i)E(X_j))\)
    • 若\(X_1,X_2,\dots,X_n\)相互独立，则\(Var(\Sigma X_i) = \Sigma Var(X_i)\)
    • 若\(X_1,X_2,\dots,X_n\)相互独立，且有共同方差\(\sigma^2 < \infty\)，则\(Var(\frac{\Sigma X_i} {n}) = \frac {\sigma^2} {n}\)

    特殊分布的方差

    部分推导使用\(Var(X)=E(X(X-1)) + E(X) + (E(X))^2\)

    • 二项分布：\(np(1-p)\)
    • 泊松分布：\(\lambda\)
    • 几何分布：\(\frac {1-p} {p^2}\)
    • 均匀分布：\(\frac {(b-a)^2} {12}\)
    • 指数分布：\(\frac 1 {\lambda^2}\)
    • 正态分布：\(\sigma^2\)
    • 超几何分布：\(n\frac M N (1 - \frac M N) \frac {N - n} { N - 1}\)

    随机变量标准化

    \(Y = \frac {X - E(X)} {\sigma}，则E(Y)=0,Var(Y) = 1\)

    有关不等式

    Markov 不等式

    对随机变量X和\(\epsilon > 0\)，有

    \[P(|X| \geq \epsilon) \leq \frac 1 {\epsilon^\alpha} E(|X|^\alpha),\forall \alpha > 0\]

    推论：

    • 称\(\alpha=2\)时的Markov不等式为Chebyshev不等式
    • \(P(|X - E(X)|\geq \epsilon) \leq \frac 1 {\epsilon^2} Var(X),\forall \epsilon > 0\)

    内积不等式

    \[|E(XY)|\leq \sqrt{E(X^2)E(Y^2)}\]

    等号成立条件：\(\exist 不全为0的a,b,s.t.P(aX+bY=0)=1\)

    协方差与相关系数

    定义

    当X,Y的期望存在时，且\(E(|(X-E(X))(Y-E(Y))|)\leq \infty\)，则协方差为

    \[E((X-E(X))(Y-E(Y)))\]

    为随机变量X,Y的协方差，记作Cov(X,Y)或\(\sigma_{XY}\)。当协方差为0，则X，Y不相关。计算协方差常用公式：

    \[\sigma_{XY}=E(XY)-E(X)E(Y)\]

    当两个变量标准差均小于\(\infty\)且大于0，则相关系数定义为

    \[\rho_{XY} = \frac {\sigma_{XY}}{\sigma_X \sigma_Y}\]

    也可由以下式子计算

    \[\rho_{XY} = E(\frac {X - E(X)} {\sigma_X} \frac {Y - E(Y)} {\sigma_Y})\]

    性质

    1. \(|\rho_{XY}|\leq 1\)
    2. \(|\rho_{XY}| = 1\Leftrightarrow \exist a,b,s.t.P(Y=a+bX) = 1\)
    3. 若X，Y独立，则X,Y不相关
    4. 相关系数绝对值越大，表示线性相关性越大

    随机向量与矩阵的有关统计特征

    期望

    对随机向量或矩阵取期望，等于对其内部所有元素分别取期望，位置不变

    对常数矩阵A，B与随机矩阵X，有\(E(AXB) = AE(X)B\)

    协方差阵

    若随机向量X的数学期望存在，且每个元素的方程小于无穷，就称

    \((\sigma_{ij})_{n\times n} = E((X - E(X))^T(X-E(X)))\)

    为X的协方差阵，易得\(\sigma_{ij} = Cov(X_i,X_j)\)。若协方差矩阵行列式为0，则称其为退化的。

    性质：

    • 为对称阵
    • 非负定
    • \(退化\Leftrightarrow \exist 不全为0的常数a_0,a_1,\dots,a_n,s.t.P(\Sigma_{i=1}^na_i(X_i-E(X_i)) = 0)=1\)
