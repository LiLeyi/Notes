    随机变量和概率分布

    随机变量

    定义

    \(X:\Omega \to R\)是定义在样本空间\(\Omega\)上的实值函数，即对于每一个样本点\(\omega\)，\(X(\omega) \in R\)

    随机变量的事件

    我们用\(\{X\leq x\}，或X \leq x\)表示事件\(\{\omega | X(\omega) \leq x\}\)。对于实数集的子集A，也可以用\(\{X\in A\}或X \in A\)表示事件\(\{\omega|X(\omega) \in A\}\)...

    离散型随机变量

    若随机变量X只能取有限个值或可列个值，则称X为离散型随机变量（或离散随机变量）

    称\(P(X=x_k)=p_k\)为X的概率分布，称\(\{p_k\}\)为概率分布列，简称分布列，可以用表格等方法表示。

    设函数f(x)定义域为\(\{x_1,x_2,\dots,x_n\}\)，且\(f(x_k)=p_k\)，则称f为随机变量X的概率质量函数

    两点分布（Bernouli分布）

    \[P(X=1)=p,P(X=0)=p-1\]

    p表示试验进行1次成功的概率。称X服从两点分布，记作\(X \sim B(1,p)或b(1,p)\)

    二项分布（Binomial分布）

    \[P(X=k)=C_n^kp^k(1-p)^{n-k}\]

    表示试验进行n次，其中k次成功的概率，p表示进行一次实验成功的概率。称X服从二项分布，记作\(X \sim B(n,p)\)

    泊松分布（Poisson分布）

    \[P(X=k)=\frac {\lambda ^ k} {k!} e^{-\lambda}\]

    表示单位时间内事件发生k次的概率，其中\(\lambda\)为参数，表示单位时间内事件发生的平均数。称X服从参数是\(\lambda\)的Poisson分布，记作\(X \sim Poisson(\lambda)或P(\lambda)\)

    具有可加性，即若两个独立随机变量\(X,Y,且X \sim P(\lambda_1),Y \sim P(\lambda_2),则X+Y\sim P(\lambda_1 + \lambda_2)\)

    超几何分布

    \[P(X=m)=\frac {C_M^m C_{N-M}^{n-m}}{C_N^n}\]

    称X满足超几何分布，记为\(X\sim H(n,M,N)\)。N件产品中恰有M件次品，从中任取n件，其中有m件次品的概率。产品充分多时可以近似为二项分布

    连续型随机变量

    设X为随机变量，若存在非负函数\(f(x)\)使得对任意满足\(-\infty\leq a < b \leq \infty\)的a,b，有

    \[P(a < X \leq b)=\int_a^bf(x)dx\]

    则称X是连续型随机变量，称f(x)为X的概率密度函数，简称概率密度或密度

    性质：

    • \(\int_{-\infty}^{+\infty}f(x)dx=1\)
    • P(x=a)=0,于是\(P(a< X\leq b)=P(a \leq X \leq b)\)
    • 对数集A（严格意义下要求可测性）：\(P(X\in A)=\int_A f(x)dx\)

    均匀分布（Uniform分布）

    \[f(x)=\begin{cases}\frac 1 {b-a}&,x\in (a,b)\\0 &, x \notin (a,b)\end{cases}=\frac 1 {(b-a)} I_{(a,b)}\]

    称X服从区间(a,b)上的均匀分布，记作\(X \sim U(a,b)\)，区间端点也可为闭的。上式中I为示性函数

    指数分布（Exponential分布）

    \[f(x)=\begin{cases}\lambda e^{-\lambda x}&,x\geq 0\\ 0 &,x < 0\end{cases}=\lambda e^{-\lambda x}I_{[0,+\infty)}\]

    称X服从参数\(\lambda\)的指数分布，记作\(X \sim E(\lambda)\)

    非负随机变量：\(P(X<0)=0\)

    无后效性：即已经等待（存活）了多长时间，对还会等待（存活）多长时间没有影响

    \(X\sim E(\lambda) \Leftrightarrow P(X>s+t|X>s)=P(X>t),s,t>0\)

    此定理说明指数分布具有无后效性。如果 X 表示某仪器的工作寿命, 无后效性 (3.5) 的解释是: 当仪器工作了s小时后再能继续工作t小时的概率等于该仪器刚开始就能工作t小时的概率.说明该仪器的使用寿命不随使用时间的增加发生变化,或说仪器是“永葆青春”的.

    正态分布（Normal或Gauss分布）

    \[f(x)= \frac 1 {\sqrt{2 \pi \sigma^2}}exp(-\frac {(x - \mu)^2}{2\sigma^2})\]

    \(\mu\)为平均数，\(\sigma\)为标准差。称x服从参数为\((\mu,\sigma^2)\)的正态分布，记作\(X \sim N(\mu,\sigma^2)\)

    特别地，\(X\sim N(0,1)\)称为标准正态分布，记为\(\varphi(x)=\frac 1 {\sqrt {2 \pi}}exp(- \frac {x^2} 2)\)

    • \(\mu\)是密度中心点和最大值点，密度在两侧对称
    • \(\sigma\)表示密度的宽度，越大越宽（标准差）

    若\(X \sim N(\mu,\sigma^2)\)，则\(aX+b\sim N(a\mu + b,a^2\sigma^2)\)

    Gamma分布

    \[ \Gamma(\alpha)=\int_0^\infty x^{\alpha - 1}e^{-x}dx\\ f(x)=\begin{cases}\frac {\lambda^\alpha}{\Gamma(\alpha)}x^{\alpha - 1} e^{-\lambda x}&,x\geq 0\\0&,x < 0\end{cases} \]

    称X服从参数\((\alpha,\lambda)\)的Gamma分布，记作\(X \sim \Gamma(\alpha,\lambda)\)

    \(\Gamma(1,\lambda)=E(\lambda)\)

    在气象学中, 干旱地区的年、季或月降水量被认为服从\(\Gamma\)分布, 指定时 间段内的最大风速等也被认为服从\(\Gamma\)分布.

    在 Poisson 分布的例子中, 用 \(S_k\) 表示从开始到观测到第 k 个\(\alpha\)粒子的时间, 可以证明 \(S_k\) 服从 \(\Gamma(k, λ)\) 分布

    Reileigh分布

    X,Y均为独立的连续随机变量，均符合正态分布，且均值均为0，方差相同，则\(Z=\sqrt{X^2 +Y^2}\)服从Reileigh分布

    \[f(z)=\frac z {\sigma^2}exp(-\frac {z^2} {2\sigma^2}),z\geq 0\]

    概率分布函数

    对随机变量X，称x的函数

    \[F(x)=P(X \leq x),-\infty \leq x \leq +\infty\]

    称为X的概率分布函数或分布函数，或累积分布函数

    • 连续随机变量：对概率密度函数f(x)，\(F(x)=\int_{-\infty}^x f(x)dt\)为其分布函数。分布函数必连续，若不连续则不为连续随机变量
    • 离散随机变量：\(F(x)=P(X\leq x)=\Sigma_{j:x_j\leq x}p_j\)，为单调不减的阶梯函数

    特点：

    • 单调不减，至少右连续
    • \(F(-\infty)=0,F(+\infty)=1\)

    对于有限和可列集A，且数集A中任意两点间距离大于正数\(\delta\)，若在A的外导数\(F'(x)\)存在且连续，则

    \[f(x)=\begin{cases}F'(x)&,x \notin A\\0 &, x \in A\end{cases}\]

    是X的密度函数
