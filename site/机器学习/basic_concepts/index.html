    基本术语

    数据集

    N个示例组成的集合

    示例，样本（sample）

    示例是一条记录，关于一个事件或者一个对象的描述，拥有标记信息的示例称为样例

    属性，特征

    反映事件和对象在某一各方面的表现或性质的事项

    属性空间，样本空间，输入空间

    属性，样本，输入所张成的空间

    学习，训练

    从数据中学得模型的过程

    标记

    关于示例结果的信息

    标记空间，输出空间

    所有标记的集合称为标记空间或输出空间

    分类，回归

    若预测的是离散值，此类学习的任务称为分类，若为连续值，称为回归

    测试，测试样本

    学得模型后，使用其进行预测的过程称为测试，被预测的样本称为测试样本

    监督学习（supervise learning），无监督学习（unsupervise learning）

    根据训练数据是否拥有标记信息，学习任务可大致分为两类，监督学习和无监督学习，分类和回归是前者的代表，聚类是后者的代表

    归纳（induction），演绎（deduction）

    前者是从特殊到一般的泛化（generalization），后者则是从一般到特殊的特化（specialization）过程，即从基础原理推演出具体情况

    分类

    • 是否有监督
    • 有监督学习
    • 无监督学习
    • 半监督学习
    • 强化学习
    • 是否有增量
    • 增量式学习
    • 批量式学习
    • 学习方式
    • 基于例子学习（instance-based learning）
    • 基于模型学习（model-based learning）

    例子

    • 监督学习，典型场景：分类、回归。任务驱动。有标签
    • 无监督学习，典型场景：聚类。数据驱动。无标签
    • 强化学习，典型场景：策略推理，更接近人的学习过程。无标签，但有反馈
    • 迁移学习...

    监督学习代表算法

    • K邻近法（K-nearest Neighbor）
    • 线性回归（Linear Regression）
    • 逻辑回归（Logistic Regression）
    • 支持向量机（Support Vector Machines，SVM）
    • 决策树和随机森林（Decision Tree and Random Forest）
