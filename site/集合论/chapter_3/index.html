    关系

    有序对

    有序对：\(<x,y>\)。性质（有序对这个定义完全由性质决定）：

    • \(x\neq y\)时，\(<x,y>\neq<y,x>\)
    • \(<x,y>=<u,v> \Leftrightarrow x = u, y = v\)

    集合表示：

    • \(<x,y> = \{ \{ \{x\},\phi \}, \{\{ y \}\} \}\)
    • \(<x,y> = \{ \{x \}, \{x,y\} \}\)(最常用)

    笛卡尔积

    \(A\times B = \{<a,b>|\forall a \in A, b \in B\}\)

    \(A \times B\)称为积集。\(|A\times B|=n\)

    三元组：\(<x,y,z> = <<x,y>,z>\)。一般\(<<x,y>,z> \neq <x,<y,z>>\)。更多元的有序数对采用归纳定义

    性质：

    • \(\phi \times B = A \times \phi = \phi\)
    • \(A\neq B\)时，一般\(A\times B \neq B \times A\)
    • 笛卡尔积不满足结合律
    • 对交并分配率：\(A\times (B\cup C) = (A \times B) \cup (A \times C)\)，对交同理，右分配律同理
    • \(A\neq \phi\)时，\(A \times B = A \times C \Leftrightarrow B = C\)

    \(\forall x \in B,\exist y \in A,<y,x> \in A \times B = A \times C \Rightarrow <y,x> \in A\times C\Rightarrow x \in C \Rightarrow B \subset C。同理：C\subset B，即B=C\)

    关系

    定义

    对于集合\(R\)，\(R=\phi\)或\(R\)元素都为二元有序对，则称R为二元关系

    若\(<x,y>\in R\)，则称\(<x,y>\)满足关系R，写作\(xRy\)。不满足写作\(x\not{R} y\)

    集合A到B的关系，定义为\(R\subset A\times B\)。空集称为A到B的空关系。\(A\times B\)称为全域关系。若\(R \subset A\times A(或记作A^2)\)，则R称为A上的关系。恒等关系\(I_A=\{<x,x>|\forall x \in A\}\)

    运算

    • 定义域：\(dom R = \{x | \exist <x,y> \in R\}\)
    • 值域：\(ran R = \{y | \exist <x,y> \in R\}\)
    • 域：\(fld R = dom R \cup ran R\)
    • 逆：\(R^{-1} = \{<y,x>|\exist <x,y> \in R\}\)
    • 复合：\(R_1 \circ R_2 = \{<x,z>|\exist <x,y> \in R_1, <y,z> \in R_2\}\)。一般不符合交换律
    • 限制\(R \restriction A = \{<x,y>|<x,y>\in R, x\in A\}。R \restriction A = R \restriction (R \cap dom R)\)
    • A在R上的象：\(R[A] = ran(R \restriction A) = \{y|<x,y>\in R,x\in A\}\)

    性质：

    • \(R[A] \subset ran R\)
    • \(dom R^{-1} = ran R, ran R^{-1} = dom R\)
    • \((R_1 \circ R_2) ^ {-1} = R_2^{-1} \circ R_1^{-1}\)
    • \(R \circ I_A = I_A \circ R = R\)
    • \((R_1 \circ R_2)\circ R_3 = R_1 \circ (R_2 \circ R_3)\)
    • \(R[A \cup B] = R[A] \cup R[B],R[A \cap B] \subset R[A] \cap R[B]\)。\(R[A\cap B] = R[A] \cap R[B]当且仅当x,y \in dom R,x\neq y,R[\{x\}] \cap R[\{y\}] = \phi\)，即不能有\(<x,z>,<y,z>\in R\)，即要求关系是单根的
    • \(R[B^c] = R[B]^c\)
    • 在\(R[\{x\}] \cap R[\{y\}] = \phi\)时，R对交并减补同态

    \(证明：\forall y \in R[A \cup B],\exist <x,y> \in R,x \in A \cup B\Rightarrow \exist <x,y> \in R 且(x\in A 或 x\in B)\Rightarrow (\exist <x,y> \in R 且 x \in A) 或 (\exist <x,y> \in R 且 y \in B)\)

    特殊关系

    单值性，单根性

    • \(\forall x \in dom R, \exist 唯一 y \in ran R, <x,y>\in R, 则称R为单值的\)
    • \(\forall y \in ran R, \exist 唯一 x \in dom R, <x,y>\in R, 则称R为单根的\)

    结论：

    • 若\(R\)是单值的，则\(R^{-1}\)是单根的，反过来也成立
    • 关系的复合运算保持单值性，单根性

    映射关系

    单值的关系，记为映射。映射常记为f。\(f(\{x\}) = \{y\}\)，可简写为\(f(x) = y\)。判断映射是否相等：\(f \subset g, g \subset f\)（判断集合是否相等）

    若\(domf = A,ranf \in B\)，则称f为A到B的映射，B称为陪域（历史原因）

    特殊映射：

    • 若f为单根的，则为单射（只与关系本体有关）
    • 若ranf = B，则f为满射（与关系本体无关）
    • 既单又满则为双射。f为单射，则一定为\(dom f\)到\(ran f\)的双射

    性质：

    • \(f: A \rightarrow B, A_1 \subset A, B_1 \subset B, 则f^{-1}[f[A_1]]\supset A_1（非单造成，若相等，可等价于是单射）, f[f^{-1}[B_1]]\subset B_1（非满造成，若相等，可等价于是满射）\)
    • 映射的复合仍为映射。\(证明<x,y_1>,<x,y_2>\in f\circ g \Leftrightarrow y_1 = y_2：即存在<x,t>\in f,<t,y_1> \in g,且<x,u>\in f,<u,y_2>\in g，由映射得，t=u，即y_1=y_2\)。推论：
    • \(dom(f\circ g) = \{x \in domf | f(x) \in dom g\}\)
    • \((f \circ g)(x) = g(f(x))\)
    • \(f:A\rightarrow B, g:B\rightarrow C, 则 f\circ g:A\rightarrow C，且dom(f\circ g) = dom f\)
    • 若f为A到B的双射，则\(f^{-1}\)为B到A的双射。\(f \circ f^{-1} = I_A, f \circ f^{-1} = I_B\)。若A=B，则\(f \circ f^{-1} = f^{-1} \circ f= I\)。逆命题成立：\(若对f:A\rightarrow B，f为双射，当且仅当\exist g:B\rightarrow A, f \circ g = I_A, g \circ f = I_B\)
    • \(f:A\rightarrow B\)为单射 \(\Leftrightarrow\) \(\exist g : B \rightarrow A,f \circ g = I_A\)。此时\(g\)唯一，为\(f^{-1}\)。
    • \(f:A\rightarrow B\)为满射 \(\Leftrightarrow\) \(\exist g : B \rightarrow A,g \circ f = I_B\)。此时\(g\)唯一，为\(f^{-1}\)。
    • 若\(f:A\rightarrow B, g:B \rightarrow C, f\circ g = I_A，则f为单射，g为满射\)
    • 若\(f:A\rightarrow B,g:B\rightarrow C, f \circ g :A \rightarrow C为单射，则f为单射\)
    • 若\(f:A\rightarrow B,g:B\rightarrow C, f \circ g :A \rightarrow C为满射，则g为满射\)
    • \(f\)单值，\(f^{-1}\)单根，则\(f^{-1}\)对集合的交并减同态

    等价关系

    特殊关系的性质

    给定集合A，考虑A上的关系R，关系的性质：

    • \(\forall x \in A\)，若\(<x,x>\in R\)，则有自反性。例如恒等关系，全域关系
    • \(\forall x \in A\)，若\(<x,x>\notin R\)，则有反自反性
    • \(\forall x,y \in A\)，若\(<x,y>\in R,\)可得到\(<y,x> \in R\)，则称R是对称的
    • \(\forall x,y \in A\)，若\(<x,y>\in R,<y,x>\in R\)，等价于x=y，则称R是反对称的（存在关系既不是对称的也不是反对称的，也有既是对称的，也是反对称的）
    • \(\forall x,y,z \in A\)，若\(<x,y>\in R,<y,z>\in R\)，则\(<x,z>\in R\)，则有传递性

    命题

    • \(R\)是自反的\(\Leftrightarrow\) \(I_A \in R\)
    • \(R\)是反自反的\(\Leftrightarrow\) \(R \cap I_A = \phi\)
    • \(R\)为对称的\(\Leftrightarrow\) \(R = R^{-1}\)
    • \(R\)为反对称的\(\Leftrightarrow\) \(R \cap R^{-1} \subset I_A\)
    • \(R\)是传递的\(\Leftrightarrow\) \(R \circ R \subset R\)

    定义

    \(A\neq \phi\)，若A上的关系\(\sim\)自反，对称，传递，则称R为等价关系，可将\(<x,y>\in\sim\)写作\(x\sim y\)

    等价类

    \([a]\)表示以a为代表元的等价类，\([a] = \{x\in A|x \sim a\}\)

    • \(\forall x,y \in A,x \sim y \Rightarrow [x] = [y]\)
    • \(\forall x \in A,[x] \neq \phi\)
    • \(\forall x,y \in A,x \not \sim y \Rightarrow [x] \cap [y] = \phi\)。等价类无交
    • \(A = \bigcup_{a\in A}[a]\)。所有等价类的并为全集

    划分：\(\pi \subset P(A)\)，且

    • \(\phi \notin \pi\)
    • \(B\neq C \in \pi \Rightarrow C \cap B = \phi\)
    • \(A = \bigcup_{B\in \pi}B\)

    等价类是一种划分。每给一种等价关系，都可以形成集合的一种划分

    商集：\(\frac A \sim = \{[a] | x\in A\}\)

    每一个划分\(\pi\)会诱导出一个等价关系\(\sim_\pi\)，且商集为该划分，即\(\frac A {\sim_\pi} = \pi\)。该等价关系定义为同属于一个划分块。等价关系和划分是一一对应的。且有\(\pi_{\sim_\pi} = \pi, \sim_{\pi_\sim} = \sim\)

    自然映射

    将映射变为单射：\(x\sim y\)定义为\(f(x) = f(y)\)，则商集到值域为单射

    自然映射：给定\(A、\sim\)，有自然映射\(n:A\mapsto A/\sim,a \mapsto [a]\)。自然映射为满射

    则\(x \mapsto f(x)\)可分解为\(x \mapsto [x] \mapsto f(x)\)，即\(f =\widetilde{f} \circ n\)，其中\(\widetilde f\)唯一，为双射，\(n\)为自然映射，为满射

    R为A上的一个关系，\(S=I_A \cup (R \cup R^{-1})\cup (R \cup R^{-1})^2 \cup \dots \cup (R \cup R^{-1})^n\cup \dots\)则S为A上的等价关系，且为包含R的最小等价关系

    证明：
    自反：\(<x,x>\in I_A \subset S\)
    对称：若\(<x,y>\in I_A \subset S\)，则\(x=y\),\(<y,x>\in S\) 若\(<x,y> \in (R\cup R^{-1}),即\exist <x,t_1>,<t_1,t_2>\dots <t_{n-1},y>\in R \cup R^{-1}。因为<t_i,t_j>\in R \cup R^{-1}，可推出<t_j,t_i>\in R \cup R^{-1}\)，则...
    传递：平凡...，非平凡时，设\(<x,y> \in (R \cup R^{-1})^m,<y,z> \in (R \cup R^{-1})^n,则<x,z> \in (R \cup R^{-1})^{m+n}\)

    假设T为A上的等价关系，且\(T\supset R\)，\(\forall <x,y> \in S\)，则\(<x,y> \in T\)
    平凡：\(<x,y> \in I_A \subset T\) 非平凡：\(<x,y> \in (R \cup R^{-1})^n\)，即存在\(<x,t_1>,<t_1,t_2>\dots <t_{n-1},y> \in R \cup R^{-1}\)，即证\(T\subset R^{-1}\)，任取\(<x,y> \in R^{-1}，则<y,x> \in R，有<y,x>\in T，则<x,y> \in T\)，即\(R^{-1} \subset T\)

    偏序关系

    定义

    R满足

    • 自反性
    • 反对称性
    • 传递性

    则称R为偏序关系，用\(\preceq\)表示。例如幂集P(A)上的包含关系为偏序关系

    • 严格偏序关系：对A上关系R，若满足反自反，传递，则R为严格偏序关系。由\(x \prec y\)可得\(y \not \prec x\)，即满足反对称。偏序和严格偏序一一对应\(\preceq - I_A = \prec\)，\(\prec \cup I_A = \preceq\)
    • 全序关系（线序关系）：集合中任意两个元素都有偏序关系（即可比），即存在三岐性。
    • 定义一：
      • 传递性
      • 三种关系有且仅有一个发生
    • 定义二（前两个条件为严格偏序的定义）：
      • 传递性
      • \(a \not \prec a\)
      • 三个关系至少满足一个（由前两个条件可加强为有且仅有一个）
    • 良序：任何一个子集必有最小元。决定第二归纳法是否成立

    三岐性：三种情况当且只有一种情况发生

    哈斯图：偏序集\(<A,\prec>\)，若\(x\prec y\)，且不存在\(z\in A,x< z < y\)，则把x画在y下。全序集的哈斯图为一条竖线，可为直线可为射线，射线时为良序

    最元和极元

    \(<A,\preceq>\)，若\(B\subset A,m\in B,\forall x \in A, x \preceq m\)，则称m为最大元。\(\forall x \in A, 如果 m \preceq x, 则x=m\)，则m为B的极大元。

    最大元为全局性的，极大元为局域的。最大元一定为极大元。

    全序关系中，极大元一定是最大元 。

    确界

    \(<A,\preceq>\)中，\(B\subset A, m \in B\)

    • \(\forall x \in B,x \leq m\)，则m为B的一个上界
    • 上界的集合，如果有一个最小元，则这个最小元为上确界，记为\(SupB\)

    若B有最大元，则上确界为最大元。\(若B有上界m\in B\)，则m为B的最大元

    有上界不一定有上确界，例如在Q中的集合\((0,\sqrt 2) \cap Q\)
