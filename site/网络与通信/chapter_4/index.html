    信息传输技术基础

    信息传输和信道

    信道

    信息传输就是将携带信息的信号通过媒体传送到目的地的过程

    flowchart LR
A[信息源] --> B[发送设备] --> C[信道] --> D[接收设备] --> E[受信者]

    前两者为发送端，后两者为接收端。信道中会加入噪声源

    是将信源端送出的信号变换或匹配成适合于特定信道介质传输信号的设备

    接收端的接收设备与发送设备实现功能相反，是将从信道接收的信号相应地恢复、还原成终端设备能识别信号的设备

    信道分类：

    • 根据传输信号特性
    • 模拟信道
    • 数字信道
    • 根据传输距离远近、作用
    • 用户线或接入信道
    • 中继或长途传输信道
    • 根据传输介质是否有形
    • 无线信道：利用电磁波（含光波）在空间中的传播来传输信号
    • 有线信道：利用人造传到电或光信号的媒体传输信号（如电线、光纤）

    信息容量

    在一定时间内，通过一个系统可以传输多少信息的度量。可以是系统容纳用户数目或系统交换、传输及处理的信息比特数。香农信息容量极限表述为：

    \[C = B \cdot log_2 (1 +\frac S N)\]

    论述了信道的信息容量C（单位b/s）与带宽B和信噪比S/N（信号的功率除噪声的功率）的关系。设噪声单边功率谱密度为\(n_0\)，则\(N = n_0 B\)，上式可改写为

    \[C = B log_2(1 + \frac S {n_0 B})\]

    由此，C与带宽B、信号功率S、噪声功率谱密度\(n_0\)三个因素有关。C是有限的，因为B增大同时，噪声功率N也增大，S/N减小，C是有限的。若带宽B无限增大，保持\(n_0, S\)，则有极限值

    \[\lim_{B\to \infty} C = \frac S {n_0} log_2 e \approx 1.44 \frac S {n_0}\]

    关于分贝：

    \[(S/N)_{dB} = 10 log_{10}(S/N)\]

    用分贝表示增益、损耗及相对值，是两个信号电平间比值的一种度量。分贝增益计算：

    \[G_{dB} = 10log_{10}(\frac {P_{out}} {P_{in}})\]

    其中\(P_{in}\)为输入功率，\(P_{out}\)为输出功率。分贝损耗：

    \[L_{dB} = - 10 log_{10}\frac {P_{out}} {P_{in}} = 10 log_{10}\frac {P_{in}} {P_{out}}\]

    信道传输特性

    有线信道传输特性：

    • 幅频传输特性：信道在各频率下衰耗与频率的关系曲线，他将影响信号幅度衰减量。理性情况下要求其带通内特性平稳，否则导致信号幅度失真（在一定区域内为平的直线）
    • 相频传输特性：信道在各频率下的相位移与频率的关系曲线，他将影响被传输信号的相位移（应该在频率范围内为线性）

    无线信道传输特性：频段划分...（看ppt）

    • 长波：频率为100~300KHz。绕射能力最强，靠地波传输。常用电台进行海上通信。
    • 中波：频率为300KHz~3MHz。较稳定。主要用于短距离广播。
    • 短波：频率为3~30MHz。利用空间电离层反射，进行长距离传输，主要用于短波通信和短波广播。但是受电离层影响较大
    • 超短波：频率为30~300MHz。主要用于调频广播、电视。
    • 微波：频率为300MHz~300GHz。波长较短，接近于光波，是直线传播。实现点到点间无遮挡的视距传播。主要用于移动通信、卫星通信

    信道的衰减与失真

    在信道中传递的信号，由于介质的特性，信号传输中必然会产生能量的损失，这种能量的损失，我们称之为衰减或衰耗。在通信工程中称之为固有衰减，这种衰减随信道种类不同而不同，传输信道距离越长衰减越大。

    常见信号衰减与失真度量参数有：幅度衰减、幅度突变、相位抖动、群时延—频率失真、频率偏移等

    • 幅度衰减是指信道对不同频率信号的幅度衰减变化。用于信道的频带是有限的，不同频率的信号通过信道时的衰减值往往是不一样的。为了减小幅度衰减，在设计信道的传输特性时，一般要求把幅度-频率失真控制在允许范围内，使衰减的特性曲线变得平坦，这种措施称为“均衡”。
    • 幅度突变是指接收信号幅度突然变化（增加或减小）的数值，一般要求门限值在1~6dB范围内选择
    • 相位抖动是指接收信号的时间位置和码元标准参考时间位置的短时偏离。在模拟通信中影响不明显；在数字通信中会引起码间干扰，严重时会造成误码。

    信道的损伤

    信道中的噪声与干扰。分类：

    • 系统内部噪声：少数载流子的随机扩散与电子-空穴对的随机复合运动产生散弹噪声。热运动造成的白噪声。
    • 系统外部干扰：通信设备工作时，处于强电磁环境中。例如受雷电、太阳黑子活动等引起的电磁抱，或受其他无线电设备发射电磁波的影响。可降低外界干扰源的干扰、增强通讯设备屏蔽能力来改善。

    加性噪声：叠加在信号上面传输的噪声，它与信道中传输的信号存在着相加的关系，与信道内信号有无没有关系。

    传输技术不仅是为了传输信号，更需要提高信息传输的有效性和可靠性。

    数字传输系统性能指标

    主要分为有效性、可靠性指标

    有效性指标常用信息传输速率、码元传输速率（符号速率）、频带利用率等表示

    可靠性指标常用误码率、抖动容限来表示

    信息传输速率指单位时间内传送的信息量，也称为比特率，单位为bit/s，符号\(f_b\)

    码元（符号）传输速率指单位时间所传输的码元数目，也称为符号传输速率，或码元速率、波特率。单位为波特（Bd，或Baud）。可折合为信息速率进行计算。转换公式

    \[f_b = f_B log_2 M\]

    其中\(f_b\)为信息传输速率（二进制传输速率），\(f_B\)表示码元传输速率，单位为波特，M为码元进制数（符号进制数）。对于二进制数字信号，M=2，码元速率和信息速率在数量上相等

    频带利用率指单位频带内的传输速率。传输的速率越高，所占用频带越宽。常用频带利用率表示数字信道频带利用情况：

    \[\eta = \frac {\text{传输速率}} {\text{频带宽度}}\]

    当传输速率是码元传输速率时，单位为Baud/Hz，为信息传输速率时，单位为B/s/Hz

    误码率：在数字通信中用脉冲信号（即0，1）携带信息，由于噪声、串音、码间干扰及其他突发因素影响，当干扰幅度超过脉冲信号再生判决的某一门限值时，会造成误判称为误码。误码率为

    \[P_e = \lim_{n\to \infty} \frac {\text{产生错误码元个数}}{\text{传输的总码元个数}}\]

    误码率越小，通信质量越高

    抖动容限。抖动指数字信号有效瞬间与其理想时间位置的短时偏移。即脉冲信号在时间间隔内不再是等间隔的，而是随时间变化的一种现象

    例：已知黑白电视图像信号每帧有30万个像素；每个像素有8个亮度电平；各电平独立地以等概率出现；图像每秒发送25帧。若要求接收图像信噪比达到30dB，试求所需传输带宽。

    因为每个像素独立地以等概率取8个亮度电平，故每个像素的信息量为\(I_p = -log(1/8) = 3\ (b/pix)\)

    每帧图像信息量为\(I_F = 300000 \times 3 = 900000 \ (b/F)\)

    因为每秒传输25帧图像，所以传输速率为\(f_b = 900000\times 25 = 22.5\times 10^6\ (b/s)\)

    信道容量不小于此\(f_b\)，带入（注：30dB要进行换算）：

    \(22.5 \times 10^6 = Blog_2(1 + 1000)\)

    得\(B\approx 2.26\ (MHz)\)

    有线传输信道

    电磁能量被约束在某种传输线上传输

    • 架空明线
    • 平行双线电缆
    • 对称电缆
    • 双绞线
    • UTP无屏蔽双绞线
    • FTP屏蔽双绞线
    • SFTP屏蔽金属箔双绞线
    • STP屏蔽双绞线
    • 同轴电缆

    无线传输信道

    • 地波：沿地球表面传播，很容易穿过一般建筑物。大气密度存在梯度，造成波阵面倾斜。
    • 优点：发射功率足够大就可用于世界上任何两地通信。大气条件改变对地波基本上无影响。
    • 缺点：需要很大发射功率，频率限制在低功频，需要大尺寸天线。地面损耗随表面材质不同会发生明显变化
    • 空间波
    • 直射波：发射天线与接收天线间直线传播
    • 反射波：分天波、地波。天波在某一方向上相对地球仰起一个很大的角度来辐射电磁波，通过电离层传播

    模拟传输技术基础

    完成模拟信号传输的系统分为

    • 基带模拟传输系统
    • 高频窄带模拟传输系统

    基带传输系统

    自然界任何非电量信息m(t)，经过非电/电量变换后的通信信号f(t)，其频率分布在接近0Hz频率到某一有限频段范围内

    若系统直接传输f(t)信号，不再进行其他变换，这种传输系统就成为基带模拟传输系统

    高频窄带传输系统

    对信源发出的电信号进行调制的变换，将频率移到某高频率载波附近，使f(t)称为已调信号S(t)的传输系统

    概念：

    • 调制信号：指来自信源的基带信号
    • 载波：未收调制的周期性振荡信号
    • 已调信号：载波受调制后成为已调信号（在信道中传播这个）
    • 解调（检波）：调制的逆过程，将已调信号中的调制信号恢复出来
    • 载波调制：用调制信号去控制载波的参数的过程

    调制目的：便于无线发射，减小天线尺寸，把多个基带信号分别搬到不同的载频处，以实现信道的多路复用，提高信道利用率。扩展信号带宽，提高抗干扰、抗衰落能力，还可实现传输带宽与信噪比之间的互换

    常见的模拟调制：

    • 幅度调制调幅、双边带、单边带、残留边带
    • 角度调制：频率调制、相位调制

    调制/解调技术：将电信号、本地振荡器产生的载波信号一起输入乘法器，得到S(t)，在信道中传播，再接收端通过乘法器，输入S(t)和本地振荡器产生的同频同相的信号，得到一个高频，一个低频成分，再通过低通滤波器得到原始信号f(t)

    载波调制：用调制信号去控制载波的参数的过程

    • 调幅：用电信号f(t)去调制载波信号C(t)的振幅\(A_0\)的调制技术
    • 调频：调制载波信号的频率\(f_0\)的调制技术
    • 调相：调制载波信号的相位\(\phi_0\)的调制技术

    常见：

    • AM：优点是接收设备简单；缺点是功率利用率低，抗干扰能力差。主要用在中波和短波调幅广播。
    • DSB调制：优点是功率利用率高，且带宽与AM相同，但设备较复杂。应用较少，一般用于点对点专用通信。
    • SSB调制：优点是功率利用率和频带利用率都较高，抗干扰能力和抗选择性衰落能力均优于AM，而带宽只有AM的一半；缺点是发送和接收设备都复杂。SSB常用于频分多路复用系统中。
    • VSB调制：抗噪声性能和频带利用率与SSB相当。在电视广播等系统中得到了广泛应用。
    • FM：FM的抗干扰能力强，广泛应用于长距离高质量的通信系统中。缺点是频带利用率低

    调幅

    调幅：

    音频信号\(f(t) = K + m(t)\)，K为外加的直流分量

    \[S(t) = C(t)f(t) = A_0 f(t) cos(\omega_c t + \theta_0)\]

    做傅里叶变换

    \[S_{AM}(\omega) = \pi A_1 [\delta(\omega + \omega_c) + \delta(\omega - \omega_c)] + \pi A_0 [M(\omega+\omega_c) + M(\omega-\omega_c)]\]

    其中\(M(\omega)\)为m(t)对应频谱

    （看ppt）

    AM信号特点：结构简单，容易实现，适合广播通信

    缺点：

    • 效率低，最大为1/3
    • 频谱效率也不高，为信号频率的两倍

    抑制载波双边带调幅（DSB）

    调幅信号中，载波分量不携带信息，信息完全由边带传送。如果将载波抑制掉，只需将f(t)信号中的直流分量K抑制掉，即可输出抑制载波的双边带调幅信号

    • 调制效率100%，节省了载波功率
    • 带宽为原始信号两倍，与AM相同

    缺点：

    • 不能用包络检波，需要相干检波，较复杂

    SSB信号的频谱

    \[S_{SSB} (\omega) = S_{DSB} H(\omega)\]
    • \(H(\omega)\)为高通滤波器，为上边带调幅（若经过低通滤波器，则为下边带调幅）

    产生方法：滤波法、相移法

    滤波法难点：

    • 边带滤波器制作
    • 滤波特性很难做到具有陡峭的截止特性

    残留边带调制（VSB）

    介于SSB和DSB之间的折中方法。克服DSB占频带宽的缺点，又解决了SSB信号实现中的困难。不像SSB完全抑制DSB信号的一个边带，而是逐渐切割，使其保留一小部分

    \[S_{VSB}(\omega) = S_{DSB}(\omega) H(\omega) = \frac 1 2 [M(\omega + \omega_c) + M(\omega - \omega_c)]\]

    调频FM

    是非线性调制。因为是频谱的非线性变换。非线性最突出的优势是其较高的抗噪声性能

    调相PM

    是非线性调制

    数字传输技术基础

    数字通信抗噪声干扰能力强，无噪声基类。便于集成、加密、处理

    分为数字基带传输、数字频带传输

    数字基带信号

    单极性不归零码

    单极性不归零码（NRZ）：设消息由二进制符号0、1组成，基带信号的0电位及正电位分别对应数字信号0、1

    含直流分量（产生冲击函数，导致基线漂移）、丰富低频分量，因此不适合做基带传输码型。只适用于计算机内部、极近距离的传输

    双极性不归零码

    1用+V，0用-V表示。当1、0等概率出现时，无直流分量，并且在接收端恢复信号的判决电平为0值，因此不受信道特性变化，抗干扰能力强

    单极性归零码

    单极性归零码（RZ）：0、1表示方式和NRZ一样，但脉冲宽度是NRZ的一半，后一半回到0。为半占空码，占空比为50%。功率特性：有很多冲激函数，可以提取定时信号，不需要额外插入同步码

    双极性归零码

    双极性归零码：相邻脉冲之间必有零电位间隔，使接收端很容易识别出每一个码元的起止时刻，便于同步

    选择的数字基带码要满足以下条件：

    • 不含直流，低频分量尽量少
    • 应含有丰富定时信息，以便提取定时信号
    • 码型应具有一定的检错能力
    • 设备简单，易于实现

    AMI码

    双极性半占空码(AMI)码（极性半占空交替反转码）：消息码的“1”(传号)交替地变换为“+1”和“-1”，而“0”(空号)保持不变

    AMI码的优点：没有直流成分，且低频分量少，编译码电路简单，且可利用传号极性交替这一规律观察误码情况；如果它是AMI-RZ波形，接收后只要全波整流，就可变为单极性RZ波形，从中可以提取位定时分量。

    AMI码的缺点：当原信码出现长连“0”串时，信号的电平长时间不跳变，造成提取定时信号的困难。

    HDB3码

    解决连“0”码问题的有效方法之一是采用HDB3码。数码流中连续出现4个以上连0时，从第一个0到四个连0中，最后一个“0”用“V”码取代，此位码称极性破坏点。各“V”码必须进行极性交替(为保证传号码极性交替不引入直流成分)。相邻“V”码间，前一“V”码与后相邻的原传号“1”码，应使之符合极性交替原则(符合AMI码变换规律)。要使“V”码前面相邻一定出现一个与之极性相同的码位(满足“V”码为极性破坏点要求)。按前三步骤变换后也可能会出现与“V”码同极性的码，当没有出现时，就将四连“0”中的第一个“0”用B码取代，使B与它后邻的取代V码同极性(为了在收端能识别出哪个是取代码以便消除)。

    CMI码

    传号反转码。规则为1码交替用11和00表示，0码用01表示，10为禁字，可用于判决误码

    Manchester码

    双相码

    对每个二进制码分别用两个具有不同相位的二进制新码取代。0用10表示，1用01表示

    无直流分量，是一种双极性NRZ波形，每个码元间隔的中心点都存在跳变，有丰富的位定时信息。但占用带宽加倍，频带利用率低

    密勒码

    延迟调制码。1用码元中心点出现跃变来表示，即用01或10表示。0有两种情况：

    • 单个0，在码元持续时间内不出现电平跳变，且与相邻码元边界处也不跃变
    • 连0时，在两个0码的边界出现电平跃变，即00与11交替

    用双相码的下降沿去触发双稳电路，即可输出密勒码

    mBnB码

    为提高线路编码性能，需要某种冗余来确保码型的同步、检错能力

    mBnB码：把原信息码流的m位二进制码分为一组，并置换为n位二进制码的新码组。因为n>m，新码组可能会有\(2^n\)种组合，故多出\(2^n - 2^m\)种组合。在\(2^n\)种组合中，以某种方式选择有利码组作为可用码组，其余作为禁用码组

    例如4B5B，用5位编码代替4位编码，对于四位分组，只有\(2^4 = 16\)种不同组合，对于5位分组，有\(w^5 = 32\)种不同组合

    双相码、密勒码、CMI码都可看做1B2B码

    优点：提供了良好的同步、检错功能

    缺点：带宽增大

    4B5B常用在光纤通信中

    误码串扰

    误码原因：

    • 码间串扰：系统传输总特性不理想，导致前后码元的波形畸变并使前面波形出现很长的拖尾，从而对当前码元的判决造成干扰
    • 信道加性噪声

    数字基带传输系统

    特性

    基带信号呈现低通型频谱特性，基带传输系统具有低通特性

    数字基带信号无失真传输条件（奈奎斯特第一准则）

    如果信道具有理想低通滤波器频幅特性，理想低通的截止频率为\(f_B / 2\)，\(f_B\)为码元速率，则在判决点无码间干扰

    带宽为B的理想基带传输系统无码间干扰速率为2B Baud

    奈奎斯特速率\(f_B = 2B\ Baud\)，为最大无码间干扰速率

    奈奎斯特频带利用率

    \[\eta = \frac {\text{传输速率}} {\text{频带宽度}} = \frac {f_B} B = 2\]

    为数字基带系统的极限频带利用率

    设信道带宽为32kHz，若码元采用8进制，求最高信息传输速率

    \(f_{Bmax} = B \eta_{max} = 64 k\text{码元}/s\)

    由于采用8进制，每个码元信息量为\(-log_2(1/8)=3bit/\text{码元}\)

    最高信息速率为\(f_b = 3f_B = 192kbit/s\)

    数字频带传输

    特性

    所谓频带传输，是指原始电信号在发送端先经过调制后，再送到线路上传输，接收端则要进行相应解调才能恢复出原来的基带信号。

    调制是通过改变一个更高频率信号的某些特征物理量或参数(如幅度、频率、相位等)的过程，这一高频信号常称载波，它一般由载波振荡器(如振荡电路、激光器等)产生。

    flowchart LR
A[基带信号] --> B[调制器<br>（上变频器）]--信道-->C[解调器<br>（下变频器）] --> 基带信号

D[载波振荡器<br>（高频信号）]-->B
E[本地载波震荡器]-->C

    前三者称为发射机，后三者称为接收机

    调制

    调制信号是二进制基带信号。二进制数字调制中，载波的幅度、频率或相位只有两种变化

    • 二进制幅移键控2ASK：基带与载波通过相乘器输出。1就正常的波，0为平的。

    解调

    非相干解调（包络检波法）

    flowchart LR
A(2ASK信号) --> B[带通滤波器] --> C[半波或全波整流器] --> D[低通滤波器] --> E[抽样判决器] --> G(输出)
F(定时脉冲)-->E

    相干解调：

    flowchart LR
A((2FSK信号)) --> B[带通滤波器] --> C[相乘器] --> D[低通滤波器] --> G[抽样判决器] --> E((输出))
F((载波))-->C
H((定时脉冲))-->G
    • 二进制频移键控2FSK。\(S_{2FSK}(t) = \begin{cases}f(t) cos(\omega_1 t + \phi_n), &\text{发送1时} \\ f(t) cos(\omega_2 t + \theta_n), & \text{发送0时}\end{cases}\)。一个2FSK信号可看成两个不同载频的ASK信号叠加

    解调：

    相干解调（最佳接收法）

    非相干解调（鉴频法）

    • 二进制相移键控2PSK：通常用初相位0、1表示二进制1和0.时域表达式为\(S_{2PSK} (t) = f(t) cos(\omega_c t + \phi_n)\)，其中\(\phi_n = \begin{cases}0, &\text{发送1时} \\ 1, &\text{发送0时}\end{cases}\)

    解调

    flowchart LR
A(("$$S_{2PSK}(t)$$")) --> B[带通滤波器] --> C[相乘器] --> D[低通滤波器] --> G[抽样判决器] --> E((输出))
F(("$$cos\omega_c t$$")) --> C
H((定时脉冲))-->G
    • 二进制相对相移键控2DPSK。调制规律：遇到基带信号1时，载波的相位相对于前一个码元相位改变\(\pi\)，遇到0时不变（也可以反过来）

    产生：

    先对二进制数字基带信号进行差分编码，即把表示数字信息序列的绝对码变换成相对码（差分码），然后再根据相对码进行绝对调相，从而产生二进制差分相移键控信号

    基带信号经过码变换电路变为相对码，在通过绝对调相电路变为2DPSK已调波

A(("$$S_{2DPSK}$$")) -->B[带通滤波器] --> C[相乘器] --> D[低通滤波器] --> G[抽样判决器] --> E[码反变换器]-->F((输出))
H(("$$cos\omega_c t$$"))--> C
I((定时脉冲))-->G

    \(M_{ary}\) Modulation多级调制技术

    例如可以用四个幅度表示两位二进制（4ASK）等

    QAM：正交幅度调制。结合ASK和PSK

    \[S(t) = d_1(t) cos2\pi f_c t + d_2(t) sin 2\pi f_c t\]

    变为二维编码，信号稳定性强，码间距离加大。能充分利用带宽。复杂性高，需要同时检测相位、幅度

    到原点长度为幅度，角度为相位。
