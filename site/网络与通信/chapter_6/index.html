    现代通信系统

    概述

    flowchart LR
A[信源] --> B[信源编码器]
--> C[信道编码器]
--> D[数字调试器]
--> E[信道]
--> F[数字解调器]
--> G[信道解码器]
--> H[信道解码器]
--> I[信宿]

J[噪声源]--> E
    • 信源编码：将模拟信号变为数字信号
    • 信道编码：将数字信号进行编码，变换为所需的传输码型
    • 从信道编码器到其对应解码器称为数字传输系统

    现代通信系统分类：

    • 数字光纤通信系统
    • 数字微波与卫星通信系统
    • 有限通信系统与无线通信系统相结合的数字移动通信系统

    数字光纤通信系统

    结构及特点

    以激光为数字信息载体。现在为以光纤为主，其他为辅的格局

    光纤结构：

    • 中间是纤芯，折射率比包层高
    • 外层是包层
    • 最外侧是保护套

    多模光纤：可存在多条不同角度入射的光纤在同一条光纤中传输。光脉冲会展宽，导致失真

    单模光纤：其直径减小到只有一个光的波长（几个微米），可使光线一直向前传播，而不会产生多次反射。光源要使用昂贵的半导体激光器，不能使用较便宜的发光二极管。不易失真

    常用波段中心：

    • 850nm
    • 1300nm
    • 1550nm

    都具有25000~30000GHz的带宽，通信容量大

    光缆：将光纤做成结实的光缆。要将多根光纤、加强芯、填充物（可放入远供电源线），最后在包上外层、保护套

    特点：

    • 传输的频带宽、通信容量大。
    • 损耗极低，传输距离长。
    • 线径细, 重量轻。
    • 不受电磁干扰、 防腐和不会锈蚀。避免电磁干扰
    • 不怕高温，防爆、防火性能强
    • 光纤通信保密性好。数根光纤之间不会相互干扰，即不会串话

    缺点：

    光纤质地脆、机械强度低；要求比较好的切断、连接技术；分路、耦合比较麻烦等

    数字光纤通信系统组成

    • 发送设备
    • 输信道
    • 接收设备

    实用数字光纤通信系统都采用数字编码信号经强度调制-直接检波方式。

    • 强度调制是利用数字电脉冲信号直接调制光源的光强度或光功率，使光强度与数字电脉冲信号电流成线性变化。
    • 直接检波是把光脉冲信号通过光接收机还原成数字电脉冲信号。
    flowchart LR
A[数字光发射机] --光纤--> B["光中继器(光接收机O/E、光发射机E/O)"] --光纤--> C[数字光接收端机]
    • 光发射机（包括激光器LD或发光二极管LED）的作用是实现电/光转换，即把电信号调制成光信号，送入光纤传输至远方。
    • 光接收机（包括光电检测器PIN）的作用是实现光/电转换，即把来自光纤的光信号还原成电信号，经放大、整形、再生恢复原形后，送至数字接收端机，完成数字信号的分接以及数/模转换。
    • 对于长距离的数字光纤通信系统，还需要光中继器 。其作用是经过长距离光纤衰减和畸变后的微弱光信号放大、整形、再生成具有一定强度的光信号，继续送向前方，以保证良好的通话质量。目前都是光-电-光的形式

    无线通信

    • 长波：频率为100～300KHz。绕射能力最强，靠地波传输。常用电台进行海上通信。
    • 中波：频率为300KHz～3MHz。较稳定。主要用于短距离广播。
    • 短波：频率为3～30MHz。利用空间电离层反射，进行长距离传输，主要用于短波通信和短波广播。
    • 超短波：频率为30～300MHz。主要用于调频广播FM、电视。
    • 微波：频率为300MHz～300GHz。波长较短，接近于光波，是直线传播。实现点到点间无遮挡的视距播。主要用于移动通信、卫星通信。

    基本特征：

    • 带宽有限 ：取决于可使用的频率资源和道的传播特性。
    • 干扰和噪声影响大：由无线通信工作的电磁环境所决定。
    • 在移动通信中存在多径衰落：在移动环境下，接收信号起伏变化。

    无线传播基本特性：

    • 电波的自由空间传播：为横波。自由空间指理想电磁波传播环境，自由空间损耗的实质是因电波扩散损失的能量，特点是接收电平与距离的平方及频率的平方均成反比关系
    • 电波的地面传播：受地平面限制、有障碍物等
    • 视距传播视距传播的实际通信距离受到地球表面曲率的限制，一般无线通信的视距距离要比可视的视距长1/3。因为微波会被大气弯曲或折射，大多数微波弯曲程度与地球曲率相同，因而比直线广播传输的远些
    • 多径传播：由于光的干涉，反射信号与直接信号干涉
    • 移动环境；多普勒效应

    天线

    无线电发射机输出的射频信号功率通过馈线（电缆）输送到天线，由天线以电磁波形式辐射出去。

    把从馈线取得的能量向周围空间辐射出去，将大部分能量向所需的方向辐射。天线根据其方向性可分为全向天线和方向性（或定向）天线

    电磁波到达接收地点后，由天线接收下来（仅接收很小一部分功率），并通过馈线送到无线电接收机。

    在无线通信中，为了充分利用信道，在多点之间实现相互间互不干扰的多边通信，常采用不同的多址技术，如频分多址(FDMA)、时分多址(TDMA)、码分多址(CDMA)、空分多址等(SDMA)。

    同时，还采用了多类先进的调制技术，如具有抗干扰能力强和信号隐蔽等突出特点的扩频技术；具有无线环境高速传输特征的正交频分复用（OFDM）技术等

    在无线通信中，一个信道只容纳一个用户进行通话；许多同时通话的用户，可以共享无线媒体；用某种方式可区分不同的用户，即为多址方式。解决多址接入的方法称为多址接入技术。

    FDMA：将给定频谱资源按频率划分，把传 输频带划分为若干较窄且互不重叠的子频带（或称信道），每个用户分配到一个固定子频带，按频带区分用户；将信号调制到该子频带内，各用户信号同时传送；接收时分别按频带提取，实现多址通信。

    应用：在模拟蜂窝移动通信系统中，采用频分多址方式是唯一的选择；在数字蜂窝移动通信系统中，则很少采用纯频分的方式。目前仍在卫星通信、移动通信、微波通信等系统中应用。

    FDM（频分复用）：将多路基带信号调制到不同频率载波上，再进行叠加形成一个复合信号的多路复用技术

    TDMA（时分多址）：在给定传输频带的条件下，把传递时间分割成周期性的帧，每一帧再分割成若干个时隙（帧或时隙均互不重叠）。各用户在同一频带中传送，用户收发各使用一个指定的时隙，时间上互不重叠。

    TDMA系统的收发双工可采用频分双工（FDD）方式或时分双工（TDD）方式。

    在FDD方式中，上行链路和下行链路的帧结构既可相同也可不同。在TDD方式中，通常收发工作在相同频率上；在一帧中，一半的时隙用于移动站发送，另一半的时隙用于移动站接收。

    TDM（时分复用）：将时间划分为一段段等长时分复用帧（TDM帧）

    • 每一个时分复用的用户在每一个 TDM 帧中占用固定序号的时隙
    • 每一个用户所占用的时隙是周期性地出现（其周期就是TDM帧的长度）的。
    • TDM 信号也称为等时 (isochronous) 信号。
    • 时分复用的所有用户在不同的时间占用同样的频带宽度

    存在的不足：使用时分复用系统传送计算机数据时，由于计算机数据的突发性质，用户对分配到的子信道的利用率一般是不高的。

    STDM（统计时分复用）：是指动态地按需分配共用信道的时隙，只将需要传送数据的终端接入共用信道，以提高信道利用率的多路复用技术

    CDMA（码分多址）：以不同的互相正交的码序列来区分用户并建立多址接入

    CDMA系统中，各用户使用相同的载波频率，占用相同频带，发射时间任意；各用户的频率和时间可相互重叠，用户划分利用不同地址码序列实现。

    CDMA方式是频谱扩展的通信方式，即扩频方式。用户信号经过扩频处理，再经载波调制后发送出去。接收端使用完全相同的扩频码序列，同步后与接收的宽带信号作相关处理，把宽带信号解扩为原始数据信 息。

    不同用户使用不同的码序列，其占用相同频带，可以同时传输，接收机虽然能收到但不能解出，这样可实现互不干扰的多址通信。

    每个站分配的码片序列不仅必须各不相同，并且还必须互相正交。在实用的系统中是使用伪随机码序列。任何一个码片向量和该码片向量自己的规格化内积都是1

    \[S\cdot T = \frac 1 m \sum S_iT_i = 0, S\cdot S = 1\]

    例：

    共有四个站进行码分多址CDMA通信。四个站的码片分别为

    • A:（-1 -1 -1 +1 +1 -1 +1 +1）
    • B:（-1 -1 +1 -1 +1 +1 +1 -1）
    • C:（-1 +1 -1 +1 +1 +1 -1 -1）
    • D:（-1 +1 -1 -1 -1 -1 +1 -1 ）

    现收到这样的码片序列：M=（-1 +1 -3 +1 -1 -3 +1 +1）

    问哪个站发送数据了？发送数据的站发送的1还是0

    使用A的码片内积M，得到1，因此是A站发送的1。同理D乘后为1；B乘后为-1，B发送了0；C乘后为0，则C没发送

    扩频通信（例如CDMA）浪费带宽的目的

    • 抗干扰、多径失真
    • 隐藏、加密信号
    • 多用户可以独立地使用相同的较宽的带宽而几乎没有相互干扰

    CDMA将是今后无线通信中主要的多址手段，应用范围已涉及数字蜂窝移动通信、卫星通信、微蜂窝系统、微波通信和无线接入网等领域

    SDMA（空分多址）：空分多址（SDMA）利用不同的用户空间特征（用户位置）区分用户，从而实现多址通信的方式。采用智能天线技术，可构成空间上用户的分割。

    配合电磁波被传播的特征，可使不同地域的用户在同一时间使用相同频率，实现互不干扰的通信。

    蜂窝移动通信中充分运用了SDMA方式，用有限的频谱构成大容量的通信系统。

    卫星通信用窄波束天线实现SDMA，提高了频谱利用率。

    智能天线技术利用数字信号处理技术产生空间定向波束，高效利用用户信号并可删除或抑制干扰信号。

    SDM（空间复用）：同一个频段在不同的空间内得到重复利用

    数字微波与卫星通信系统

    0.3~300GHz范围的电磁波，微波波长短，接近光波，是直线传播。这就要求两个通信点（信号转接点）之间无遮挡，即所谓的视距通信。

    SDH微波接力通信系统组成：一个完整的长途传输的微波接力通信系统由终端站、枢纽站、分路站及若干中继站所组成。

    flowchart TD
A[终端站] --> B1[中继站]
--> B2[中继站]
--> B3[...]
--> C[枢纽站]
--> D[分路站]
--> E[终端站]

C-->F[终端站]
    • 终端站：处于线路两端或分支线路终点的站。
    • 枢纽站：一般处在长途干线上，需要完成数个方向的通信任务。
    • 分路站：在长途线路中间，除了可以在本站上、下某收、发信波道的部分支路外，还可以沟通干线上两个方向之间通信。
    • 中继站：在线路中间，上、下话路的中间站。对已收到的已调信号进行解调、判决、再生，转发至下一方向的调制前，经过再生去掉干扰、噪声。

    数字微波通信系统由两个终端站和若干个中间站构成。由发端站、中间站和收端站组成。

    从发端送来的数字信号，经过数字基带信号处理(数字多路复用或数字压缩处理)后，经数字调制，形成数字中频调制信号，(70MHz或140MHz)再送入发送设备，进行射频调制变成为微波信号，送入发射天线向微波中间站(微波中继站)发送。

    微波中间站收到信号后经再处理，使数字信号再生后又恢复为微波信号向下一站再发送，这样一直传送到收端站。

    收端站把微波信号经过混频、中频解调恢复出数字基带信号，再分路还原为原始的数字信号。

    卫星通信系统：卫星通信是指利用人造地球卫星作为中继站转发无线电信号，在多个地球站之间进行的信息交流的通信方式。

    卫星通信部分主要包括发端地面站，收端地面站、上行线、下行线和通信卫星五大部分组成。

    flowchart TD
A((市内通信线路))--> B[终端设备]
-->C[发射设备]
--上行线路-->D[通信卫星]
--下行线路-->E[接收设备]
-->F[终端设备]
-->G((市内通信线路))

    在地面站要构成双工通信，既要向卫星发射信号，也要接收从卫星转发其它地面站送给本站的信号。所以，在实际地面站要完成双向通信过程。

    特点：

    • 覆盖面积大，通信距离远，一颗静止卫星可最大覆盖地球表面三分之一，三颗同步卫星可覆盖除两极外的全球表面，从而实现全球通信。
    • 设站灵活，容易实现多址通信。
    • 通信容量大，传送的业务类型多。
    • 卫星通信一般为恒参信道，信道特性稳定。
    • 电路使用费用与通信距离无关。
    • 建站快，投资省

    其不足主要表现为：

    • 要求卫星严格，有高可靠性、长寿命。
    • 通信地球站设备较复杂、庞大。
    • 卫星传输信号有延迟。

    地球卫星轨道

    • 赤道轨道：卫星的轨道平面与赤道平面的夹角为0°
    • 极轨道：卫星轨道平面与赤道平面夹角为90°
    • 倾斜轨道：卫星轨道平面与赤道平面夹角在0-90°之间

    当卫星运行轨道在赤道面内，称赤道轨道，如轨道呈圆形，此轨道离地面高度为35786.6km时，此轨道称为同步轨道。同步轨道只有一个，是宝贵的空间资源。

    卫星轨道分类：

    • 按形状
    • 圆轨道
    • 椭圆轨道
    • 按角度
    • 赤道轨道i=0
    • 极地轨道i=90°
    • 倾斜轨道（顺行0<i<90°、逆行90°<i<180°）
    • 按轨道周期
    • 回归/准回归轨道
    • 非回归轨道
    • 按高度
    • 低轨道
    • 中轨道
    • 高轨道

    利用同步卫星(静止卫星)来转发无线电信号组成的通信系统就称为同步卫星通信系统，作为通信用的这个卫星就叫做同步通信卫星。

    同步卫星优点：

    • 通信距离远，覆盖面积大，3颗便可覆盖全球。
    • 地球站不需要复杂的跟踪系统即可对准。

    缺点：

    • 高度高，因此信号衰减和时延大。
    • 地球的两极为通信盲区，而且高纬度地区通信效果较差。
    • 地球静止轨道只有一条，轨道上所能容纳的卫星数量有限。

    运行在中、低轨的卫星通信系统目前也得到广泛的应用。

    优点：

    • 卫星高度低，所以信号衰减小，时延小。
    • 卫星重量轻，结构简单。
    • 将卫星均匀地排布在整个地球的周围，即使是在南北极，也能使用低轨道卫星进行通信，实现真正的全球覆盖。

    缺点：

    • 覆盖整个地球需要大量的卫星，系统复杂。
    • 卫星数量多，寿命短，运行期间要及时补充发射，系统投资较高。

    卫星通信多址方式：

    • 频分多址
    • 优点：技术成熟，设备较简单，在大容量线路工作时效率较高
    • 缺点多个载波信号同时通过转发器时，会发生转发器有效输出功率降低、和生互调噪专用和可懂串话、强信号对弱信号的抑制等现象，因而有效容量将随载波增多而急剧降低，并且大、小站难以兼容，各站发射功率必须保持稳定。不灵活，要重新分配频率比较困难
    • 时分多址
    • 优点：充分利用卫星功率且无互调；充分利用转发器的频带；不会产生弱信号受抑制问题，上行功率不需要精确控制；各站所发分帧的码速率可以不同，便于大、小站兼空。TDMA方式在中、小容量的线路工作时，也可以得到相当高的效率。它的容量随站数增加而下降的速度比FDMA的缓慢得多。
    • 缺点：要有精确的同步，以保证各突发到达转发器的时间不发生重叠，并且保证接收站能正确识别站址和迅速建立载波、位定时的同步，而这是比较复杂、困难的技术。
    • 码分多址
    • 优点：具有较强的抗干扰能力；有一定的保密能力；改变地址比较灵活。
    • 缺点：要占用很宽的频带，频带利用率一般较低；要选择数量足够的可用地址码组较为困难；接收时，对地址码的捕获与同步需有一定时间。CDMA方式特别适用于军事卫星通信系统及小容量的系统
    • 空分多址：
    • 优点：卫星天线增益高；卫星功率可得到合理有效的利用；不同区域地球站所发信号在空间互不重叠，即使在同一时间用相同频率，也不会相到干扰，因而可实现频率重复使用，这就成倍地扩大了系统的通信容量
    • 缺点：SDMA方式对卫星的稳定及姿态控制提出很高的要求；卫星天线及馈线装置也比较庞大和复杂

    移动通信

    蜂窝数字移动通信系统

    这里主要指的是陆地移动通信，它一般由移动台(MS)，基地站(BS)及移动交换中心MSC组成移动通信网PLMN。

    移动通信网又通过中继线与市话通信网(PSTN)连接。

    大区制：一个基站天线覆盖区内的移动用户，只能在此区域完成联络与控制。通常用于基群通信。设备简单，投资少，见效快，但频率利用率低，扩容困难，不能漫游。

    小区制：小区制就是整个业务区(服务区)划分为若干小区，在小区中分别设置基站，负责本小区移动通信的联络控制。小区制通信中各基站的频率组配备，一定要使相邻基站天线覆盖区不能相同，否则会引起干扰

    小区形状考虑：

    • 邻接小区中心间距d越大越好，间隔大干扰就会愈小。从这个角度看正六边形小区为优。
    • 单位小区的有效面积越大越好，面积大则使一个区域小区个数少，使用频率数少。从图中可看出正六边形面积大，以此小区彼此邻接构成面状服务区最经济。
    • 交叠区域面积小为好，这使同频干扰最小，从图中看出正六边形为好
    • 交叠距离要小，使移动通信便于跟踪交接，从图中看出正六边形为好。
    • 所需无线电频率个数越少越好

    所以选择正六边形。在在单个小区半径R相同，发射功率、天线增益同等的情况下

    • 正六边形单位小区的面积最大，覆盖同样面积的服务区所需小区个数最少，即基站数最少，也就最经济；
    • 正六边形小区的中心间距最大，各基站之间的干扰最小；
    • 六边形最接近于圆形的辐射模式，全向基站天线和自由空间传播的辐射模式就是圆形的

    蜂窝小区频率复用

    共同使用全部可用频率的N个小区叫做一簇，若系统中簇复制得越多，则系统容量越大，频率的利用率越高。

    • 全部频谱在簇内分配使用
    • 对于全向覆盖并由N个小区组成的簇，频道总数被分成N组，每个小区分给一个频率组，全部频道分给簇。
    • 簇获得全部频率，相邻的簇重复使用相同的频率分配模式。

    N: 一个重用模型（在模型中每个蜂窝使用唯一的频率）的蜂窝区个数，称重用因子。典型值3、4、7、9、12 、…。满足小区条件的簇的形状和N是有限的:

    \[N = i^2 + ij + j^2\]

    i、j为簇之间的二维距离，i、j为正整数，不能同时为零

    找某一小区的相距最近的同频相邻小区按下步进行：沿着任意一条六边形边的垂线方向移动i个小区，并逆时针方向旋转60°，再移动j个小区。

    相邻小区距离\(\sqrt 3 R\)。以相邻小区距离进行归一化后得到最近同频小区间隔为

    \[D_{norm}^2 = j^2cos^2(30°) + (i + jsin(30°))^2 = i^2 + j^2 + ij\]

    则同频小区间隔：

    \[D = \sqrt 3 RD_{norm} = \sqrt{3(i^2 + j^2 + ij)} R\]

    区群中小区数目N：将候选小区六个最近的相邻同信道小区的中心连接，得到更大的六边形，其半径等于同信道小区之间的间隔D，面积等于

    \[A_{Large} = \frac 3 2 \sqrt 3 D^2 = \frac 3 2\sqrt 3 [3(i^2 + j^2 + ij) R^2]\]

    小区（小六边形）面积等于

    \[S_{small} = \frac 3 2 \sqrt 3 R^2\]

    则大六边形中包含的小区数为

    \[\frac {A_{Large}} {A_{Small}} = 3(i^2 + j^2 + ij) = N + 6 \cdot\frac 1 3 N = 3N\]

    所以有：

    \[D = \sqrt {3N} R\]
    • D: 使用相同频率的蜂窝区的最小中心距离
    • R: 一个蜂窝区的半径
    • N: 一个重用模型（在模型中每个蜂窝使用唯一的频率）的蜂窝区个数，称重用因子。

    同频复用比，对于六边形：

    \[Q = D/R = \sqrt {3N}\]

    N的选取：

    • N越大，则意味着同频小区间距离越远，同频干扰越小,但频率利用率低 。
    • N越小，则意味着一个系统中可有更多的簇，频率利用 率高，有更多的容量。
    • N的选取原则：从提高频率利用率的角度，在保持满意 的通信质量的前提下，N应取最小值最好。
    • 蜂窝系统的频率复用因子为1/N，因为一个簇中的每个小区都只分配给系统中所有可用信道的1/N。

    切换：

    • 硬切换：必须断掉原来小区的无线信道，才能使用新小区的无线信道进行通信；硬切换在空中接口是先断后通的过程。
    • 软切换：是当移动终端的通信被连到另一个小区的业务信道时，不需要中断当前服务小区的业务信道。当一个移动单元进入软切换状态时，它将同时与两个（或以上）基站连接。会浪费资源
    • 接力切换：终端从源小区接收下行数据，向目标小区发送上行数据，即上下行通信链路先后转移到目标小区。提前获取切换后的上行信道发送时间、功率信息，提高了切换成功率，缩短了切换时延

    漫游。漫游是指移动台在某地登记进网后，可在异地，同样进行呼叫处理通信。

    GSM是欧州邮电主管部门会议(CEPT)建立和开发的泛欧蜂窝全数字化的移动通信系统

    码分多址CDMA，特点：

    • 码分多址利用码型区分用户，必须有足够多地址码，且有良好的自相关、互相关特性
    • 接收端必须有本地地址码且结构与发端一样，并与发端同步才能在收端对全部信号进行相关检测取出所需信号
    • 在一个小区可使用同一频率，各用户可同时发送和接收信号，这是其它移动系统无能为力的，各用户在频率上复用，可克服同频干扰。
    • 码分多址CDMA各个用户在同一频带内各自占用相同带宽，要使各用户之间干扰降低到最低限度，码分系统必须与扩频技术相结合才能发挥其优势，才能有广阔的前途和实用价值。

    扩频

    扩频通信：系统占用频带宽度远大于传输的原始信号带宽（或信息比特率），且与原始信号带宽（信息比特率）无关。扩频100倍以上的调制信号称为扩频调制，即扩频增益\(G_p = W/B > 100\)，其中W为扩频信号带宽，B为信息带宽。理论基础位移香农信息容量极限公式：\(C= Wlog_2(1+S/N)\)

    扩频通信的特点

    1. 抗干扰能力强，可提高接收信噪比
    2. 可增加容量，可降低成本、质量好
    3. 移动手机功率可做得很小
    4. 频率规划简单

    分类：

    1. 直接序列(DS)扩频系统
    2. 跳频(FH)扩频系统
    3. 跳时(TH)扩频系统
    4. 脉冲线性调频系统，

    优点：

    1. 抑制干扰能力很强，经过接收机对信号的处理，可把信道中加进的并与扩频信号频带重叠的各种干扰信号强度减弱到原有的1/GP左右,因而使信扰比值提高了近GP倍。扩频通信并不能抑制自噪声的干扰，但能够有效地减弱各种窄带信号的干扰；
    2. 信号的功率谱密度很低；
    3. 信号便于隐蔽和保密；
    4. 可用以实现具有随意选址能力的码分多址通信
