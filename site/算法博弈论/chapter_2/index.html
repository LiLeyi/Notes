    复杂类Complexity Class

    常见问题

    计算机理论研究分两方面：

    • 算法
    • 复杂度

    NP search问题：\(R \subset \{0, 1\}^* \times \{0, 1\}^*\)，

    问题类\(Q_R\)：

    • 给定输入\(x\in \{0,1\}^*\)
    • 输出\(y\in \{0,1\}\)，使得\((x,y)\in R\)，否则无解输出no

    常见问题：

    • 3SAT
    • 旅行商问题

    TFNP（Total Functional NP，Total且为NP search问题）

    total：任何输入x，都一定存在y，使得\((x,y) \in R\)

    规约：对两个不同问题的\(Q_{R1},Q_{R2}\)。如果存在两个函数\(f,g\)，使得对任何输入\(x\in Q_{R_1}\)，如果\((f(x), y) \in R_2\)（因为为TFNP，所以一定有解y），则有\((x,g(y)) \in R_1\)。注：可能\(R_2\)有很多y的解，但\(g(y)\)都要是x在\(R_1\)中的解

    思想：若\(R_2\)能解决，则构造f，g就能解决\(R_1\)

    \(R_1 \leq R_2\)，即\(R_2\)比\(R_1\)难，\(R_2\)解决了\(R_1\)也能解决，但\(R_1\)解决了，\(R_2\)不一定解决。若\(R_1\leq R_2, R_2\leq R_1\)，则称\(R_1,R_2\)等价

    根据怎么证明TFNP一定有解可以对其进行细分

    • PPAD：在有向图中，每个点入度出度至多为1
    • PPA：在无向图中每个点度数至多为2
    • PPP：鸽笼原理

    PPAD：为问题集合，\(\{Q| Q\leq LEAFD\}\)，LEAFD即End-of-A-Line问题

    • 输入：\((M, 0^k)\)，M为图灵机，\(\forall v \in\{0, 1\}^k, M(v) = (u_1, u_2), u_1, u_2 \in \{0,1\}^k \cup \{no\}\)，即向M输入一个点，输出向前是哪个点，向后是哪个点；\(0^k\)为原点，\(M(0^k) = (no, 1^k)\)
    • 输出点\(v^* \neq 0^k\)，使得\(M(v^*)= \{u ,no\} or \{no, u\}\)，即\(v^*\)度为1

    LEAFD为PPAD中最难的问题

    若\(Q\leq LEAFD，则称在PPAD中\)。若\(LEAFD\leq Q\)，则称Q为PPAD-hard。如果\(Q\in PPAD\)。如果\(Q\in PPAD\)且\(Q\leq PPAD-hard\)，则称为PPAD完备的

    证明2D-sperner问题为PPAD-hard

    • 输入：\((F,0^k)\)，其中F用于确定每个点的颜色
    • 输出：\((p, p+e_1, p + e_2)\ or\ (p, p-e_1, p - e_2)\)，即全色三角形

    ...

    2D-Brouwer问题

    • 输入：与2D-sperner一致
    • 输出：输出顶点有三种颜色的正方形

    2D-Brouwer*问题：

    • 输入：与2D-sperner一致
    • 输出：正方形对角线只有一种方向，化为多个三角形。输出顶点有三种颜色的三角形

    易得\(2D-Brouwer\leq 2D-sperner\)，证\(2D-sperner\leq 2D-Brouwer\)。接下来将2D-Brouwer*问题转化为2D-Brouwer问题（即是否有全色三角形）

    将变成扩大三倍。原来的正方形是：

    1 2
    3 4

    变为：

    1 1 2 2
    1 2 2 2
    3 3 3 4
    3 3 4 4

    然后进行三角剖分即可（这样只有中间的左右两个正方形可能有三个颜色，其余正方形都只可能有两个颜色）

    2人博弈纳什均衡求解是PPAD-hard

    对于\(rank(A+B) \geq 2\)时，为PPAD-hard

B--> C[PL-Brouwer]
C--> D[2-Nash]
A -.-> C

    Piecewise-Linear：分段线性

    起点：2D-Sperner，先证其可规约到2D-Brouwer

    设正方形区域（内部为三角形），下边为黄色，左边为红色，其余两边为蓝色。对颜色进行编码，保证其不移动出整个区域：

    • \(D(Blue) = (-1, -1)\)
    • \(D(Red) = (1, 0)\)
    • \(D(Yellow) = (0,1)\)

    \(g(p) = p + D(p)\)。

    用布尔电路，输入为经过编码后的\(p_1, p_2\)，经过与或非组合电路（可分别转化1为min、max、1-x），输出\(a_1,s_1,a_2,s_2\)，\(D(p) = (a_1 - s_1, a_2 - s_2)\)

    考虑差值\(q = \sum_i \alpha_i p^i \Rightarrow g(q) = \sum_i \alpha_i g(p^i)\)，所以有\(g(q) = q + \sum_i \alpha_i D(p^i) = q + D(q)\)

    断言：q是g的不动点\(\Leftrightarrow D(q) = 0 \Leftrightarrow\)q是全色三角形的中心

    接下来要依据q去求全色三角形的三个顶点坐标。不能用取整，因为不连续。可以在间断点加入线性函数（斜率很大），这样q只有在非常接近整数时，结果才不正确...

    2D-Sperner可以在多项式时间内规约到2D-Brouwer

    纳什均衡求解

    Lemke-Howson算法

    该算法类似单纯形算法，且最坏情况下时间复杂度为指数（对于单纯形问题，若每个常参数都有一个期望为0的小扰动，则时间复杂度大概率为多项式。但对Lemke-Howson算法目前还没证明出）

    Lemke-Howson算法：对应二人纳什均衡。可以得出纳什均衡存在，且求解属于PPAD。\(G=(A,B)\)，A是行玩家的收益矩阵，B是列玩家的收益矩阵。

    对行玩家：

    \[x\in \Delta m = \{x\in R^n | x_i \geq 0, \sum_i x_i = 1\}\]

    对列玩家

    \[y\in \Delta n = \{y\in R^m | y_i \geq 0, \sum_i y_i = 1\}\]

    行玩家收益：\(x^T Ay\)，列玩家收益：\(x^T By\)。对两个收益矩阵，全部加上同一个数，纳什均衡不变（相对大小不变），全部同时乘一个正数也不变。

    定义\(P=\{x\in R^m | x\geq \vec 0, B^T x \leq \vec 1\}, Q = \{y\in R^n | y\geq \vec 0, Ay \leq 1\}\)（两个凸多面体，包围的空间均含原点）。对P、Q中每个非零向量，进行标准化，变为混合策略。

    接下来的思路：使用算法找到\(x\in P,y \in Q\)，对应纳什均衡，再经过标准化得到纳什均衡解。

    Label：该问题中，总共m+n个。Labels 1...m对应行玩家纯策略，m+1...m+n对应列玩家纯策略。

    对行玩家，\(x\in P\)如果\(x_i = 0\)，称为有label\(i\in \{1,\dots, m\}\)。如果\(m+j \in \{m+1,\dots, m+n\}\)，使得\((B^T x)_j = 1\)（即在第j个不等式取等），称为有label\(m+j\)。（对同一个x，这两种label最多满足一个）

    \(y\in Q\)，如果\(y_j = 0\)，则称y有label\(m+j\in \{ m+1, \dots, m+n\}\)。\((Ay)_i = 1\)，则称y有label\(i\in [n]\)。（对同一个y，这两种label最多满足一个）

    要使\(x,y\)的label并集包含所有label（且x，y均非零），此时为纳什均衡（为什么？）

    沿着边界移动，经过交点时，一个不等式变为等式，一个等式变为不等式

    1. 从原点开始
    2. Drop Label k，Label l出现
    3. 当l出现时，原来已经有l了，则放弃原来的老的l。若所有label都满足，则结束。否则回到2（因为老的l消失，还会有新的label出现）

    支持枚举算法

    适用2人博弈。时间为拟多项式

    \(G=(A,B), A, B \in \mathbb R^{m\times n}\)。纳什均衡定义为：

    \[x^T A y\geq x'^T Ay, x^T B y\geq x^T By',\forall x', y'\]

    等价定义

    \[x^T Ay\geq e_i^T Ay, x^T B y \geq x^T B e_j, \forall i,j\]

    即纳什均衡。定义行玩家支撑集，\(S_R(x) = \{i\in [m] | x_i \neq 0\}\)，定义列玩家支撑集，\(S_C(y) = \{j\in [n] | y_j \neq 0\}\)。nash均衡中动作的无差别性决定了采取在集合中的动作应该有相同的期望收益，有：

    \[e_i^T Ay \geq e_j^T Ay, \forall i \in S_R, j\in[m]\]
    \[x^T Be_i \geq x^T Be_j, \forall i \in S_R, j\in[n]\]
    \[\sum x_i = 1, \sum y_i = 1\]
    \[x_i = 0, i \notin S_R\]
    \[y_j = 0, j \notin S_C\]

    前两个式子：如果存在\(e_j\)，使得右边大于左边，那么可以把i的策略全部调整到j使收益最大，此时\(x_i = 0\)，与前提不符。

    然后可以通过线性规划判断是否存在纳什均衡，以及通过线性规划求解，求解时间为多项式。要通过枚举支撑集（为策略集合子集，枚举时间为指数）

    LMM算法

    为近似算法。时间为拟多项式

    假设\(A,B \in [0,1]^{n\times n}\)（矩阵所有元素加上里面元素最小值，再都除里面变化后的最大值。不是正方形可以补为正方形，全补0）

    \(\epsilon\)近似的纳什均衡（\(\epsilon-ANE\)）：给定\(\epsilon > 0\)，近似纳什均衡对应策略x，y满足条件\(x^T A y \geq x'^T A y -\epsilon\)，\(x^T A y \geq x^T A y' - \epsilon\)

    \(\epsilon -WSNE\)（Well-supported）：\(e_i^T Ay \geq e_k^T Ay -\epsilon , x_i\geq 0, x^T B e_j \geq x^T Be_k - \epsilon, y_j>0\)。这个可以推出上一个，\(\frac {\epsilon^2} 8-NE\)可以规约到\(\epsilon-WSNE\)

    [LMM03]\(\forall \epsilon \in (0,1), \exists \epsilon-ANE (x,y), |S_R(x)|,|S_C(y)|\leq O(\frac {logn} {\epsilon^2})\)，所以只需要检查规模小于\(O(\frac {logn} {\epsilon^2})\)的子集，约为\(n^{O(logn/(\epsilon^2))} << n^{logn} <<2^n\)（即不需要用到所有策略，只需要用到对数级别的策略即可）。对这个子集进行支持枚举即可

    通过概率方法证明，在概率空间中支撑集小于\(O(logn/(\epsilon^2))\)

    设\((x^*,y^*)\)为精确纳什均衡，定义策略k-uniform：存在分布\(x^*\)，输出为\([1,\dots, n]\)。，取样k次，得到\(\tilde x=(x_1,\dots, x_n), x_i = \frac 1 k c_i\)，其中\(c_i\)为选择到\(x^*\)中\(x_i\)的次数。

    接下来证取样次数\(k= O(logn/(\epsilon^2))\)时，能保证\(Pr[|e_i^T A y - e_i^T A \tilde{y}| \leq \epsilon / 2,\forall i \in[n]]>0\)或\(Pr[|x^T A e_j - \tilde x^T A e_j| \leq \epsilon / 2,\forall i \in[n]]>0\)，先证明\(\tilde x, \tilde y\)为近似纳什均衡（即证\(e_i^T A\tilde y \leq \tilde x^T A \tilde y + \epsilon\)）

    \[e_i^T A\tilde y \leq e_i^T Ay + \epsilon / 2 \leq e_{r_i}^T Ay + \epsilon / 2\]

    \(e_{r_i}\)为某次取样的纯策略，在支持集中。\(\tilde x\)的生成方式和\(e_{r_i}\)相关：

    \[e_{r_i}^T Ay + \epsilon/2 \leq \frac 1 k (\sum_{i=1}^k e_{r_i}^T)Ay + \epsilon / 2 \leq \frac 1 k (\sum_{i=1}^k e_{r_i}^T)A\tilde y + \epsilon / 2 + \epsilon / 2 = \tilde x^T A \tilde y + \epsilon\]

    接下来证那两个事件的概率大于0，即证存在混合策略，使得事件内不等式存在

    思路：

    要证：

    \[Pr[\bar E_1 \cap \bar E_2 \cap \dots \cap \bar E_n] > 0\]

    即证，在\(Pr[E_i] \leq s\)时：

    \[Pr[\bar E_1 \cap \dots\cap \bar E_n] = 1 - Pr[E_1 \cup E_2 \cup \dots \cup E_n] \geq 1 - \sum Pr[E_i] \geq 1 - ns\]

    即证

    \[1-ns > 0\]

    对于该题，要证\(Pr[|e_1^T Ay - e_1^T A\tilde{ y}|\geq \epsilon/2]\)（事件设为\(E_1\)）小于某个小值。可以通过集中不等式证明其有上界。Chernoff不等式：对\(X_i\sim D, X = \frac 1 k \sum X_i\)

    \[Pr[|X - E[X]| > \delta] \leq 2exp(-2k\delta^2)\]

    则有

    \[Pr[E_1] \leq 2 exp(-2k(\frac {\epsilon} 2)^2) = 2exp(- \frac {k\epsilon^2} 2)\]

    因为对行玩家有n个事件，列玩家有n个事件，则有不等式：

    \[4nexp(-k\epsilon^2 / 2) < 1\]

    即

    \[k> \frac {2log(4n)} {\epsilon^2}\]

    2016，Avaid Rubinstein证明存在理论下界：\(n^{logn^{1-\delta}}\)

    [DFM'22]对于\(\epsilon=1/3\)的近似纳什平衡,求解有多项式算法。对于\(\epsilon=1/2\)的WSNE，求解有多项式算法

    [KPS'06]提出\(\epsilon-NE\)在\(\epsilon=0.75\)时有多项式算法，后来有0.5、0.38、0.36、0.3339、1/3

    方法：Search & Mixing

    找A、B的最大元素，设在各自的矩阵中分别为\((i_1, j_1), (i_2, j_2)\)，混合策略为\(x=(e_{i1} + e_{i_2}) / 2, y=(e_{j1} + e_{j_2}) / 2\)。可得\(\epsilon = 0.75\)

    类似对0.5有类似证明

    对零和博弈，即\(A+B = 0\)，冯诺依曼证明零和博弈的纳什平衡求解等价于线性规划

    对于\(rank(A+B) \geq 2\)时，为PPAD-hard。\(rank(A+B)=1\)时，为P

    [DFM'22]对于博弈\((A,B)\)，对\((A, -A)\)算出纳什均衡\((x_A,y_A)\)，对\((B, -B)\)算出纳什均衡\((x_B,y_B)\)，则\((x_A, y_B)\)是\(1/2 + \delta-WSNE\)（用到了K-uniform，且取\(k=1/\delta\)）
