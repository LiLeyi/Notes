    通路与回路

    通路

    给定图中，G中定点与边依次交替出现的交替序列\(\Gamma = v_0 e_1 v_1 e_2 \dots e_l v_l\)，其中\(v_i,v_{i-1}\)是\(e_i\)的端点，则称\(\Gamma\)为通路。若\(v_0 = v_l\)，则称为回路，l为回路长度

    • 若所有边各异，则称为简单通路，若为回路，则称为简单回路。若有重复边，则称为重复...
    • 所有顶点（除了首位定点）、边均不一样称为初级通路（路径），类似有初级回路（圈）

    无向简单图中圈长度大于等于3，有向简单图中回路长度大于等于。圈：

    • 定义意义下：图中长度为l的圈，在定义意义下有2l个；有向图中，长度为l的圈（l大于等于3），定义意义下有l个
    • 同构意义下：长度相同的圈均为1个

    定理：

    • n阶图G中，若两点间存在通路，则这两点存在长度小于等于n-1的通路
    • n阶图G中，若两点间存在通路，则这两点存在长度小于等于n-1的初级通路
    • 在一个n阶图G中，若存在v到自身的回路，则一定存在v到自身长度小于等于n的回路
    • 在一个n阶图G中，若存在v到自身的回路，则一定存在v到自身长度小于等于n的初级回路

    连通性

    无向图

    连通性

    若\(v_i\)与\(v_j\)之间有通路，则\(v_i\sim v_j\)，为等价关系。若\(\forall u,v\in V,u\sim v\)，则称G连通。\(V/R=\{V_1,V_2\dots V_k\}\)，称\(G[V_1],G[V_2],\dots,G[V_k]\)为G的连通分支，个数\(p(G)=k\)，k=1时连通

    u，v之间的短程线指\(u\sim v\)时，之间的最短通路。u，v的距离指短程线的长度。

    • \(d(u,v)\geq 0\)，取等当且仅当两点相同。
    • \(d(u,v)=d(v,u)\)。
    • \(d(u,v)+d(v,w)\geq d(u,w)\)

    删除：

    • \(G-v\)，从G中将v及关联的边去掉
    • \(G-V'\)，从G中删除V'的所有顶点
    • \(G-e\)，将e从G中去掉
    • \(G-E'\)，将E'中的边从G中删除

    点割集与边割集

    • \(V'\)是V的子集，且\(p(G-V')>p(G)\)，且V'是满足该条件的最小集合，则称V'为点割集。当\(\{v\}\)为点割集时，称v为割点
    • \(E'\)是E的子集，且\(p(G-E')>p(G)\)，且E'是满足该条件的最小集合，则称E'为边割集。当\(\{e\}\)为边割集时，称e为割边（桥）

    注：

    • \(K_n\)中无点割集
    • \(N_n\)中既无点割集，也无边割集（\(N_n\)为n阶零图）
    • 若G连通，\(E'\)为边割集，则\(p(G-E')=2\)
    • 若G连通，\(V'\)为点割集，则\(p(G-V')\geq 2\)

    G为连通非完全图，点连通度\(\kappa(G) = min\{|V'||V'为点割集\}\)。若\(\kappa(G)\geq k\)，则称G为k-连通图

    \(\lambda(G) = min\{|E'||E'为边割集\}\)，若G非联通，则\(\lambda(G) = 0\)，若\(\lambda(G)\geq r\)，则称G为r边-弱连通图

    • 规定\(\kappa(K_n) = \lambda(G) = n-1\)。
    • 若G非连通，则\(\kappa(G) = \lambda(G) = 0\)

    定理：\(\kappa(G)\leq \lambda(G) \leq \delta(G)\)

    有向图

    连通性

    若v到u有通路，则记作\(v\rightarrow u\)。若互相可达，则记为\(v\leftrightarrow u\)，为等价关系。

    有向图中有短程线与距离，定义类似。但距离\(d<v,u>\)无对称性

    • 弱连通（连通）：基图为无向连通图
    • 单向连通：\(\forall u,v\)，\(u\rightarrow v\)或\(u\leftarrow v\)
    • 强连通：\(\forall u,v, u\leftrightarrow v\)

    强连通\(\Rightarrow\)单向连通\(\Rightarrow\)弱连通

    • D强连通当且仅当D中存在经过每个顶点至少一次的回路
    • D单向连通当且仅当D中存在经过每个顶点至少一次的通路

    扩大路径法

    设G为n阶无向图，E不空，设\(\Gamma_l\)为G中的一条路径，若次路径的终点与起点与路径外的顶点相连，就将他们扩到路径中，继续这一过程，直到最后得到的路径不与路径外的顶点相邻为止。。

    设最后得到的路径为\(\Gamma_{l+k}\)，称其为极大路径。有向图中类似，只不过需要注意保证有向边的一致性

    极大路径不唯一，不一定是图中最长的路径。无向简单图中极大路径长度\(\geq \delta (G)\)

    例：设G为n阶(\(n\geq 3\))无向简单图，\(\delta \geq 2\)证G中存在长度\(\geq \delta +1\)的圈

    设\(\Gamma=v_0v_1\dots v_l\)是由初始路径\(\Gamma_0\)用扩大路径法得到的极大路径，则\(l\geq \delta\)。

    因为\(v_0\)不与\(\Gamma\)外顶点相邻，且\(d(v_0)\geq \delta\)，因此在\(\Gamma\)上除了\(v_1\)外，至少还有\(\delta - 1\)个顶点与\(v_0\)相邻

    设\(v_x\)是离\(v_0\)最远的顶点，于是\(v_0v_1\dots v_x v_0\)为G中长度\(\geq \delta + 1\)的圈

    二部图

    设G为无向图，若能将V分成\(V_1,V_2\)，且\(V_1\cup V_2 = V,V_1\cap V_2 = \phi\)，使得G中每条边两个顶点都是一个属于\(V_1\)，一个属于\(V_2\)，则称G为二部图（或二分图、偶图），称\(V_1,V_2\)为互补顶点集。常将二部图极为\(<V_1,V_2,E>\)

    若G是简单二部图，且\(V_1\)中每个顶点均与\(V_2\)中所有顶点相邻，则称G为完全二部图，记为\(K_{r,s}\)，其中\(r=|V_1|,s=|V_2|\)。

    n阶零图为二部图

    无向图为二部图当且仅当G中无奇数长度的圈（奇圈）
