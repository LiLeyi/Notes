    图的基本概念

    多重集合：元素可以重复出现的集合

    无序积\(A\& b = \{(x,y)|x\in A \land y \in B\}\)

    定义

    图

    无向图：\(G=<V,E>\)，其中

    • \(V\neq \phi\)为顶点集，元素称为顶点
    • E为\(V\& V\)的多重子集，其元素称为无向边，简称边

    有向图\(D=<V,E>\)，其中E是\(V\times V\)的多重子集

    有n个节点的图称为n阶图

    • n阶零图：无边
    • 平凡图：只有一个顶点，无边。即一阶零图
    • 空图\(\phi\)：无顶点

    两个节点间若有边，则称为是相邻的。若一条边的终点是另外一条边的起点，则称这两条边是相邻的

    对一条边\(e_k\)与\(v_i\),\(v_j\)，\(v_i\neq v_j\)，则这条边与\(v_i\)的关联次数为1；若\(v_i=v_j\)，则关联次数为2。若\(e_k\)的两端不是\(v_i\)和\(v_j\)，则关联次数为0

    关联次数为2的边为环

    邻域与关联集

    • 无向图
    • 邻域\(N(v) = \{u|u\in V(G) \land (u,v)\in E(G) \land y\neq v \}\)，相邻节点的集合（不包括自己）
    • 闭邻域\(\bar N(v) = N(v) \cup \{v\}\)。相邻节点的集合（包括自己
    • v的关联集\(I(v) = \{e | e\in E(G) \land e与v关联\}\)。是相连边的集合
    • 有向图中：
    • v的后继元集\(\Gamma^+(v) = \{u| u\in V(D)\land <v,u>\in E(D) \land u\neq v\}\)
    • v的前驱元集\(\Gamma^-(v) = \{u| u\in V(D)\land <u,v>\in E(D) \land u\neq v\}\)
    • 邻域\(N(v) = \Gamma^+(v) \cup \Gamma^-(v)\)
    • 闭邻域

    标定图与非标定图（为顶点与边指定符号）

    有向图的基图，即把所有边换成无向边

    重数

    • 无向图中两节点间平行边的重数：即平行边的个数
    • 有向图中一个节点到另外一个节点的平行边的重数：即平行边的个数，注意方向性
    • 多重图：含平行边的图
    • 简单图：既不含平行边也不含环的图

    度数

    • 无向图：度数为节点的关联集的元素个数，即相连边的个数
    • 有向图
    • \(d^+(v)\)：v的出度，即以该节点为起点的边的个数
    • \(d^-(v)\)：v的入度，即以该节点为终节点的边的个数
    • \(d(v) = d^+(v) + d^-(v)\)：度数
    • \(\Delta(G)\)为最大度，\(\delta (G)\)为最小度
    • \(\Delta ^+ (D),\Delta ^- (D),\delta^+ (D),\delta ^- (D),\Delta(D),\delta(D)\)
    • 奇度定点、偶度定点

    握手定理：

    • 任何无向图中，所有顶点度数之和等于边的两倍
    • 任何有向图中，所以定点度数之和等于边的两倍。所有顶点的入度之和等于所有顶点的出度之和，等于边数

    推论：

    • 任何图中，奇度定点的个数为偶数

    \(V=\{v_1,v_2,\dots,v_n\}\)为无向图的顶点集，称\(d(v_1),d(v_2),\dots,d(v_n)\)为G的度数列

    对于有向图，有度数列、出度列、入度列。

    对于定点标定的无向图，度数序列唯一。若对于非负整数列d，若存在V为顶点集的n阶无向图G，使得\(d(v_i) = d_i\)，则称d为可图化的，若得到的图为简单图，则称d为可简单图化的。判断是否可简单图化：Havel定理

    非负整数列是可图化的当且仅当\(\sum d_i\)为偶数

    设G为任意n阶简单无向图，则\(\Delta (G) \leq n-1\)

    同构

    若\(G_1,G_2\)为两个无向图（有向图），若存在双射函数f，对于\(V_i,v_j\in V_1\)，\((v_i,v_j)\in E_1\)当且仅当\((f(v_i),f(v_j))\in E_2\)（\(<v_i,v_j>\in E_1\)当且仅当\(<f(v_i),f(v_j)>\in E_2\)），并且\((v_i,v_j)\)与\((f(v_i),f(v_j))\)重数相同（\(<v_i,v_j>\)与\(<f(v_i),f(v_j)>\)重数相同），则称\(G_1,G_2\)同构，记为\(G_1\cong G_2\)

    同构为等价关系

    必要条件：

    • 边数相同
    • 顶点数相同
    • 度数列相同
    • ...

    n阶完全图与竞赛图

    n阶无向完全图：每个顶点与其余定点均相邻的无向简单图，记作\(K_n\)。

    \[m = \frac {n(n-1)} 2,\Delta = \delta = n-1\]

    其中边数为\(m\)

    n阶有向完全图：每个顶点之间均匀两条方向相反的有向边的有向简单图

    \[m=n(n-1), \Delta = \delta = 2(n-1),\Delta^+ = \delta^+ = n-1,\Delta^- = \delta^- = n-1\]

    竞赛图：基图为\(K_n\)的有向简单图

    设G为简单无向图，每个节点度数为k，则称为k-正则图

    \[m = \frac {nk} 2\]
    • n阶零图为0-正则图
    • \(K_n\)为\((n-1)\)-正则图
    • 彼得松图为3-正则图

    子图

    \(G=<V,E>,G' = <V',E'>\)，若\(V\subseteq V'\land E\subseteq E'\)，则称G'为G的子图，记作\(G'\subseteq G\)，G为G'的母图

    若\(G'\subseteq G\land V' = V\)，则称G'为G的生成子图（顶点相同）

    G中两个端点都在V'中的边组成边集E'，称为V'的生成子图，记为\(G[V']\)

    E'中关联的G中顶点为顶点集V'，称为E'的生成子图，记为\(G[E']\)

    补图

    以V为顶点集，以所有使G成为完全图的添加边的集合为边集，称为G的补图，记为\(\bar G\)

    若\(G\cong\bar G\)，则称G为自补图。自补图边数为完全图的一半，且完全图边数要为偶数，即自补图的阶数为\(n=4k\)或\(n=4k+1\)
