    运算法

    运算符

    算数运算符

    • 加法 +
    • 减法 -
    • 乘法 *
    • 除法 /, a div b
    • 取模 %, mod(a,b)

    比较运算符

    真返回1，假返回0，不确定返回null

    • <, >
    • <=, >=
    • <>, != 不等于
    • <=> null安全的等于，可以判断null=null
    • x between a and b x是否在a，b之间
    • in 是否在指定集合中
    • not in
    • is null
    • is not null
    • like 通配符比较

    例:

    select
sum(case when 0.8 * grade_1 + 0.2 * grade_2 between 90 and 100 then 1 else 0 end) as NinetyToOneHundred,
sum(case when 0.8 * grade_1 + 0.2 * grade_2 between 80 and 89 then 1 else 0 end) as EightyToNinety,
sum(case when 0.8 * grade_1 + 0.2 * grade_2 between 70 and 79 then 1 else 0 end) as SeventyToEighty,
sum(case when 0.8 * grade_1 + 0.2 * grade_2 between 60 and 69 then 1 else 0 end) as SixtyToSeventy
from grade

    逻辑运算符

    • and, &&
    • or, ||
    • not, !
    • xor 异或，不等返回1
    select
sum(case when grade_1 >= 90 and grade_2 > 90 then 1 else 0 end) as sum1,
sum(case when grade_1 >= 90 or grade_2 > 90 then 1 else 0 end) as sum2
from grade;

    位运算符

    • & 位与
    • | 位或
    • ^ 位异或
    • ~ 位非
    • >> 位后移
    • << 位前移
    select binary 1 $ 2;
select binary 1 | 2;
select binary 1 ^ 2;
select binary 1 ~ 2;

select binary 2 >> 1;
select binary 2 << 1;

    常用函数

    select、insert、update、delete语句都可以使用函数

    • length(str)
    • trim(str) 去除字符串开头、结尾的空格
    • concat(str_1,str_2,...) 连接字符串
    • insert(str,x,y,instr) 将字符串str从第x位置开始，y个字符长的子串替换为instr。索引从0开始，先删除x-y这一段，再插入
    • lower(str) 小写
    • upper(str) 大写
    • left(str,x) 返回字符串最左边x个字符
    • right(str,x) 返回字符串最右边x个字符
    • lpad(str,x,pad_str) 在字符串左侧填充x个pad_str
    • rpad(str,x,pad_str)
    • ...

    数值函数

    • abs(x) 绝对值
    • ceil(x) 向上取整
    • floor(x) 向下取整
    • mod(x,y) 取余
    • round(x,y) 四舍五入，保留y位小数
    • rand() 返回0~1间随机数
    • truncate(x,y) 截断，保留y位小数
    • pow(x,y) x的y次方

    日期和时间函数

    • curdate() 返回当前日期
    • curtime() 返回当前时间
    • now() 返回当前日期和时间
    • unix_timestamp(date) 返回日期date的unix时间戳
    • from_unixtime(timestamp) 返回unix时间戳timestamp对应的日期
    • week(date)
    • year(date)
    • ...

    其他

    • database() 返回当前数据库名
    • version() 返回当前数据库版本
    • user() 返回当前用户名
    • inet_aton(ip) 返回当前用户的ip
    • inet_ntoa(num) 返回谁代表的ip地址
    • password(str) 返回str加密后的字符串
    • md5(str) 返回字符串的md5值

    流程函数

    • if(value,t,f) 判断value是否为真，是返回t，假返回f
    • ifnull(value_1,value_2) 如果value_1为null，返回value_2，否则返回value_1
    select
id,
ifnull(if(grade_1 >= 60,'及格','不及格'),'未参加') as '课程1',
ifnull(if(grade_2 >= 60,'及格','不及格'),'未参加') as '课程2'
from grade;
