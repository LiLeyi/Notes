    DML

    定义

    对数据库中数据进行操作

    关键字 insert,delete等

    对值的操作

    查看表的数据：

    • select * from 表名;

    插入

    • insert into 表名 values(列对应值,列对应值...);
    • insert into 表名(列名) values(该列的值); 将其他值设置为默认值
    • insert into 表名 values(列对应值,列对应值...),(列对应值,列对应值...),...; 插入多个数据
    • insert into 表名(列名) values(该列的值1),(该列的值2),...; 插入多个数据

    删除

    • delete from 表名; 删除表内所有数据
    • delete from 表名 where 条件; 删除表内符合条件的数据
    • delete 表名,表名,... from 需要删除数据的表名 where 条件; 几个表之间不能有关系（例如外键、联结）

    字符串、日期数据类型加单引号

    更新值

    • update 表名 set 列1=值1,列2=值2,... [where 条件];
    • update 表名,表名... set 表名.列名=值,表名.列名=值... [where 条件];

    查询

    • select *或者列名 from 表名 [where 条件];
    • select distinct 列名 from 表名; 查询不重复
    • select * from 表名 order by 列名 [asc|desc];

    附带限制等：

    • select ... limit 选择行数 offset 从多少行开始; //offset 默认从0开始
    • select 聚合函数 from 表名; 使用聚合函数来得到数据
    • select ... from 表名 [where 条件] [group by 列名 [with rollup]] [having 条件] ;

    聚合函数：

    • count(*|列名):统计行数
    • sum(列名)：求和
    • avg(列名)：平均值
    • max(列名)：最大值
    • min(列名)：最小值
    • group by 列名：分类聚合，例如用来统计name列中姓李的有几个
    • with rollup:对分类聚合后的结果进行再汇总，即求和值放在最后一行
    • having 条件: 过滤分组后的结果，where在聚合前进行筛查

    连接

    内连接
    • select 列1,列2... from 表1,表2... [where 条件]; 若两表中有重复的列名，则必须指明表名，例如tableName.columnName
    外连接
    • 左连接：包含所有左边表中的记录和右边表中与左表匹配的记录

      select 列1,列2,... from 左表名 left join 右表名 on 条件; 会把左表中所有数据全部显示，右表只显示满足条件的，若不满足，则值变为NULL

    • 右连接：包含所有右边表中的记录和左边表中与右表匹配的记录

      select 列1,列2,... from 左表名 right join 右表名 on 条件; 会把右表中所有数据全部显示，左表只显示满足条件的，若不满足，则值变为NULL

    子查询

    • select ... where 条件带子查询关键字(select ...)

    子查询关键字：

    • in
    • not in
    • exists
    • any
    • all

    记录联合

    • select ... union select ...;会去除重复数据
    • select ... union all select ...;

    例如:统计出某一门课的成绩在90-100、80-89、70-79、60-69、0-59的人数，要输出课程名和成绩区间的总人数。

    SELECT 
    c.name AS course_name,
    SUM(CASE WHEN s.score BETWEEN 90 AND 100 THEN 1 ELSE 0 END) AS '90-100',
    SUM(CASE WHEN s.score BETWEEN 80 AND 89 THEN 1 ELSE 0 END) AS '80-89',
    SUM(CASE WHEN s.score BETWEEN 70 AND 79 THEN 1 ELSE 0 END) AS '70-79',
    SUM(CASE WHEN s.score BETWEEN 60 AND 69 THEN 1 ELSE 0 END) AS '60-69',
    SUM(CASE WHEN s.score BETWEEN 0 AND 59 THEN 1 ELSE 0 END) AS '0-59'
FROM 
    score s
JOIN 
    course c ON s.courseid = c.id
GROUP BY 
    c.name;

    例：找出成绩表中90-100分的学生中成绩为100的同学

    select studentinfo.name from studentinfo left join score on score.studentid=studentinfo.id where score.score = 100;
